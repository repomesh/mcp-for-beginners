# اجرای نمونه

در اینجا فرض می‌کنیم که شما قبلاً کد سرور عملی دارید. لطفاً یک سرور را از یکی از فصل‌های قبلی پیدا کنید.

## تنظیم mcp.json

این یک فایلی است که برای مرجع استفاده می‌کنید، [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json).

ورودی سرور را به گونه‌ای که لازم است تغییر دهید تا مسیر مطلق سرور شما را از جمله فرمان کامل لازم برای اجرا نشان دهد.

در فایل نمونه که بالاتر به آن اشاره شد، ورودی سرور به این صورت است:

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

ممکن است لازم باشد مسیر ریشه مخزن GitHub را وارد کنید، که می‌توان آن را با دستور `git rev-parse --show-toplevel` به دست آورد.

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

این معادل اجرای دستوری مانند این است: `node build/index.js`.

- این ورودی سرور را تغییر دهید تا با محل فایل سرور شما یا آنچه برای راه‌اندازی سرور با توجه به زمان اجرای انتخاب شده و محل سرور لازم است مطابقت داشته باشد.

## استفاده از قابلیت‌ها در سرور

- روی آیکون `play` کلیک کنید، پس از اینکه *mcp.json* را به پوشه *./vscode* اضافه کردید،

    مشاهده کنید که آیکون ابزار تغییر می‌کند و تعداد ابزارهای در دسترس افزایش می‌یابد. آیکون ابزار درست بالای کادر چت در GitHub Copilot قرار دارد.

## اجرای یک ابزار

- در پنجره چت خود یک فرمان تایپ کنید که با توضیح ابزار شما مطابقت دارد. برای مثال برای فعال کردن ابزار `add` چیزی مانند "add 3 to 20" تایپ کنید.

    باید ابزاری را بالای کادر متن چت ببینید که از شما می‌خواهد برای اجرای ابزار آن را انتخاب کنید، مشابه این تصویر:

    ![VS Code indicating it wanting to run a tool](../../../../../translated_images/fa/vscode-agent.d5a0e0b897331060.webp)

    انتخاب ابزار باید نتیجه عددی‌ای مانند "23" تولید کند اگر فرمان شما مانند آنچه قبلاً گفتیم باشد.
