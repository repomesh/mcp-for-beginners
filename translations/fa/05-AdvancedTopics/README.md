# موضوعات پیشرفته در MCP

[![MCP پیشرفته: عامل‌های هوش مصنوعی امن، مقیاس‌پذیر و چندرسانه‌ای](../../../translated_images/fa/06.42259eaf91fccfc6.webp)](https://youtu.be/4yjmGvJzYdY)

_(برای مشاهده ویدیوی این درس روی تصویر بالا کلیک کنید)_

این فصل شامل مجموعه‌ای از موضوعات پیشرفته در پیاده‌سازی پروتکل زمینه مدل (MCP) است، از جمله یکپارچه‌سازی چندرسانه‌ای، مقیاس‌پذیری، بهترین روش‌های امنیتی و یکپارچه‌سازی سازمانی. این موضوعات برای ساخت برنامه‌های MCP مقاوم و آماده تولید که می‌توانند نیازهای سیستم‌های هوش مصنوعی مدرن را برآورده کنند، حیاتی هستند.

## مرور کلی

این درس مفاهیم پیشرفته در پیاده‌سازی پروتکل زمینه مدل را بررسی می‌کند، با تمرکز بر یکپارچه‌سازی چندرسانه‌ای، مقیاس‌پذیری، بهترین روش‌های امنیتی و یکپارچه‌سازی سازمانی. این موضوعات برای ساخت برنامه‌های MCP با درجه تولیدی که بتوانند نیازهای پیچیده در محیط‌های سازمانی را برآورده کنند، اساسی هستند.

> **نگاهی به آینده:** چندین موضوع زیر تحت تأثیر نامزد انتشار مشخصات MCP در تاریخ `2026-07-28` هستند — زمینه‌های ریشه‌ای (5.4) و نمونه‌برداری (5.6) بر مبنای ابتداییاتی ساخته شده‌اند که نامزد انتشار به عنوان منسوخ علامت زده است، و ویژگی تجربی وظایف که در ویژگی‌های پروتکل (5.16) اشاره شده به یک افزونه اختصاصی توابع منتقل می‌شود. برای جزئیات بیشتر به [چه تغییراتی در MCP اتفاق افتاده: نامزد انتشار 2026-07-28](../01-CoreConcepts/mcp-2026-07-28-release-candidate.md) مراجعه کنید.

## اهداف یادگیری

در پایان این درس، شما قادر خواهید بود:

- پیاده‌سازی قابلیت‌های چندرسانه‌ای در چارچوب‌های MCP
- طراحی معماری‌های مقیاس‌پذیر MCP برای سناریوهای با تقاضای بالا
- به‌کارگیری بهترین روش‌های امنیتی مطابق با اصول امنیتی MCP
- یکپارچه‌سازی MCP با سیستم‌ها و چارچوب‌های هوش مصنوعی سازمانی
- بهینه‌سازی عملکرد و قابلیت اطمینان در محیط‌های تولیدی

## نکات کلیدی

- پیاده‌سازی‌های چندرسانه‌ای MCP قابلیت‌های هوش مصنوعی را فراتر از پردازش متن گسترش می‌دهند
- مقیاس‌پذیری برای استقرارهای سازمانی ضروری است و می‌توان آن را از طریق مقیاس‌گذاری افقی و عمودی برطرف کرد
- اقدامات امنیتی جامع از داده‌ها محافظت کرده و کنترل دسترسی مناسب را تضمین می‌کنند
- یکپارچه‌سازی سازمانی با پلتفرم‌هایی مانند Azure OpenAI و Microsoft AI Foundry قابلیت‌های MCP را افزایش می‌دهد
- پیاده‌سازی‌های پیشرفته MCP از معماری‌های بهینه‌شده و مدیریت منابع دقیق بهره‌مند می‌شوند

## تمرین

طراحی یک پیاده‌سازی MCP با درجه سازمانی برای یک مورد استفاده خاص:

1. نیازمندی‌های چندرسانه‌ای مورد استفاده خود را شناسایی کنید
2. کنترل‌های امنیتی لازم برای حفاظت از داده‌های حساس را ترسیم کنید
3. معماری مقیاس‌پذیری را طراحی کنید که بتواند بارهای متغیر را اداره کند
4. نقاط یکپارچه‌سازی با سیستم‌های هوش مصنوعی سازمانی را برنامه‌ریزی کنید
5. گلوگاه‌های بالقوه عملکرد و راهکارهای کاهش آن را مستندسازی کنید
