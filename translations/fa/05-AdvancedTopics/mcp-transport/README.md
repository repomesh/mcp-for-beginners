# ШұШ§ЩҮЩҶЩ…Ш§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„вҖҢЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ MCP

ЩҫШұЩҲШӘЪ©Щ„ Щ…ШҜЩ„ Ъ©Ш§ЩҶШӘЪ©ШіШӘ (MCP) Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ШҜШұ Щ…Ъ©Ш§ЩҶЫҢШІЩ…вҖҢЩҮШ§ЫҢ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ ШұШ§ ЩҒШұШ§ЩҮЩ… Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ Ш§Щ…Ъ©Ш§ЩҶ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢвҖҢЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ ШЁШұШ§ЫҢ Щ…ШӯЫҢШ·вҖҢЩҮШ§ЫҢ ШіШ§ШІЩ…Ш§ЩҶЫҢ ШӘШ®ШөШөЫҢ ШұШ§ Щ…ЫҢвҖҢШҜЩҮШҜ. Ш§ЫҢЩҶ ШұШ§ЩҮЩҶЩ…Ш§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢвҖҢЩҮШ§ЫҢ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ ШіЩҒШ§ШұШҙЫҢ ШұШ§ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Azure Event Grid ЩҲ Azure Event Hubs ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ Щ…Ш«Ш§Щ„вҖҢЩҮШ§ЫҢ Ш№Щ…Щ„ЫҢ ШЁШұШ§ЫҢ ШіШ§Ш®ШӘ ШұШ§ЩҮвҖҢШӯЩ„вҖҢЩҮШ§ЫҢ MCP ШЁЩҲЩ…ЫҢ Ш§ШЁШұЫҢ ЩҲ Щ…ЩӮЫҢШ§ШівҖҢЩҫШ°ЫҢШұ ШЁШұШұШіЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

> **ЩҶЪҜШ§ЩҮЫҢ ШЁЩҮ ШўЫҢЩҶШҜЩҮ:** Ш§ЫҢЩҶ ШұШ§ЩҮЩҶЩ…Ш§ ШЁШұ Ш§ШіШ§Ші **Щ…ШҙШ®ШөШ§ШӘ MCP 2025-11-25** ЩҶЩҲШҙШӘЩҮ ШҙШҜЩҮ Ш§ШіШӘШҢ Ш¬Ш§ЫҢЫҢ Ъ©ЩҮ ШӘШұШӘЫҢШЁ Ш¬Щ„ШіШ§ШӘ ШЁШ§ЫҢШҜ ШЁШұШ§ЫҢ ЩҮШұ Ш¬Щ„ШіЩҮ ШӯЩҒШё ШҙЩҲШҜ (ШЁЩҮ ЩҫШұЩҲШӘЪ©Щ„ ЩҫЫҢШ§Щ… ШҜШұ Ш§ШҜШ§Щ…ЩҮ Щ…ШұШ§Ш¬Ш№ЩҮ Ъ©ЩҶЫҢШҜ). ЩҶШіШ®ЩҮ Ъ©Ш§ЩҶШҜЫҢШҜ `2026-07-28` Ъ©Щ„ ШіШ·Шӯ Ш¬Щ„ШіЩҮ ШҜШұ ЩҫШұЩҲШӘЪ©Щ„ ШұШ§ ШӯШ°ЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜ ЩҲ ЩҶЫҢШ§ШІЩ…ЩҶШҜ ЩҮШҜШұЩҮШ§ЫҢ `Mcp-Method`/`Mcp-Name` Ш§ШіШӘ ШӘШ§ ЪҜЫҢШӘвҖҢЩҲЫҢвҖҢЩҮШ§ ЩҲ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„вҖҢЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ ШЁШӘЩҲШ§ЩҶЩҶШҜ ШЁЩҮ Ш§ШІШ§ЫҢ ЩҮШұ ШҜШұШ®ЩҲШ§ШіШӘ Щ…ШіЫҢШұвҖҢЫҢШ§ШЁЫҢ Ъ©ЩҶЩҶШҜ ЩҶЩҮ ШЁЩҮ Ш§ШІШ§ЫҢ ЩҮШұ Ш¬Щ„ШіЩҮ. ## Щ…ЩӮШҜЩ…ЩҮ

ШҜШұ ШӯШ§Щ„ЫҢ Ъ©ЩҮ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„вҖҢЩҮШ§ЫҢ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ MCP (stdio ЩҲ HTTP streaming) Ш§Ъ©Ш«Шұ Щ…ЩҲШ§ШұШҜ Ш§ШіШӘЩҒШ§ШҜЩҮ ШұШ§ ЩҫЩҲШҙШҙ Щ…ЫҢвҖҢШҜЩҮЩҶШҜШҢ Щ…ШӯЫҢШ·вҖҢЩҮШ§ЫҢ ШіШ§ШІЩ…Ш§ЩҶЫҢ Ш§ШәЩ„ШЁ ШЁЩҮ Щ…Ъ©Ш§ЩҶЫҢШІЩ…вҖҢЩҮШ§ЫҢ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ ШӘШ®ШөШөЫҢ ШЁШұШ§ЫҢ ШЁЩҮШЁЩҲШҜ Щ…ЩӮЫҢШ§ШівҖҢЩҫШ°ЫҢШұЫҢШҢ ЩӮШ§ШЁЩ„ЫҢШӘ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ ЩҲ ЫҢЪ©ЩҫШ§ШұЪҶЪҜЫҢ ШЁШ§ ШІЫҢШұШіШ§Ш®ШӘвҖҢЩҮШ§ЫҢ Щ…ЩҲШ¬ЩҲШҜ Ш§ШЁШұЫҢ ЩҶЫҢШ§ШІ ШҜШ§ШұЩҶШҜ. ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„вҖҢЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ ШЁЩҮ MCP Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮЩҶШҜ Ш§ШІ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ ЩҫЫҢШ§Щ…вҖҢШұШіШ§ЩҶЫҢ ШЁЩҲЩ…ЫҢ Ш§ШЁШұЫҢ ШЁШұШ§ЫҢ Ш§ШұШӘШЁШ§Ш· ЩҶШ§ЩҮЩ…ШІЩ…Ш§ЩҶШҢ Щ…Ш№Щ…Ш§ШұЫҢвҖҢЩҮШ§ЫҢ ШұЩҲЫҢШҜШ§ШҜЩ…ШӯЩҲШұ ЩҲ ЩҫШұШҜШ§ШІШҙ ШӘЩҲШІЫҢШ№вҖҢШҙШҜЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶШҜ.

Ш§ЫҢЩҶ ШҜШұШі ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢвҖҢЩҮШ§ЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ ШЁШұ Ш§ШіШ§Ші Ш¬ШҜЫҢШҜШӘШұЫҢЩҶ Щ…ШҙШ®ШөШ§ШӘ MCP (2025-11-25)ШҢ ШіШұЩҲЫҢШівҖҢЩҮШ§ЫҢ ЩҫЫҢШ§Щ…вҖҢШұШіШ§ЩҶЫҢ Azure ЩҲ Ш§Щ„ЪҜЩҲЩҮШ§ЫҢ ЫҢЪ©ЩҫШ§ШұЪҶЩҮвҖҢШіШ§ШІЫҢ ШіШ§ШІЩ…Ш§ЩҶЫҢ ШӘШ«ШЁЫҢШӘвҖҢШҙШҜЩҮ ШұШ§ ШЁШұШұШіЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ.

### **Щ…Ш№Щ…Ш§ШұЫҢ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ MCP**

**Ш§ШІ Щ…ШҙШ®ШөШ§ШӘ MCP (2025-11-25):**

- **ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„вҖҢЩҮШ§ЫҢ Ш§ШіШӘШ§ЩҶШҜШ§ШұШҜ**: stdio (ШӘЩҲШөЫҢЩҮвҖҢШҙШҜЩҮ)ШҢ HTTP streaming (ШЁШұШ§ЫҢ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ШҜЩҲШұШҜШіШӘ)
- **ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„вҖҢЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ**: ЩҮШұ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ЫҢ Ъ©ЩҮ ЩҫШұЩҲШӘЪ©Щ„ ШӘШЁШ§ШҜЩ„ ЩҫЫҢШ§Щ… MCP ШұШ§ ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ Ъ©ЩҶШҜ
- **ЩӮШ§Щ„ШЁ ЩҫЫҢШ§Щ…**: JSON-RPC 2.0 ШЁШ§ Ш§ЩҒШІЩҲЩҶЩҮвҖҢЩҮШ§ЫҢ Ш®Ш§Шө MCP
- **Ш§ШұШӘШЁШ§Ш· ШҜЩҲШ·ШұЩҒЩҮ**: Ш§ШұШӘШЁШ§Ш· ШҜЩҲШ·ШұЩҒЩҮ Ъ©Ш§Щ…Щ„ ШЁШұШ§ЫҢ Ш§Ш·Щ„Ш§Ш№вҖҢШұШіШ§ЩҶЫҢ ЩҲ ЩҫШ§ШіШ®вҖҢЩҮШ§ Ш§Щ„ШІШ§Щ…ЫҢ Ш§ШіШӘ

## Ш§ЩҮШҜШ§ЩҒ ЫҢШ§ШҜЪҜЫҢШұЫҢ

ШҜШұ ЩҫШ§ЫҢШ§ЩҶ Ш§ЫҢЩҶ ШҜШұШі ЩҫЫҢШҙШұЩҒШӘЩҮШҢ ШҙЩ…Ш§ ЩӮШ§ШҜШұ Ш®ЩҲШ§ЩҮЫҢШҜ ШЁЩҲШҜ:

- **ШҜШұЪ© Ш§Щ„ШІШ§Щ…Ш§ШӘ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ ШіЩҒШ§ШұШҙЫҢ**: ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ЩҫШұЩҲШӘЪ©Щ„ MCP ШұЩҲЫҢ ЩҮШұ Щ„Ш§ЫҢЩҮ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ ШҜШұ ШӯШ§Щ„ЫҢ Ъ©ЩҮ Ш§ЩҶШ·ШЁШ§ЩӮ ШӯЩҒШё ШҙЩҲШҜ
- **ШіШ§Ш®ШӘ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ Azure Event Grid**: Ш§ЫҢШ¬Ш§ШҜ ШіШұЩҲШұЩҮШ§ЫҢ MCP ШұЩҲЫҢШҜШ§ШҜЩ…ШӯЩҲШұ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Azure Event Grid ШЁШұШ§ЫҢ Щ…ЩӮЫҢШ§ШівҖҢЩҫШ°ЫҢШұЫҢ ШЁШҜЩҲЩҶ ШіШұЩҲШұ
- **ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ Azure Event Hubs**: Ш·ШұШ§ШӯЫҢ ШұШ§ЩҮвҖҢШӯЩ„вҖҢЩҮШ§ЫҢ MCP ШЁШ§ ЩҶШұШ® ШҜШ§ШҜЩҮ ШЁШ§Щ„Ш§ ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ Azure Event Hubs ШЁШұШ§ЫҢ Ш§ШіШӘШұЫҢЩ…ЫҢЩҶЪҜ ШҜШұ ШІЩ…Ш§ЩҶ ЩҲШ§ЩӮШ№ЫҢ
- **Ъ©Ш§ШұШЁШұШҜ Ш§Щ„ЪҜЩҲЩҮШ§ЫҢ ШіШ§ШІЩ…Ш§ЩҶЫҢ**: ЫҢЪ©ЩҫШ§ШұЪҶЩҮвҖҢШіШ§ШІЫҢ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„вҖҢЩҮШ§ЫҢ ШіЩҒШ§ШұШҙЫҢ ШЁШ§ ШІЫҢШұШіШ§Ш®ШӘвҖҢЩҮШ§ ЩҲ Щ…ШҜЩ„вҖҢЩҮШ§ЫҢ Ш§Щ…ЩҶЫҢШӘЫҢ Щ…ЩҲШ¬ЩҲШҜ Azure
- **Щ…ШҜЫҢШұЫҢШӘ ЩӮШ§ШЁЩ„ЫҢШӘ Ш§Ш·Щ…ЫҢЩҶШ§ЩҶ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„**: ЩҫЫҢШ§ШҜЩҮвҖҢШіШ§ШІЫҢ ШҜЩҲШ§Щ… ЩҫЫҢШ§Щ…ШҢ ШӘШұШӘЫҢШЁ ЩҲ Щ…ШҜЫҢШұЫҢШӘ Ш®Ш·Ш§ ШЁШұШ§ЫҢ ШіЩҶШ§ШұЫҢЩҲЩҮШ§ЫҢ ШіШ§ШІЩ…Ш§ЩҶЫҢ
- **ШЁЩҮЫҢЩҶЩҮвҖҢШіШ§ШІЫҢ Ш№Щ…Щ„Ъ©ШұШҜ**: Ш·ШұШ§ШӯЫҢ ШұШ§ЩҮвҖҢШӯЩ„вҖҢЩҮШ§ЫҢ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ ШЁШұШ§ЫҢ ЩҶЫҢШ§ШІЩҮШ§ЫҢ Щ…ЩӮЫҢШ§ШіШҢ ШӘШЈШ®ЫҢШұ ЩҲ ШӘЩҲШ§ЩҶ Ш№Щ…Щ„ЫҢШ§ШӘЫҢ

```csharp
public class BatchingEventGridTransport : IMcpTransport
{
    private readonly List<McpMessage> _messageBuffer = new();
    private readonly Timer _flushTimer;
    private const int MaxBatchSize = 100;
    
    public async Task SendMessageAsync(McpMessage message)
    {
        lock (_messageBuffer)
        {
            _messageBuffer.Add(message);
            
            if (_messageBuffer.Count >= MaxBatchSize)
            {
                _ = Task.Run(FlushMessages);
            }
        }
    }
    
    private async Task FlushMessages()
    {
        List<McpMessage> toSend;
        lock (_messageBuffer)
        {
            toSend = new List<McpMessage>(_messageBuffer);
            _messageBuffer.Clear();
        }
        
        if (toSend.Any())
        {
            var events = toSend.Select(CreateEventGridEvent);
            await _publisher.SendEventsAsync(events);
        }
    }
}
```
