# Запуск прикладу

Тут ми припускаємо, що у вас уже є робочий код сервера. Будь ласка, знайдіть сервер з однієї з попередніх глав.

## Налаштування mcp.json

Ось файл, який ви використовуєте для довідки, [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json).

Змініть запис сервера за потреби, вказуючи абсолютний шлях до вашого сервера, включно з повною необхідною командою для запуску.

У прикладі файлу, на який посилаються вище, запис сервера виглядає так:

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

Можливо, вам доведеться вказати корінь репозиторію GitHub, який можна отримати за командою `git rev-parse --show-toplevel`.

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

Це відповідає запуску команди на кшталт: `node build/index.js`.

- Змініть цей запис сервера відповідно до місця розташування вашого серверного файлу або того, що потрібно для запуску сервера залежно від вашого вибраного середовища виконання та розташування сервера.

## Використання функцій сервера

- Натисніть на іконку `play`, як тільки ви додасте *mcp.json* у папку *./vscode*,

    Спостерігайте, як іконка інструментів зміниться, збільшуючи кількість доступних інструментів. Іконка інструментів розташована безпосередньо над полем для чату в GitHub Copilot.

## Запуск інструменту

- Введіть запит у вікні чату, який відповідає опису вашого інструменту. Наприклад, щоб викликати інструмент `add`, введіть щось на кшталт "add 3 to 20".

    Ви маєте побачити інструмент, який з’явиться над полем введення тексту чату, пропонуючи вибрати запуск інструменту, як на цьому зображенні:

    ![VS Code indicating it wanting to run a tool](../../../../../translated_images/uk/vscode-agent.d5a0e0b897331060.webp)

    Вибір інструменту має дати числовий результат із значенням "23", якщо ваш запит був, як ми згадували раніше.
