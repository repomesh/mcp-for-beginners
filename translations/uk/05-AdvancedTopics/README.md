# Розширені теми в MCP

[![Розширений MCP: Безпечні, масштабовані та мультимодальні AI-агенти](../../../translated_images/uk/06.42259eaf91fccfc6.webp)](https://youtu.be/4yjmGvJzYdY)

_(Натисніть на зображення вище, щоб переглянути відеоурок)_

Цей розділ охоплює серію розширених тем з реалізації Протоколу Контексту Моделі (MCP), включаючи мультимодальну інтеграцію, масштабованість, найкращі практики безпеки та інтеграцію для підприємств. Ці теми є вирішальними для побудови надійних і готових до виробництва додатків MCP, які можуть відповідати вимогам сучасних AI-систем.

## Огляд

Цей урок досліджує розширені концепції реалізації Протоколу Контексту Моделі, зосереджуючись на мультимодальній інтеграції, масштабованості, найкращих практиках безпеки та інтеграції для підприємств. Ці теми є необхідними для створення додатків MCP виробничого рівня, які можуть впоратися зі складними вимогами в корпоративному середовищі.

> **Дивлячись вперед:** кілька тем нижче залежать від кандидата у реліз специфікації MCP від `2026-07-28` — Кореневі контексти (5.4) та Відбір (5.6) базуються на примітивах, які кандидат у реліз позначає як застарілі, а експериментальна функція Завдань, згадана у Особливостях Протоколу (5.16), переходить до окремого розширення Завдань. Деталі дивіться у [Що змінюється у MCP: Кандидат у реліз 2026-07-28](../01-CoreConcepts/mcp-2026-07-28-release-candidate.md).

## Навчальні цілі

Після цього уроку ви зможете:

- Реалізувати мультимодальні можливості в рамках MCP
- Проєктувати масштабовані архітектури MCP для сценаріїв з високим навантаженням
- Застосовувати найкращі практики безпеки відповідно до принципів безпеки MCP
- Інтегрувати MCP з корпоративними AI-системами та фреймворками
- Оптимізувати продуктивність та надійність у виробничих середовищах

## Ключові висновки

- Реалізації MCP з підтримкою мульти-модальності розширюють можливості AI за межі обробки тексту
- Масштабованість є критичною для впроваджень у підприємствах і може бути досягнута горизонтальним та вертикальним масштабуванням
- Комплексні заходи безпеки захищають дані та забезпечують належний контроль доступу
- Інтеграція з корпоративними платформами, такими як Azure OpenAI та Microsoft AI Foundry, підсилює можливості MCP
- Розширені реалізації MCP виграють від оптимізованих архітектур та ретельного управління ресурсами

## Вправа

Спроєктуйте реалізацію MCP корпоративного рівня для конкретного випадку використання:

1. Визначте мультимодальні вимоги для вашого випадку використання
2. Окресліть заходи безпеки для захисту чутливих даних
3. Розробіть масштабовану архітектуру, здатну впоратися з різним навантаженням
4. Сплануйте точки інтеграції з корпоративними AI-системами
5. Документуйте потенційні вузькі місця продуктивності та стратегії їх усунення
