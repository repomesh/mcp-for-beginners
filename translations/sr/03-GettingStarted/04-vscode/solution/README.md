# Покретање примера

Овде претпостављамо да већ имате радни серверски код. Молимо вас да пронађете сервер из једног од претходних поглавља.

## Подешавање mcp.json

Ево датотеке коју користите као референцу, [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json).

Промените унос сервера према потреби како бисте навели апсолутну путању до вашег сервера, укључујући потребну пуну команду за покретање.

У пример датотеци наведеном горе унос сервера изгледа овако:

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

Можда ћете морати да унесете корен GitHub репозиторијума, који можете добити командом `git rev-parse --show-toplevel`.

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

Ово одговара покретању команде као што је: `node build/index.js`.

- Промените овај унос сервера да одговара месту где се налази ваша серверска датотека или шта је потребно за покретање сервера у зависности од изабраног окружења и локације сервера.

## Користите функције у серверу

- Кликните на икону `play`, након што додате *mcp.json* у фасциклу *./vscode*,

    Посматрајте како се икона инструмената мења и повећава број доступних алата. Икона инструмената се налази непосредно изнад полиња за чет у GitHub Copilot-у.

## Покрените алат

- Унесите упит у прозор за ћаскање који одговара опису вашег алата. На пример, да бисте покренули алат `add`, унесите нешто попут "add 3 to 20".

    Требало би да видите алат приказан изнад текстуалног оквира за ћаскање који вам указује да изаберете да покренете алат као на овој слици:

    ![VS Code индикатор који жели да покрене алат](../../../../../translated_images/sr/vscode-agent.d5a0e0b897331060.webp)

    Избор алата треба да произведе нумерички резултат "23" ако је ваш упит био као што смо претходно навели.
