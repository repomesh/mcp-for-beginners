# бһҸбһҫбһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮбһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡвҖӢбһҖбҹ’бһ“бһ»бһ„ MCPбҹ– бһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҜбһҖбһҹбһ¶бһҡбһҖбһҖбҹӢбһ”бһүбҹ’бһ…бһ”бҹӢ бҹўбҹ бҹўбҹҰ-бҹ бҹ§-бҹўбҹЁ

> **бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бҹ–** бһҜбһҖбһҹбһ¶бһҡбһҖбһҖбҹӢбһ”бһүбҹ’бһ…бһ”бҹӢбҹ” бһӣбһҖбҹ’бһҒбһҺбҹҲбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ `2026-07-28` бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ“бҹ…бһ–бҹҒбһӣбһҹбһҡбһҹбҹҒбһҡбһ“бҹҒбҹҮбһ‘бҹҒбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҖбһ¶бһҹбһ“бҹ…бһҗбҹ’бһ„бҹғбһ‘бһё бҹўбҹЎ бһҒбҹӮбһ§бһҹбһ—бһ¶ бһҶбҹ’бһ“бһ¶бҹҶ бҹўбҹ бҹўбҹҰ бһ бһҫбһҷбһҳбһ¶бһ“бһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бһ…бҹҒбһүбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡвҖӢбһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘ бҹўбҹЁ бһҒбҹӮбһҖбһҖбҹ’бһҖбһҠбһ¶ бҹўбҹ бҹўбҹҰбҹ” бһўбҹ’бһңбһёбҹ—бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ–бһҺбҹҢбһ“бһ¶бһўбҹҶбһ–бһёбһҜбһҖбһҹбһ¶бһҡбһҖбһҖбҹӢбһ”бһүбҹ’бһ…бһ”бҹӢ; бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ [бһӣбһҖбҹ’бһҒбһҺбҹҲбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҒбһ¶бһ„бһҳбһ»бһҒ](https://modelcontextprotocol.io/specification/draft) бһ“бһ·бһ„ [бһ”бһүбҹ’бһҮбһёбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ](https://modelcontextprotocol.io/specification/draft/changelog) бһҡбһ”бһҹбҹӢбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҠбһ№бһ„бһ–бһёбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҳбһ»бһ“бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҖбһҹбһ¶бһ„бһҮбһ¶бһҳбһҪбһҷбһңбһ¶бҹ” бһҳбҹҒбһҡбҹҖбһ“бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ“бҹғбһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ‘бҹҶбһ“бһҫбһ”бһ”бҹҶбһ•бһ»бһҸ бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ MCP Specification 2025-11-25 бһ бһҫбһҷбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё `2026-07-28` бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбҹ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

`2026-07-28` бһҮбһ¶бһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ’бҹҶбһ”бҹҶбһ•бһ»бһҸбһ“бҹғ MCP бһ…бһ¶бһ”бҹӢбһҸбһ¶бҹҶбһ„бһ–бһёбһңбһ¶бһ”бҹ’бһҡбһҖбһ¶бһҹбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһҖбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ”бһҪбһ“бһҳбһҪбһҷ (SEPs) бһҠбҹӮбһӣбһҠбһҖбһ…бҹҒбһүбһҹбһ„бҹ’бһҒбҹҒбһ”бһҖбһҳбҹ’бһҡбһ·бһҸ protocol-level sessions бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһІбҹ’бһҷ MCP бһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹ…бһӣбһҫбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“бһ‘бҹҒ; бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҮбһ¶бһҸбһҳбҹ’бһӣбҹғбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһ“бҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ—, бһ бһҫбһҷбһӣбһҖбҹ’бһҒбһҺбҹҲбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһ–бһёбһҳбһ»бһ“ (Roots, Sampling, Logging) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҒбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ‘бҹҖбһҸбһҖбҹ’бһҡбҹ„бһҳбһӮбҹ„бһӣбһ“бһҷбҹ„бһ”бһ¶бһҷбһҮбһёбһңбһ…бһҖбҹ’бһҡбһҗбҹ’бһҳбһёбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҹбһ„бҹ’бһҒбҹҒбһ”бһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ, бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһңбһ¶бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ бһ“бһ·бһ„бһңбһ¶бһҳбһ¶бһ“бһ“бҹҗбһҷбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһ…бҹҶбһ–бҹ„бҹҮбһҖбһјбһҠбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҸбһ¶бһҳ `2025-11-25` бһҳбһ»бһ“бҹ”

бһ”бҹ’бһҡбһ—бһ–бҹ– [бһҜбһҖбһҹбһ¶бһҡбһҖбһҖбҹӢбһ”бһүбҹ’бһ…бһ”бҹӢ MCP бҹўбҹ бҹўбҹҰ-бҹ бҹ§-бҹўбҹЁ](https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/) (бһ”бҹ’бһӣбһҖбҹӢ Model Context Protocol, David Soria Parra бһ“бһ·бһ„ Den Delimarsky)бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–

- бһўбһ–бһ”бҹ’бһҡбһ¶бһ”бҹӢбһҗбһ¶бһ бҹҒбһҸбһ»бһўбҹ’бһңбһё MCP бһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бҹ…бһҮбһ¶бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– бһ“бһ·бһ„бһ”бһүбҹ’бһ бһ¶бһўбҹ’бһңбһёбһҠбҹӮбһӣбһңбһ¶бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҠбҹӮбһӣбһңбһ·бһӣбһ”бһүбҹ’бһ…бһјбһ“бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһ‘бһјбһӣбһ¶бһҷбҹ”
- бһ–бһҺбҹҢбһ“бһ¶бһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһҠбҹғ `initialize`/`initialized` бһ“бһ·бһ„бһҖбҹ’бһ”бһ¶бһӣ `Mcp-Session-Id` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҮбҹҶбһ“бһҪбһҹбҹ”
- бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ–бһёбһҖбҹ’бһ”бһ¶бһӣвҖӢбһҗбҹ’бһҳбһё `Mcp-Method` бһ“бһ·бһ„ `Mcp-Name` бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ metadata бһӣбһҳбҹ’бһўбһ·бһҸ `ttlMs`/`cacheScope`бҹ”
- бһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһҹбҹҶбһҺбһ»бҹҶбһ”бһҺбҹ’бһҺбһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ„бҹ’бһҡбһёбһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ–бһёбһҡбһҠбҹӮбһӣбһ”бһүбҹ’бһҮбһјбһ“бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһ“бҹҒбҹҮбҹ– MCP Apps бһ“бһ·бһ„ Tasksбҹ”
- бһҡбһ¶бһҷбһ“бһ¶бһҳбһ…бҹҶбһ“бһҪбһ“бһ”бҹ’бһҡбһ¶бҹҶбһҳбһҪбһҷ SEP бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҠбҹӮбһӣбһ–бһ„бҹ’бһҡбһ№бһ„бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ OAuth 2.0 / OIDCбҹ”
- бһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҹбҹ’бһ“бһјбһӣбһҺбһ¶бһҳбһҪбһҷ (Roots, Sampling, Logging) бһҠбҹӮбһӣбһҘбһЎбһјбһңбһ“бҹҒбҹҮбһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһҳбһ¶бһ“бһ“бҹҗбһҷбһўбҹ’бһңбһёбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ”
- бһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ Full JSON Schema 2020-12 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ `inputSchema`/`outputSchema`бҹ”

## бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–

бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ…бһҳбҹ’бһ”бһ„бҹ– MCP бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹ…бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢ protocolбҹ”

### бһҳбһ»бһ“ (2025-11-25): бһҹбһҳбҹҗбһҷбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҳбһҪбһҷ

бһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбһ¶бһҳ Streamable HTTP бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһҠбҹғ `initialize`бҹ” бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹ„бһҷбһҖбҹ’бһ”бһ¶бһӣ `Mcp-Session-Id` бһҠбҹӮбһӣбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҡбһҹбһ»бҹҶбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ—бһӮбҹ’бһҡбһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбҹ’бһҡбһјбһңбһҷбһҖбһ‘бҹ…бһҠбһ¶бһҖбҹӢбҹ–

### бһҳбһ¶бһ“бһ—бһ¶бһ–бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ (Routable), бһўбһ¶бһ…бһ•бҹ’бһ‘бһ»бһҖбһҮбһ¶бһҹбһ¶бһҡ (cacheable), бһ бһҫбһҷбһўбһ¶бһ…бһҸбһ¶бһҳбһҠбһ¶бһ“ (traceable)

бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһјбһ…бһ”бһёбһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ…бһҡбһ“бҹ’бһҸбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ:

- **бһҖбҹ’бһ”бһ¶бһӣ `Mcp-Method` бһ“бһ·бһ„ `Mcp-Name` бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһ¶бһ“бһ“бҹ… Streamable HTTP** ([SEP-2243](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/2243)) бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷ load balancers, gateways бһ“бһ·бһ„ rate limiters бһўбһ¶бһ…бһ”бһүбҹ’бһҮбһјбһ“бһҹбһ¶бһҡбһ”бһ¶бһ“бһҠбҹ„бһҷбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҳбһҫбһӣ JSON bodyбҹ” бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ”бһҠбһ·бһҹбҹҒбһ’бһҹбҹҶбһҺбһҫбһҡвҖӢбһҠбҹӮбһӣбһҖбҹ’бһ”бһ¶бһӣбһ“бһ·бһ„бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһӮбҹ’бһ“бһ¶бҹ”
- **`tools/list` бһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣ resource read бһҳбһ¶бһ“ `ttlMs` бһ“бһ·бһ„ `cacheScope`** ([SEP-2549](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/2549)) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҶбһҡбһјбһӣбһҫ HTTP `Cache-Control`бҹ” бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҠбһ№бһ„бһҗбһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣбһ”бһүбҹ’бһҮбһёбһҗбҹ’бһҳбһёбһҡбһҷбҹҲбһ–бҹҒбһӣбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“ бһ бһҫбһҷбһңбһ¶бһўбһ¶бһ…бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ‘бҹ…бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ•бҹ’бһҹбһ¶бһҡбһ“бҹ…бһӣбһҫ stream SSE бһҮбһ¶бһҷбһјбһҡбҹ”
- **бһҖбһ¶бһҡбһ”бһҳбҹ’бһҡбҹӮбһ”бһҳбҹ’бһҡбһҪбһӣбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҹбҹ’бһ“бһ¶бһҠбҹғ W3C бһ“бҹ… `_meta` бһ”бһ¶бһ“бһҜбһҖбһҹбһ¶бһҡ** ([SEP-414](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/414)) бһҖбҹӮбһӣбһҳбҹ’бһўбһҲбҹ’бһҳбҹ„бҹҮ key `traceparent`, `tracestate`, бһ“бһ·бһ„ `baggage` бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ бҹ…бһҶбҹ’бһӣбһ„бһҸбһ¶бһҳ client SDK, MCP server бһ“бһ·бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбҹ’бһҡбҹ„бһҳбһҖбҹ’бһ“бһ»бһ„ backend бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡ [OpenTelemetry](https://opentelemetry.io/)бҹ”

## бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҳбһҪбһҷбһҸбһҳбҹ’бһӣбҹғбһҖбҹ’бһҡбһҳбһҒбһҺбҹ’бһҢбһ–бһёбһҡбһҠбһ„

бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҳбһ¶бһ“бһҡбһҪбһ…бһҳбһҖ informally бһ“бҹ… `2025-11-25`бҹ” [SEP-2133](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/2133) бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ–бһҪбһҖбһңбһ¶бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһЎбһҫбһ„бҹ–

- бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҠбҹ„бһҷ ID reverse-DNSбҹ”
- бһ–бһҪбһҖбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһҡбһ…бһ¶бһҡбһҸбһ¶бһҳбһҡбһҷбҹҲбһ•бҹӮбһ“бһ‘бһё `extensions` бһӣбһҫбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– client бһ“бһ·бһ„ serverбҹ”
- бһ–бһҪбһҖбһңбһ¶бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣ ext-* бһ”бҹ’бһҡбһ—бһ–бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһҳбһ¶бһ“бһ”бҹҶбһҺбһ„бһҮбҹҶбһ“бһ¶бһ“бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһ–бһё core specificationбҹ”
- бһҸбһ¶бһҳбһҡбһҷбҹҲбһ•бҹ’бһӣбһјбһң SEP бһҳбһ¶бһ“ бһ•бҹ’бһ‘бһ¶бҹҶбһ„ Extensions бһҗбҹ’бһҳбһё бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ•бҹ’бһӣбһјбһңбһ–бһёбһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ‘бҹ…бһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбһ•бҹ’бһӣбһјбһңбҹ”

бһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһ“бҹҒбҹҮбһ•бҹ’бһүбһҫбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ–бһёбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бҹ”

### MCP Appsбҹ– бһҳбһ»бһҒбһ„бһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ”бһ„бҹ’бһ бһ¶бһү

[MCP Apps](https://blog.modelcontextprotocol.io/posts/2026-01-26-mcp-apps/) ([SEP-1865](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/1865)) бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ•бҹ’бһүбһҫбһ…бҹҶбһ“бһ»бһ…бһ•бҹ’бһ‘бһ¶бҹҶбһ„ HTML бһҳбһҪбһҷбһҠбҹӮбһӣбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ•бҹ’бһ‘бҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ iframe бһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ¶бһ“бһ”бҹ’бһҡбһҖбһ¶бһҹ UI template бһҳбһ»бһ“бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһҳбҹ’бһ…бһ¶бһҹбҹӢбһ•бҹ’бһ‘бҹҮбһўбһ¶бһ… prefetch, cache бһ“бһ·бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ»бһ“бһ–бҹҒбһӣбһўбҹ’бһңбһёбҹ—бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹҒбҹҮбһҡбһҪбһ…бһ“бҹ… [бһҳбҹҒбһҡбҹҖбһ“ 15: MCP Apps](../03-GettingStarted/15-mcp-apps/README.md) вҖ” бһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһҺбһ»бҹҶбһ”бһҺбҹ’бһҺ Extensions, MCP Apps бһҘбһЎбһјбһңбһҮбһ¶бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһӮбһ“бҹ’бһӣбһ„бһҹбҹҶбһҡбһ¶бһ”бҹӢбһҖбһјбһҡбҹ”

### Tasks бһЎбһҫбһ„бһҮбһ¶бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖ

Tasks бһ”бһ¶бһ“бһҠбһ№бһҖбһ“бһ¶бҹҶбһҠбҹ„бһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ core бһ“бҹ… `2025-11-25`бҹ” бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҗбҹ’бһҳбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ·бһҷбһ¶бһӣбҹҗбһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҳбһҪбһҷбҹ– [бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖ Tasks](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/2663) бһ”бҹ’бһҠбһјбһҡбһҮбһёбһңбһ…бһҖбҹ’бһҡбһҮбһ»бҹҶбһңбһ·бһүбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– вҖ” бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһўбһ¶бһ…бһҶбҹ’бһӣбһҫбһҷбһҸбһ” `tools/call` бһҮбһ¶бһҳбһҪбһҷбһӣбҹҒбһҒбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ task бһ“бһ·бһ„бһўбһҸбһ·бһҗбһ·бһҮбһ“бһ“бһ¶бҹҶбһҳбһ»бһҒбһңбһ¶бһҮбһ¶бһҳбһҪбһҷ `tasks/get`, `tasks/update`, бһ“бһ·бһ„ `tasks/cancel`бҹ” бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ task бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбҹ– бһўбһҸбһ·бһҗбһ·бһҮбһ“бһ”бҹ’бһҡбһҖбһ¶бһҹбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖ бһ бһҫбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҹбһҳбҹ’бһҡбҹҒбһ…бһҗбһ¶бһҸбһҫбһӮбһҪбһҡбһўбҹ„бһҷбһ бҹ…бһҳбһҪбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҮбһ¶ task бһ¬бһ‘бҹҒбҹ” `tasks/list` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһҖбһ…бҹҒбһүбһҹбһҡбһ»бһ” бһҠбҹ„бһҷбһҹбһ¶бһҡбһңбһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҹбһҳбҹҗбһҷбһ‘бҹҒбҹ”

> **бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ¶бҹҶбһ”бһҳбҹ’бһӣбҹӮбһ„бҹ–** бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸ API Tasks бһӮбһ“бҹ’бһӣбһ„бһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ `2025-11-25`, бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһ”бһҳбҹ’бһӣбҹӮбһ„бһ‘бҹ…бһҮбһёбһңбһ…бһҖбҹ’бһҡвҖӢбһ”бҹҶбһ•бһ»бһҸбһҗбҹ’бһҳбһё вҖ” бһңбһ¶бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһӮбҹ’бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһ»бһ“бһ‘бҹҒбҹ”

## Roots, Sampling бһ“бһ·бһ„ Logging бһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ‘бҹҖбһҸ

бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ [бһӮбҹ„бһӣбһ“бһҷбҹ„бһ”бһ¶бһҷбһҮбһёбһңбһ…бһҖбҹ’бһҡ (feature lifecycle policy)](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/2577) ([SEP-2577](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/2577)), бһ–бһёбһҡбһ”бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһ…бһҳбҹ’бһ”бһ„бһҡбһ”бһҹбҹӢ client бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһ“бҹ… [Core Concepts](./README.md#roots) бһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ‘бҹ…бһҮбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– **Deprecated**бҹ–

| бһҳбһ»бһҒбһ„бһ¶бһҡ | бһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҹбһҠбҹӮбһӣбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶ |
|---|---|
| Roots | бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ§бһ”бһҖбһҡбһҺбҹҚ, URI бһ’бһ“бһ’бһ¶бһ“, бһ¬бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ |
| Sampling | бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҮбһ¶бһҳбһҪбһҷ API бһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢ LLM |
| Logging | `stderr` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бһҡбһ“бҹ’бһҸ stdio; OpenTelemetry бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҳбһҫбһӣбһғбһҫбһүбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ |

бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ– бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ, бһ”бҹ’бһҡбһ—бҹҒбһ‘, бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ“бҹ…бһ”бһ“бҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбҹӮбһ“бҹҒбҹҮ бһ“бһ·бһ„бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһ•бҹ’бһҹбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһҳбһҪбһҷбһҶбҹ’бһ“бһ¶бҹҶбһҖбҹ’бһҡбҹ„бһҷбһҳбһҖбҹ” бһҖбһ¶бһҡбһҠбһҖбһ…бҹҒбһүбһ–бһҪбһҖбһңбһ¶бһ…бҹҒбһүбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһҸбһҳбҹ’бһҡбһјбһңбһІбҹ’бһҷбһҳбһ¶бһ“ SEP бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҖбҹ’бһҡбҹ„бһҳбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҮбһёбһңбһ…бһҖбҹ’бһҡ вҖ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҳбһ·бһ“бһҳбһ¶бһ“бһўбҹ’бһңбһёбһҒбһјбһ…бһҒбһ¶бһҸбһҖбҹ’бһ“бһ»бһ„бһӮбҹҶбһҡбһј [Sampling](../03-GettingStarted/14-sampling/README.md) бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҗбҹ’бһ„бҹғбһ“бҹҒбҹҮбһ‘бҹҒ бһҸбҹӮбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҗбҹ’бһҳбһёбһӮбһҪбһҡбһҸбҹӮбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһӮбҹҶбһҡбһјбһҮбҹҶбһ“бһҪбһҹбһҒбһ¶бһ„бһӣбһҫбҹ”

## Full JSON Schema 2020-12 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ

бһ§бһ”бһҖбһҡбһҺбҹҚ `inputSchema` бһ“бһ·бһ„ `outputSchema` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӣбһҫбһҖбһЎбһҫбһ„бһ‘бҹ…бһҮбһ¶бһҖбҹҶбһҺбҹӮбһ–бҹҒбһү [JSON Schema 2020-12](https://json-schema.org/draft/2020-12) ([SEP-2106](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/2106))бҹ–

- бһҹбҹ’бһҖбҹҒбһҳ input бһ“бҹ…бһҸбҹӮбһҡбһҖбҹ’бһҹбһ¶ `type: "object"` бһҮбһ¶бһҡбҹҶбһҹбһ¶бһӣбһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҘбһЎбһјбһңбһ“бҹҒбҹҮбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ composition (`oneOf`, `anyOf`, `allOf`), conditionals бһ“бһ·бһ„ references (`$ref`, `$defs`) бһ”бһ¶бһ“бҹ”
- бһҹбҹ’бһҖбҹҒбһҳ output бһӮбҹ’бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„ `structuredContent` бһҘбһЎбһјбһңбһўбһ¶бһ…бһҮбһ¶бһҸбһҳбҹ’бһӣбҹғ JSON бһҳбһҪбһҷбһҺбһ¶бһҷбһҖбһӣбһҫбһҹбһ–бһёбһҸбҹӮбһҮбһ¶бһңбһҸбҹ’бһҗбһ»бҹ”
- бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбҹ’бһҡбһјбһңбһҳбһ·бһ“бһ”бҹҶбһ”бҹӮбһҖ `$ref` URI бһ—бһ¶бһӮбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҠбһ¶бһҖбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҮбҹҶбһ бһ¶бһ“бһҮбһ¶бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ бһ“бһ·бһ„бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ (бһӮбһ·бһҸбһ–бһёбһҖбһ¶бһҡбһ”бһҠбһ·бһҹбҹҒбһ’бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбҹ’бһҡбҹ„бһҳбһҖбһ¶бһҡбһӮбһ·бһҸбһ–бһёбһ”бһүбҹ’бһ бһ¶бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–)бҹ”

бһ•бҹ’бһ“бҹӮбһҖбһ”бһүбҹ’бһ бһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҡбһјбһ”бһ·бһҷбһ”бҹҗбһҺбҹ’бһҺбһҖбһјбһҠбһҖбҹҶбһ бһ»бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бһ“бһ’бһ¶бһ“бһҒбҹ’бһңбҹҮбһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ–бһё MCP-custom `-32002` бһ‘бҹ…бһҮбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ JSON-RPC бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶ `-32602` (Invalid Params) ([SEP-2164](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/2164))бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ‘бҹ…бһӣбһҫбһҸбһҳбҹ’бһӣбҹғ `-32002` бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһңбһ¶бҹ”

## бһҡбһ”бҹҖбһ” protocol бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ–бһёбһ‘бһёбһ“бҹҒбҹҮбһ‘бҹ…

бһҖбһ¶бһҡбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ”бһҳбҹ’бһӣбҹӮбһ„бһ”бһ„бҹ’бһҖбһҖ, бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶ MCP бһҳбһ·бһ“бһҳбһ¶бһ“бһӮбһҳбҹ’бһҡбҹ„бһ„бһІбҹ’бһҷбһңбһ¶бһҮбһ¶бһҡбһҝбһ„бһ’бһҳбҹ’бһҳбһҸбһ¶бһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбһ‘бҹҒбҹ” бһ”бһё SEP бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ”бһ„бҹ’бһҖбһ¶бһҡбһІбҹ’бһҷбһңбһ¶бһ”бһ“бҹ’бһҸбһҠбҹӮбһҡбҹ–

- **бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҮбһёбһңбһ…бһҖбҹ’бһҡ** бһ•бҹ’бһҸбһӣбҹӢбһ•бҹ’бһӣбһјбһңбһҳбһ»бһҒбһҮбһёбһңбҹ– бһ–бһё Active вҶ’ Deprecated вҶ’ Removed бһҮбһ¶бһҳбһҪбһҷбһҡбһҷбҹҲбһ–бҹҒбһӣбһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢ бҹЎбҹў бһҒбҹӮбһ…бһ“бҹ’бһӣбҹ„бҹҮбһ–бһёбһҖбһ¶бһҡбһҮбһ·бһҸбһўбһҸбһ·бһ•бһҡбһҺбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһҖбһ…бҹҒбһүбһ–бҹ’бһҷбһҪбһҡбһ–бҹҒбһӣбһ”бһ¶бһ“бһ”бҹҶбһ•бһ»бһҸбҹ”
- **бһҹбҹҶбһҺбһ»бҹҶбһ”бһҺбҹ’бһҺ Extensions** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһ•бҹ’бһҹбһ¶бһҡбһҮбһ¶бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ бһ“бһ·бһ„бһҳбһ¶бһ“бһҹбҹ’бһҗбһ·бһҡбһ—бһ¶бһ–бһ“бҹ…бһ‘бһёбһ“бҹ„бҹҮ бһҳбһ»бһ“бһ–бҹҒбһӣ (бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“) бһ•бҹ’бһӣбһ¶бһҹбҹӢбһІбҹ’бһҷбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһҡбһ”бһҹбҹӢ core specificationбҹ”

- SEP бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҸбһ¶бһҳбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ бһҳбһ·бһ“бһўбһ¶бһ…бһ‘бҹ…бһҠбһӣбҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһЎбһҫбһҷ бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҳбһ¶бһ“бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһ”бһҡбһ·бһҷбһ¶бһҷбһ—бһ¶бһ–бһҶбҹ’бһӣбһҫбһҷбһҶбҹ’бһӣбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„ [conformance suite](https://github.com/modelcontextprotocol/conformance) ([SEP-2484](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/2484)) вҖ” бһҮбһ¶бһҹбҹҠбһ»бһҳбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷ [SDK tier system](https://github.com/modelcontextprotocol/modelcontextprotocol/pull/1777) бһҠбҹӮбһӣбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғ SDKs бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбҹ”

## бһҖбһ¶бһӣбһңбһ·бһ—бһ¶бһӮбһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷ бһ“бһ·бһ„ бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ

- бһҖбҹҶбһҺбҹӮбһ”бҹҒбһҖбҹ’бһҒбһҮбһ“бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ¶бһҖбҹӢбһҒбҹ’бһ‘бһ”бҹӢбһ“бҹ…бһҗбҹ’бһ„бҹғбһ‘бһё 21 бһҒбҹӮбһ§бһҹбһ—бһ¶ бһҶбҹ’бһ“бһ¶бҹҶ 2026бҹ”
- бһӣбһҖбҹ’бһҒбһҺбҹҲбһ”бһүбҹ’бһ…бһ”бҹӢбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ“бҹ…бһҗбҹ’бһ„бҹғбһ‘бһё 28 бһҒбҹӮбһҖбһҖбҹ’бһҖбһҠбһ¶ бһҶбҹ’бһ“бһ¶бҹҶ 2026бҹ”
- бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бҹЎбҹ бһҹбһ”бҹ’бһҸбһ¶бһ бҹҚбһ…бһ“бҹ’бһӣбҹ„бҹҮбһ–бһёбһ–бһёбһҡбһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҗбҹӮбһ‘бһ¶бҹҶ SDK бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһҸбһ·бһҗбһ·бһҮбһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ‘бһӣбҹӢбһ“бһ№бһ„бһ”бҹҶбһҺбһҫбһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ; Tier 1 SDKs бһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбҹҶбһ–бһ№бһ„бһҗбһ¶бһ“бһ№бһ„бһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһ“бҹҒбҹҮбһҖбҹ’бһҡбҹ„бһҳ [SDK tier system](https://modelcontextprotocol.io/docs/sdk)бҹ”
- бһҸбһ¶бһҳбһҠбһ¶бһ“бһҹбҹҶбһҺбһ»бҹҶбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһ“бҹ… [draft specification](https://modelcontextprotocol.io/specification/draft) бһ“бһ·бһ„ [changelog](https://modelcontextprotocol.io/specification/draft/changelog)бҹ”

## бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ

бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһҡбһ бһјбһҸбһҳбһҖбһҠбһӣбҹӢбһ–бҹҒбһӣбһ“бҹҒбҹҮбһҖбҹ’бһ“бһ»бһ„бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮ бһӮбҹ„бһӣбһҠбҹ…бһ‘бҹ…бһӣбһҫ **2025-11-25** бһҠбҹӮбһӣбһ“бҹ…бһҸбҹӮбһҖбҹҶбһ–бһ»бһ„бһҮбһ¶бһӣбһҖбҹ’бһҒбһҺбҹҲбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбҹҒбһҡбһ–бҹҒбһӣбһ‘бһӣбҹӢбһҳбһ»бһ“ `2026-07-28` бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбҹ” бһҷбҹүбһ¶бһ„бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ–

- **бһҹбһҳбҹҗбһҷбһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„ бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһҠбҹғбһӮбһј `initialize`** (бһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҠбһ”бҹӢбһ“бҹ… [Core Concepts](./README.md) бһ“бһ·бһ„ [Lesson 6: HTTP Streaming](../03-GettingStarted/06-http-streaming/README.md)) бһ“бҹ…бһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹҒбһ…бһ„бҹӢбһұбҹ’бһҷбһҡбҹҶбһ–бһ№бһ„бһҗбһ¶бһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҮбҹҶбһ“бһҪбһҹбһҠбҹ„бһҷбһҳбҹүбһјбһҢбҹӮбһӣбһҹбҹҶбһҺбһҫбһҡ Stateless бһ“бҹ…бһӣбһҫ бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ‘бҹ…бһұбҹ’бһҷ SDKs бһҠбҹӮбһӣбһҶбһ”бһӮбҹ’бһ“бһ¶бһ“бһ№бһ„ `2026-07-28` бҹ”
- **бһҖбһ¶бһҡбһ…бҹӮбһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһ“бһ·бһ„ бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“** (бһҖбҹҸбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҠбһ”бҹӢбһ“бҹ… [Core Concepts](./README.md)) бһ“бҹ…бһҸбҹӮбһҖбһҫбһҸбһҳбһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ–бҹ’бһҡбһҳбһ¶бһ“бһҗбһ¶бһўбһ¶бһ…бһҳбһ·бһ“бһ”бҹ’бһҡбһҫбһ”бһ“бҹ’бһҸбһ‘бҹҖбһҸ вҖ” бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҗбҹ’бһҳбһёбһӮбһҪбһҡбһҸбҹӮбһҮбҹ’бһҡбһҫбһҹбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбҹӮбһӣбһҮбҹҶбһ“бһҪбһҹбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһ¶бһҷбһ”бһүбҹ’бһҮбһёбһҒбһ¶бһ„бһӣбһҫбҹ”
- **бһҳбһ»бһҒбһ„бһ¶бһҡ бһҖбһ¶бһҡбһ„бһ¶бһҡ Experimental** бһ”бҹ’бһҡбһҹбһ·бһ“бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫ бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҮбҹҶбһ“бһҪбһҷбһ•бҹ’бһ“бҹӮбһҖ Tasks бһ–бһёбһҡбһ”бҹҖбһ”бһҮбһёбһңбһ·бһҸбһҗбҹ’бһҳбһёбһҡбһ”бһҹбҹӢбһңбһ¶бҹ”
- **бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MCP** ([Lesson 15](../03-GettingStarted/15-mcp-apps/README.md)) бһҳбһ·бһ“бһҳбһ¶бһ“бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһ“бҹ„бҹҮбһ‘бҹҒ; бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҠбһ»бһҳбһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ“бҹғ Extensionsбҹ”
