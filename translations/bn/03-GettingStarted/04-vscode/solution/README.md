# নমুনা চালানো

এখানে আমরা ধরে নিচ্ছি যে আপনার কাছে ইতিমধ্যেই একটি কার্যকরী সার্ভার কোড রয়েছে। অনুগ্রহ করে আগের অধ্যায়ের কোনো একটি থেকে একটি সার্ভার সন্ধান করুন।

## mcp.json সেট আপ করা

এখানে একটি ফাইল দেওয়া হয়েছে যা আপনি রেফারেন্স হিসাবে ব্যবহার করবেন, [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json)।

আপনার সার্ভারের সম্পূর্ণ কমান্ড সহ সার্ভারের আপেক্ষিক পথ নির্দেশ করার জন্য সার্ভার এন্ট্রি প্রয়োজন অনুযায়ী পরিবর্তন করুন।

উপরে উল্লেখিত উদাহরণ ফাইলে সার্ভার এন্ট্রিটি এরূপ দেখায়:

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

আপনাকে হতে পারে GitHub রিপোজিটরির মূল ডিরেক্টরি লিখতে হবে, যা কমান্ড `git rev-parse --show-toplevel` থেকে পাওয়া যাবে।

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

এটি একটি কমান্ড চালানোর সমতুল্য যেমন: `node build/index.js`।

- আপনার সার্ভার ফাইল কোথায় অবস্থিত তার উপর ভিত্তি করে অথবা আপনার নির্বাচিত রuntime এবং সার্ভার অবস্থান অনুযায়ী আপনার সার্ভার চালু করার জন্য যা প্রয়োজন সেই অনুযায়ী এই সার্ভার এন্ট্রি পরিবর্তন করুন।

## সার্ভারের ফিচারগুলি ব্যবহার করুন

- একবার আপনি *mcp.json* কে *./vscode* ফোল্ডারে যোগ করলে `play` আইকনে ক্লিক করুন,

    দেখতে পাবেন টুলিং আইকন পরিবর্তিত হচ্ছে যা উপলব্ধ টুলগুলোর সংখ্যা বৃদ্ধি করছে। টুলিং আইকন GitHub Copilot এর চ্যাট ফিল্ডের ঠিক ওপরেই অবস্থিত।

## একটি টুল চালান

- আপনার চ্যাট উইন্ডোতে এমন একটি প্রম্পট টাইপ করুন যা আপনার টুলের বর্ণনার সাথে মেলে। উদাহরণস্বরূপ, টুল `add` চালানোর জন্য "add 3 to 20" টাইপ করুন।

    আপনি দেখতে পাবেন চ্যাট টেক্সট বক্সের উপরে একটি টুল উপস্থাপিত হচ্ছে যা আপনাকে টুলটি চালানোর জন্য নির্বাচন করতে বলছে, এই ভিজ্যুয়ালের মত:

    ![VS Code indicating it wanting to run a tool](../../../../../translated_images/bn/vscode-agent.d5a0e0b897331060.webp)

    টুলটি নির্বাচন করলে একটি সংখ্যাসূচক ফলাফল দেখাবে "23" যদি আপনার প্রম্পট উপরে যেমন বলা হয়েছে তেমনই থাকে।
