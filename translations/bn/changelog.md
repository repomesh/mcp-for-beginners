# পরিবর্তনলিপি: MCP ফর বিগিনার্স কারিকুলাম

এই ডকুমেন্টটি মডেল কনটেক্সট প্রোটোকল (MCP) ফর বিগিনার্স কারিকুলামে করা সমস্ত গুরুত্বপূর্ণ পরিবর্তনের রেকর্ড হিসেবে কাজ করে। পরিবর্তনগুলি উলটো কালানুক্রমিক ক্রমে (নতুন পরিবর্তনগুলি প্রথমে) নথিভুক্ত করা হয়েছে।

আসন্ন `২০২৬-০৭-২৮` MCP স্পেসিফিকেশন রিলিজ ক্যান্ডিডেটের (ঘোষণা ২১ মে, ২০২৬; চূড়ান্ত রিলিজ নির্ধারিত জুলাই ২৮, ২০২৬) কাভারেজ যোগ করা হয়েছে, যা [অফিসিয়াল ঘোষণা ব্লগ পোস্ট](https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/) থেকে সারসংক্ষেপ করা হয়েছে। কারিকুলামের ভিত্তি এখনও **MCP স্পেসিফিকেশন ২০২৫-১১-২৫** যা নতুন সংস্করণ চালু না হওয়া পর্যন্ত থাকবে, তাই এটি বর্তমান পাঠগুলির পুনঃলিখনের পরিবর্তে ভবিষ্যত নির্দেশিকা হিসেবে উপস্থাপন করা হয়েছে।

### MCP স্পেসিফিকেশন ২০২৫-১১-২৫ সম্মতি পর্যালোচনা

#### কেন্দ্রীয় ধারণা উন্নতি (01-CoreConcepts/)
- **নতুন ক্লায়েন্ট প্রিমিটিভ - রুট**: রুটস ক্লায়েন্ট প্রিমিটিভের ব্যাপক ডকুমেন্টেশন যোগ করা হয়েছে, যা সার্ভারকে ফাইলসিস্টেম সীমানা এবং অ্যাক্সেস অনুমতি বুঝতে সহায়তা করে
- **টুল এনোটেশনস**: টুলের আচরণগত এনোটেশনগুলি (`readOnlyHint`, `destructiveHint`) সংক্রান্ত ডকুমেন্টেশন সংযোজন, ভালো টুল এক্সিকিউশনের জন্য
- **সাম্পলিং এ টুল কল করা**: সাম্পলিং ডকুমেন্টেশনে আপডেট হয়েছে, যাতে আছে `tools` এবং `toolChoice` প্যারামিটার, মডেল-চালিত টুল কলের জন্য
- **URL মোড উদয়**: সার্ভার-আরম্ভিত বাহ্যিক ওয়েব ইন্টারঅ্যাকশনের জন্য URL ভিত্তিক উদয় সংক্রান্ত ডকুমেন্টেশন যোগ করা হয়েছে
- **টাস্ক (পরীক্ষামূলক)**: ট্যাস্ক ফিচারের জন্য নতুন সেকশন যোগ করা হয়েছে, যা স্থায়ী এক্সিকিউশন র‍্যাপার এবং বিলম্বিত ফলাফল আহরণের জন্য
- **আইকন সমর্থন**: টুলস, রিসোর্স, রিসোর্স টেমপ্লেটস এবং প্রম্পটস এখন অতিরিক্ত মেটাডেটা হিসেবে আইকন অন্তর্ভুক্ত করতে পারে

#### ডকুমেন্টেশন আপডেট
- **README.md**: MCP স্পেসিফিকেশন ২০২৫-১১-২৫ সংস্করণ রেফারেন্স এবং তারিখ-ভিত্তিক সংস্করণ ব্যাখ্যা যোগ করা হয়েছে
- **study_guide.md**: মূল ধারণা সেকশনে টাস্ক এবং টুল এনোটেশনস যুক্ত করে পাঠ্যক্রম মানচিত্র আপডেট; ডকুমেন্ট টাইমস্ট্যাম্প আপডেট

#### স্পেসিফিকেশন সম্মতি যাচাই
- **প্রোটোকল ভার্শন**: সমস্ত ডকুমেন্টেশন MCP স্পেসিফিকেশন ২০২৫-১১-২৫ বর্তমান রেফারেন্স নিশ্চিত
- **আর্কিটেকচার সঙ্গতি**: দুই স্তরের আর্কিটেকচার (ডেটা লেয়ার + ট্রান্সপোর্ট লেয়ার) ডকুমেন্টেশন যথার্থতা নিশ্চিত
- **প্রিমিটিভস ডকুমেন্টেশন**: সার্ভার প্রিমিটিভস (রিসোর্স, প্রম্পটস, টুলস) এবং ক্লায়েন্ট প্রিমিটিভস (সাম্পলিং, এলিসিটেশন, লগিং, রুটস) যাচাই
- **ট্রান্সপোর্ট মেকানিজম**: STDIO এবং স্ট্রিমেবল HTTP ট্রান্সপোর্ট ডকুমেন্টেশন যথার্থতা নিশ্চিত
- **নিরাপত্তা দিকনির্দেশনা**: MCP নিরাপত্তার সেরা অনুশীলন ডকুমেন্টেশনের সাথে সামঞ্জস্য নিশ্চিত

  - Azure সিকিউরিটি এবং কমপ্লায়েন্স ফ্রেমওয়ার্কসমূহ
- **প্রয়োগ রিসোর্সেস**: সমস্ত প্রয়োগ গাইড লিঙ্ক এবং রিসোর্স যাচাই
  - Azure API ব্যবস্থাপনা প্রমাণীকরণ প্যাটার্নসমূহ

## অক্টোবর ৬, ২০২৫

### শুরু বিভাগ সম্প্রসারণ – উন্নত সার্ভার ব্যবহার এবং সাধারণ প্রমাণীকরণ

#### উন্নত সার্ভার ব্যবহার (03-GettingStarted/10-advanced)
- **নতুন অধ্যায় যোগ করা হয়েছে**: উন্নত MCP সার্ভার ব্যবহারের ব্যাপক গাইড, যেখানে নিয়মিত এবং নিম্নস্তরের উভয় সার্ভার আর্কিটেকচার কভার করা হয়েছে।
  - **নিয়মিত বনাম নেভ-লেভেল সার্ভার**: উভয় পদ্ধতির জন্য পাইথন এবং টাইপস্ক্রিপ্ট কোড উদাহরণসহ বিস্তারিত তুলনা।
  - **হ্যান্ডলার-ভিত্তিক ডিজাইন**: স্কেলযোগ্য, নমনীয় সার্ভার বাস্তবায়নের জন্য হ্যান্ডলার-ভিত্তিক টুল/রিসোর্স/প্রম্পট পরিচালনার ব্যাখ্যা।
  - **প্রায়োগিক প্যাটার্ন**: এমন বাস্তব জীবনের দৃশ্য যেখানে নিম্নস্তরের সার্ভার প্যাটার্ন উন্নত বৈশিষ্ট্য এবং স্থাপত্যের জন্য উপকারী।

#### সাধারণ প্রমাণীকরণ (03-GettingStarted/11-simple-auth)
- **নতুন অধ্যায় যোগ করা হয়েছে**: MCP সার্ভারে সহজ প্রমাণীকরণ বাস্তবায়নের ধাপে ধাপে গাইড।
  - **অথ কনসেপ্টস**: প্রমাণীকরণ বনাম অনুমোদনের স্পষ্ট ব্যাখ্যা এবং ক্রেডেনশিয়াল পরিচালনা।
  - **বেসিক অথ বাস্তবায়ন**: পাইথন (স্টারলেট) এবং টাইপস্ক্রিপ্ট (এক্সপ্রেস) এ মিডলওয়্যার-ভিত্তিক প্রমাণীকরণ প্যাটার্ন সহ কোড নমুনা।
  - **উন্নত সিকিউরিটিতে প্রবগতি**: সহজ অথ দিয়ে শুরু করে OAuth 2.1 এবং RBAC তে উন্নতির নির্দেশনা, উন্নত সিকিউরিটি মডিউল রেফারেন্সসহ।

এই সংযোজনগুলি আরো মজবুত, সুরক্ষিত, এবং নমনীয় MCP সার্ভার বাস্তবায়নের জন্য ব্যবহারিক, হ্যান্ডস-অন নির্দেশনা প্রদান করে, মৌলিক ধারণাগুলির সাথে উন্নত উৎপাদন প্যাটার্নের সেতুবন্ধন করে।

#### MCP কাস্টম ট্রান্সপোর্টস (05-AdvancedTopics/mcp-transport/) - নতুন উন্নত বাস্তবায়ন গাইড

  - **Azure ইভেন্ট গ্রিড ট্রান্সপোর্ট**: ব্যাপক সার্ভারবিহীন ইভেন্ট-চালিত ট্রান্সপোর্ট বাস্তবায়ন

  - বর্তমান ডিসকভারি (`*/list`), রিট্রিভাল (`*/get`), ও এক্সিকিউশন (`*/call`) পদ্ধতি প্যাটার্ন সহ আপডেট
- **প্রোটোকল স্থাপত্য**: দুই স্তরের স্থাপত্য মডেল পরিচিত করানো হয়েছে
  - ডেটা স্তর: JSON-RPC 2.0 ভিত্তি সহ জীবনচক্র ব্যবস্থাপনা ও প্রিমিটিভ
  - পরিবহন স্তর: STDIO (স্থানীয়) ও স্ট্রিমেবল HTTP এর SSE (দূরবর্তী) পরিবহন প্রক্রিয়া
- **নিরাপত্তা কাঠামো**: বিস্তৃত নিরাপত্তা নীতিমালা স্পষ্ট ব্যবহারকারী সম্মতি, ডেটা গোপনীয়তা সুরক্ষা, টুল কার্যকারিতা সুরক্ষা ও পরিবহন স্তর সুরক্ষা সহ
- **যোগাযোগ প্যাটার্ন**: প্রোটোকল বার্তাসমূহ আপডেট, আরম্ভ, আবিষ্কার, কার্যকরী করণ, ও বিজ্ঞপ্তি প্রবাহ প্রদর্শন
- **কোড উদাহরণসমূহ**: নতুন MCP SDK প্যাটার্ন প্রতিফলিত করে বহু-ভাষার উদাহরণ (.NET, জাভা, পাইথন, জাভাস্ক্রিপ্ট) রিফ্রেশ করা হয়েছে

#### নিরাপত্তা (02-Security/) - ব্যাপক নিরাপত্তা আধুনিকায়ন  
- **স্ট্যান্ডার্ড সঙ্গতি**: MCP স্পেসিফিকেশন 2025-06-18 নিরাপত্তা শর্তাবলীর পূর্ণ সঙ্গতি
- **প্রমাণীকরণ বিবর্তন**: কাস্টম OAuth সার্ভার থেকে বাহ্যিক পরিচয় প্রদানকারী প্রতিনিধিত্ব (Microsoft Entra ID) হিসাবে বিবর্তন দলিলীকরণ
- **AI-নির্দিষ্ট হুমকি বিশ্লেষণ**: আধুনিক AI আক্রমণ ভেক্টরের বিস্তৃত কভারেজ
  - বাস্তব উদাহরণসহ বিস্তারিত প্রম্পট ইনজেকশন আক্রমণ দৃশ্যপট
  - টুল পুয়জনিং প্রক্রিয়া ও "রাগ পুল" আক্রমণ প্যাটার্ন
  - প্রেক্ষাপট উইন্ডো পুয়জনিং ও মডেল বিভ্রান্তি আক্রমণ
- **মাইক্রোসফট AI নিরাপত্তা সমাধানসমূহ**: মাইক্রোসফট নিরাপত্তা ইকোসিস্টেম ব্যাপক কভারেজ
  - AI প্রম্পট শিল্ডস উন্নত সনাক্তকরণ, স্পটলাইটিং, ও ডিলিমিটার কৌশলসহ
  - Azure কন্টেন্ট সেফটি ইন্টিগ্রেশন প্যাটার্ন
  - সাপ্লাই চেইন সুরক্ষার জন্য GitHub উন্নত নিরাপত্তা
- **উন্নত হুমকি প্রশমন**: বিস্তারিত নিরাপত্তা নিয়ন্ত্রণসমূহ
  - MCP-নির্দিষ্ট সেশন হাইজ্যাকিং আক্রমণ দৃশ্যপট ও ক্রিপ্টোগ্রাফিক সেশন আইডি শর্তাবলী সহ
  - MCP প্রক্সি দৃশ্যপটে কনফিউজড ডেপুটি সমস্যা স্পষ্ট সম্মতি শর্তোপযোগিতা সহ
  - বাধ্যতামূলক যাচাইকরণ নিয়ন্ত্রণ সহ টোকেন পাসত্রু দুর্বলতা
- **সাপ্লাই চেইন নিরাপত্তা**: ফাউন্ডেশন মডেল, এম্বেডিং সার্ভিস, প্রেক্ষাপট প্রদানকারী এবং তৃতীয়-পক্ষ API সহ AI সাপ্লাই চেইন বিস্তার
- **ফাউন্ডেশন সিকিউরিটি**: শূন্য বিশ্বাস স্থাপত্য এবং মাইক্রোসফট সিকিউরিটি ইকোসিস্টেমসহ এন্টারপ্রাইজ সিকিউরিটি প্যাটার্নে উন্নত ইন্টিগ্রেশন
