# 🚀 面向初学者的模型上下文协议 (MCP) 课程

## **通过 C#、Java、JavaScript、Rust、Python 和 TypeScript 的实战代码学习 MCP**

## 🧠 模型上下文协议课程概览
欢迎踏上模型上下文协议的旅程！如果你曾好奇 AI 应用是如何与不同工具和服务通信的，你即将发现这个优雅的解决方案，它正在改变开发者构建智能系统的方式。

把 MCP 想象成 AI 应用的通用翻译器——就像 USB 端口允许你将任何设备连接到计算机一样，MCP 让 AI 模型以标准化的方式连接到任何工具或服务。无论你是在构建第一个聊天机器人还是正在开发复杂的 AI 工作流，理解 MCP 都会赋予你创建更强大、更灵活应用的能力。

本课程专为你的学习旅程精心设计。我们将从你已经理解的简单概念开始，并通过你喜爱的编程语言逐步构建你的专长。每一步都包含清晰的说明、实用的示例，以及充足的鼓励。

完成本课程时，你将有信心构建自己的 MCP 服务器，将它们集成到流行的 AI 平台中，并理解这项技术如何重塑 AI 开发的未来。让我们一起启程，踏上这段激动人心的冒险之旅！

### 官方文档与规范

本课程与 **MCP Specification 2025-11-25**（最新稳定版本）保持一致。MCP 规范使用基于日期的版本控制（YYYY-MM-DD 格式），以确保协议版本的清晰跟踪。

> **展望未来：** 下一版本规范的候选发布版 **2026-07-28** 计划于 2026 年 7 月 28 日发布。它使协议在传输层无状态，正式化了扩展框架（MCP 应用和任务），强化了授权，并弃用 Roots、Sampling 和 Logging。详见 [MCP 的变更：2026-07-28 发布候选版](./01-CoreConcepts/mcp-2026-07-28-release-candidate.md) 了解完整细节。

随着你的理解加深，这些资源会变得更有价值，但不用急于立即全部阅读。先从你最感兴趣的部分开始吧！
- 📘 [MCP 文档](https://modelcontextprotocol.io/) – 这是你的首选资源，提供逐步教程和用户指南。文档特别适合初学者，提供清晰的示例，便于你按自己的节奏学习。
- 📜 [MCP 规范](https://modelcontextprotocol.io/specification/2025-11-25) – 可视为你的综合参考手册。在学习过程中，你会反复查阅这里的细节并探索高级功能。
- 📜 [MCP 规范版本控制](https://modelcontextprotocol.io/specification/versioning) – 包含协议版本历史及 MCP 如何利用基于日期的版本控制（YYYY-MM-DD 格式）的信息。
- 🧑‍💻 [MCP GitHub 仓库](https://github.com/modelcontextprotocol) – 这里有多种编程语言的 SDK、工具和代码示例，是实用示例和现成组件的宝库。

## 学习目标

本课程结束时，你将对自己的新技能充满信心和激动。你将实现以下目标：

• **理解 MCP 基础**：你将了解模型上下文协议是什么，以及它为何正在革新 AI 应用之间的协作，采用通俗易懂的类比和示例。

• **构建你的第一个 MCP 服务器**：你将用自己喜欢的编程语言创建一个工作中的 MCP 服务器，从简单示例开始，逐步提升技能。

• **连接 AI 模型与真实工具**：你将学会如何搭建 AI 模型与实际服务之间的桥梁，赋予应用强大新能力。

• <strong>实施安全最佳实践</strong>：你将理解如何保持 MCP 实现的安全，保护你的应用和用户。

• <strong>自信部署</strong>：你将学会如何将 MCP 项目从开发环境推向生产环境，掌握切实可行的部署策略。

• **加入 MCP 社区**：成为日益壮大的开发者社区一员，共同塑造 AI 应用开发的未来。

## 必备背景知识

在深入 MCP 细节之前，让我们确保你对一些基础概念感到熟悉。如果你不是这些领域的专家，不用担心 — 我们会逐步解释你需要知道的一切！

### 理解协议（基础）

协议就像对话的规则。当你给朋友打电话时，大家都知道接听时说“你好”，轮流说话，结束时说“再见”。计算机程序也需要类似规则来有效通信。

MCP 是一种协议——一套公认的规则，帮助 AI 模型和应用与工具及服务进行有效“对话”。就如同对话规则让人类沟通更顺畅，MCP 使 AI 应用通信更可靠、更强大。

### 客户端-服务器关系（程序协作方式）

你每天都使用客户端-服务器关系！当你使用网页浏览器（客户端）访问网站时，就是连接到发送页面内容的服务器。浏览器知道如何请求信息，服务器知道如何响应。

在 MCP 中，我们有类似的关系：AI 模型作为客户端请求信息或执行操作，而 MCP 服务器提供这些能力。就像有一个乐于助人的助手（服务器），AI 可以请求它执行特定任务。

### 为什么标准化重要（让万物协同工作）

想象如果每个汽车制造商使用不同形状的油泵 —— 你得为每辆车准备不同的适配器！标准化意味着达成共识，让事物可以无缝协作。

MCP 为 AI 应用提供了这种标准化。AI 模型不必为每个工具写定制代码，MCP 创造了一种通用通信方法。这意味着开发者只需构建一次工具，就能让许多不同 AI 系统使用。

## 🧭 你的学习路径概览

你的 MCP 旅程结构精心设计，逐步建立你的信心和技能。每阶段引入新概念，同时强化已学知识。

### 🌱 基础阶段：理解基础（模块 0-2）

这是你的冒险起点！我们将用熟悉的类比和简单的示例介绍 MCP 概念。你将了解 MCP 是什么，为什么存在，以及它在 AI 开发大局中的作用。

• **模块 0 - MCP 介绍**：我们将从探索 MCP 是什么及其对现代 AI 应用的重要性开始。你将看到 MCP 实际应用的案例，并理解它如何解决开发者常见的问题。


• **模块 1 - 核心概念讲解**：在这里你将学习 MCP 的基本构建模块。我们会使用大量类比和视觉示例，确保这些概念直观且易于理解。

• **模块 2 - MCP 中的安全性**：安全性听起来可能令人望而生畏，但我们将展示 MCP 内置的安全功能，并教你最佳实践，从一开始就保护你的应用程序。

### 🔨 构建阶段：创建你的第一个实现（模块 3）

现在真正的乐趣开始了！你将亲自动手构建实际的 MCP 服务器和客户端。别担心——我们会从简单开始，引导你完成每一步。

本模块包括多个动手指南，允许你使用偏好的编程语言练习。你将创建第一个服务器，构建一个客户端连接它，甚至集成流行的开发工具如 VS Code。

每个指南都包含完整的代码示例、故障排除提示以及为什么采用特定设计选择的解释。到本阶段结束时，你将拥有值得自豪的 MCP 实现！

### 🚀 成长阶段：高级概念与实际应用（模块 4-5）

掌握基础后，你已经准备好探索更复杂的 MCP 功能。我们将涵盖实用的实现策略、调试技巧以及诸如多模态 AI 集成等高级主题。

你还将学习如何针对生产环境扩展你的 MCP 实现，并集成如 Azure 的云平台。这些模块将助你构建可应对现实需求的 MCP 解决方案。

### 🌟 精通阶段：社区与专精（模块 6-11）

最后一阶段聚焦于加入 MCP 社区，并专注于你最感兴趣的领域。你将学习如何为开源 MCP 项目贡献代码，实施高级身份验证模式，构建全面的数据库集成解决方案。

模块 11 值得特别提及——它是一个完整的包含 13 个实验的动手学习路径，教你构建带有 PostgreSQL 集成的生产级 MCP 服务器。它就像一个汇总项目，将你学到的一切融合在一起！
