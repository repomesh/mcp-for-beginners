# MCP இல் மேம்பட்ட தலைப்புகள்

[![Advanced MCP: Secure, Scalable, and Multi-modal AI Agents](../../../translated_images/ta/06.42259eaf91fccfc6.webp)](https://youtu.be/4yjmGvJzYdY)

_(இந்த பாடத்தின் வீடியோவைப் பார்க்க மேலுள்ள படத்தை கிளிக் செய்க)_

இந்த அத்தியாயம் Model Context Protocol (MCP) செயல்பாட்டில் பல்வேறு மேம்பட்ட தலைப்புகளை உள்ளடக்கியது, அதில் பன்முகக் ஒருங்கிணைப்பு, அளவீட்டுத்திறன், பாதுகாப்பு சிறந்த நடைமுறைகள் மற்றும் நிறுவன ஒருங்கிணைப்பு ஆகியவை அடங்கும். இந்த தலைப்புகள் நவீன AI அமைப்புகளை பூர்த்தி செய்யக்கூடிய வலுவான மற்றும் தயாரிப்புக்கேற்ற MCP பயன்பாடுகளை உருவாக்க மிகவும் அவசியமாக உள்ளன.

## பார்வை

இந்த பாடம் Model Context Protocol செயல்பாட்டில் மேம்பட்ட கருத்துக்களை ஆய்வு செய்கிறது, பன்முகக் ஒருங்கிணைப்பு, அளவீட்டுத்திறன், பாதுகாப்பு சிறந்த நடைமுறைகள் மற்றும் நிறுவன ஒருங்கிணைப்பு ஆகியவற்றில் கவனம் செலுத்துகிறது. இந்த தலைப்புகள் நிறுவன சூழல்களில் சிக்கலான தேவைகளை கையாளும் தயாரிப்புக்கேற்ற MCP பயன்பாடுகளை உருவாக்க முக்கியமான ஆகும்.

> **எதிர்பார்க்கலாம்:** கீழே கொடுக்கப்பட்ட சில தலைப்புகள் `2026-07-28` MCP குறிப்புரு வெளியீட்டு அணி மூலம் பாதிக்கப்பட்டுள்ளன — Root Contexts (5.4) மற்றும் Sampling (5.6) வெளியீட்டு அணிக்கு பழுதுபடுத்தப்பட்ட நகர்வுகளை அடிப்படையாகக் கொண்டு கட்டமைக்கப்பட்டுள்ளன, மேலும் Protocol Features (5.16) இல் குறிப்பிட்டுள்ள சோதனை Tasks அம்சம் தனித்த Tasks நீட்டிப்பிற்கு நகர்கிறது. விரிவான தகவலுக்கு [What's Changing in MCP: The 2026-07-28 Release Candidate](../01-CoreConcepts/mcp-2026-07-28-release-candidate.md) பார்க்கவும்.

## கற்றல் இலக்குகள்

இந்த பாடத்தின் முடிவில், நீங்கள் முடிந்துவிடுவீர்கள்:

- MCP கட்டமைப்புகளில் பன்முகக் திறன்களை செயல்படுத்த
- உயர் தேவையான சூழல்களுக்கு உகந்த அளவீட்டு MCP வடிவமைப்புகளை வடிவமைக்க
- MCP பாதுகாப்பு கோட்பாடுகளுடன் ஒத்துப்போவதுபோல் பாதுகாப்பு சிறந்த நடைமுறைகளை விண்ணப்பிக்க
- MCP ஐ நிறுவன AI அமைப்புகள் மற்றும் கட்டமைப்புகளுடன் ஒருங்கிணைக்க
- தயாரிப்பு சூழல்களில் செயல்திறன் மற்றும் நம்பகத்தன்மையை மேம்படுத்த

> **MCP குறிப்புரை 2025-11-25 இல் புதியவை**: குறிப்புரை இப்போது **Tasks** (நெடுங்கால இயங்குகளோடு முன்னேற்ற கண்காணிப்பு), **Tool Annotations** (பாதுகாப்பு காரணமாக கருவி நடத்தைகள் பற்றிய தரவுத்தொகுப்பு), **URL Mode Elicitation** (வாடிக்கையாளர்களிடமிருந்து குறிப்பிட்ட URL உள்ளடக்கத்தை கோருதல்) மற்றும் மேம்பட்ட **Roots** (வேலைப்பகுப்பு சூழல் மேலாண்மை) ஆகியவற்றுக்கு சோதனை ஆதரவுகளை இணைக்கிறது. முழு விவரங்களுக்கு [MCP குறிப்புரை மாற்ற அம்சங்கள் பட்டியல்](https://spec.modelcontextprotocol.io/) பார்க்கவும்.

ஒரு குறிப்பிட்ட பயன்படுத்தல் வழக்குக்கான நிறுவன தர MCP செயல்பாட்டை வடிவமைக்க:

1. உங்கள் பயன்படுத்தல் விதிகளுக்கான பன்முக தேவைகளை அடையாளம் காண்க
2. நுண்ணறிவு தரவை பாதுகாக்க தேவையான பாதுகாப்பு கட்டுப்பாடுகளை வரையறுக்கவும்
3. மாறும் சுமைகளை கையாளக்கூடிய அளவீட்டுத்திறன் கொண்ட கட்டமைப்பை வடிவமைக்கவும்
4. நிறுவன AI அமைப்புகளுடன் ஒருங்கிணைப்பு புள்ளிகளை திட்டமிடவும்
5. நிகழக்கூடிய செயல்திறன் தடுப்புகளை மற்றும் தடுப்பு திட்டங்களை ஆவணப்படுத்தவும்
