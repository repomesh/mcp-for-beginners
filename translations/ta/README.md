# 🚀 ஆரம்பதாரர்களுக்கான மாடல் சூழல் நெறிமுறை (MCP) பாடத்திட்டம்

## **C#, Java, JavaScript, Rust, Python மற்றும் TypeScript இல் செய்முறை குறியீடு உதவியுடன் MCP கற்றுக்கொள்ளுங்கள்**

## 🧠 மாடல் சூழல் நெறிமுறை பாடத்திட்டத்தின் அறிமுகம்
மாடல் சூழல் நெறிமுறை சார்ந்த உங்கள் பயணத்திற்கு வரவேற்கிறோம்! ஏதாவது AI செயலிகள் வெவ்வேறு கருவிகள் மற்றும் சேவைகளுடன் எவ்வாறு தொடர்புகொள்வது என்பது பற்றி யோசித்திருப்பதுண்டா? தந்திரமான தீர்வை நீங்கள் கண்டுபிடிக்க இருக்கிறீர்கள், இது டெவலப்பர்களுக்கு புத்திசாலி அமைப்புகளை உருவாக்க புதிய வழியைத் தருகிறது.

MCP ஐ AI செயலிகளுக்கான பொதுவான மொழிபெயர்ப்பாளராக நினைத்துக்கொள்ளுங்கள் – USB போர்ட்கள் எப்படி எந்த சாதனத்தையும் உங்கள் கணினியில் இணைக்க உதவுகிறது போல, MCP AI மாடல்கள் எந்த கருவி அல்லது சேவையையும் ஒருங்கிணைக்க ஏற்பாடுகளை வழங்குகிறது. நீங்கள் முதல் சாட்பாடை உருவாக்குகிறீர்களா அல்லது சிக்கலான AI பணிகளை செய்யிறீர்களா என்பது முக்கியமில்லை, MCP பற்றி புரிந்துக்கொண்டால், நீங்கள் திறமை மிக்க, நெறிமுறை மாறுபாடுகளுடன் கூடிய செயலிகளை உருவாக்க பல்லாயிரம் சாத்தியங்கள் உண்டு.

இந்த பாடத்திட்டம் உங்கள் கற்றல் பயணத்திற்கு பொறுமையுடன் மற்றும் கவனமாக வடிவமைக்கப்பட்டுள்ளது. நீங்கள் ஏற்கனவே அறிந்த எளிய கருத்துகளிலிருந்து துவங்கி முறையான பயிற்சிகளின் மூலம் உங்கள் திறமைகளை கட்டியெழுப்புவீர்கள். ஒவ்வொரு படியும் தெளிவான விளக்கங்கள், மேலான உதாரணங்கள் மற்றும் முழுமையான ஊக்கங்களுடன் இருக்கிறது.

இந்த பயணத்தை முடித்தபோது, நீங்கள் உங்கள் சொந்த MCP சேவைகளை உருவாக்க, பிரசித்திபெற்ற AI தளங்களுடன் இணைக்க மற்றும் இந்த தொழில்நுட்பம் எவ்வாறு AI வளர்ச்சியின் எதிர்காலத்தை மாற்றி வருகிறது என்பது பற்றி நன்றாக புரிந்து கொள்வீர்கள். இந்த தென்றல் பயணத்தை சேர்ந்து துவங்குவோம்!

## கற்றல் நோக்கங்கள்

இந்த பாடத்திட்டம் முடிவுக்கு வந்தபோது, உங்கள் புதிய திறமைகளுக்கு நீங்கள் தெளிவும் உற்சாகமும் பெறுவீர்கள். நீங்கள் அடையும் வளர்ந்த நிலைகள் இவை:

• **MCP அடிப்படைகளை புரிந்து கொள்ளுதல்**: மாடல் சூழல் நெறிமுறை என்றால் என்ன மற்றும் அது ஏன் AI செயலிகள் இணைந்து செயற்படும் முறையை மாற்றி வருகிறது என்பதை காட்சிகள் மற்றும் உதாரணங்களுடன் புரிந்துகொள்வீர்கள்.

• **உங்கள் முதல் MCP சேவையகத்தை கட்டமைத்தல்**: உங்கள் விருப்ப programming மொழியில் எளிய உதாரணங்களைத் தொடங்கி செயல்பாடுகளுக்காக செயலாற்ற MCP சேவையகத்தை உருவாக்குவீர்கள்.

• **AI மாடல்களை உண்மையான கருவிகளுடன் இணைத்தல்**: AI மாடல்களுக்கும் உண்மையான சேவைகளுக்கும் இடையே தோராயமாக பாலம் அமைப்பதை கற்றுக்கொள்வீர்கள், அதனால் உங்கள் செயலிகள் சக்திவாய்ந்த புதிய திறன்களை பெறும்.

• **பாதுகாப்பு சிறந்த நடைமுறைகளை அமలు செய்தல்**: உங்கள் MCP அமல்படுத்தல்களை பாதுகாப்பாய் வைத்துக்கொள்ள எப்படி என்பது பற்றிய அறிவை பெறுவீர்கள், உங்கள் பயன்பாடுகளும் பயனர்களும் பாதுகாப்பாக இருப்பார்கள்.

• **விடுவிப்பதில் நம்பிக்கை**: உங்கள் MCP திட்டங்களை வளர்ச்சி சூழலிலிருந்து உற்பத்தி சூழலுக்கு எவ்வாறு மாற்றுவது என்பது பற்றிய நடைமுறை வழிகளைக் கற்றுக்கொள்வீர்கள்.

• **MCP சமூகத்தில் இணையுதல்**: AI செயலி வளர்ச்சியின் எதிர்காலத்தை வடிவாக்கும் வளர்ந்து வரும் டெவலப்பர் சமூகத்தில் நீங்கள் பங்கு கொள்ளும்.

## அவசியமான பின்னணி

MCP விவரங்களில் நுழைய முன், சில அடிப்படை கருத்துக்களில் நீங்கள் நன்றாக செறிந்ததாக உணர்வதற்கு கவனம் செலுத்துவோம். இது துறை நிபுணர் ஆகாதிருந்தாலும் கவலை வேண்டாம் - தேவையான அனைத்தும் எளிதாக விளக்கப்படும்!

### நெறிமுறைகளைப் புரிந்துகொள்ளுதல் (அடித்தளம்)

ஒரு நெறிமுறை என்பதை உரையாடலின் விதிகள் போல நினைத்துக் கொள்ளுங்கள். நீங்கள் ஒரு நண்பரை அழைக்கும் போது, இருவரும் "வணக்கம்" சொல்வது, பேசும் பங்கள்களை மாற்றிக் கொள்ளுதல் மற்றும் முடிந்தவுடன் "பிரியா" என்று கூறுவது தெரிகிறது. கணினி செயலிகள் கூட சரியான தொடர்புக்கு இதைப் போல விதிகள் தேவை.

MCP ஒன்று நெறிமுறை ஆகும் – அனைத்து AI மாடல்களும் கருவிகளும் மற்றும் சேவைகளுடன் "உரையாட" உதவும் ஒப்புக்கொள்ளப்பட்ட விதிமுறைகள் தொகுப்பு. மனித உரையாடலுக்குக் காணும் விதிகளால் தொடர்பு சிறப்பாக நடைபெறும் போல, MCP உண்ணிய AI செயலி தொடர்புக்கு நம்பகத்தன்மையுடன் சக்தியையும் தருகிறது.

### கிளையன்ட்-சேவையகம் உறவுகள் (செயலிகள் உடன் ஒத்துழைப்பு)

நீங்கள் ஏற்கனவே தினசரி கிளையன்-சேவையகம் உறவுகளை பயன்படுத்துகிறீர்கள்! நீங்கள் ஒரு வலை உலாவியை (கிளையன்) பயன்படுத்தி ஒரு இணையதளத்தை பார்ப்பதற்காக உலாவி ஒரு வலை சேவையகத்துடன் இணைகிறது, அந்த சேவையகம் பக்க உள்ளடக்கத்தை அனுப்புகிறது. உலாவி தகவல்களை கேட்க தெரியும், சேவையகம் பதில் சொல்வதை தெரியும்.

MCP இல் இதுபோன்ற உறவு உள்ளது: AI மாடல்கள் தகவல் அல்லது செயல் கோரிக்கை செய்யும் கிளையன்ட் ஆக நடக்கிறார்கள், MCP சேவையகங்கள் அதற்கான திறன்களை வழங்குகின்றன. இது AIக் கேள்விகள் கேட்டால் உதவும் உதவியாளராயிருக்கும் போன்றது.

### ஒழுங்குபடுத்தல் ஏன் முக்கியம் (பணிகளை ஒன்றிணைத்தல்)

ஒவ்வொரு கார் உற்பத்தியாளர் வெவ்வேறு வடிவ புவி உபகரணங்களைப் பயன்படுத்தினால், ஒவ்வொரு காருக்குமான தனித்த அடாப்டர்னு வேண்டும்! ஒழுங்குபடுத்தல் என்பது பொதுவான முறைகளை ஒப்புக்கொண்டு வேலை செய்யும் வாய்ப்புகளை உருவாக்குவது ஆகும்.

MCP AI செயலிகளுக்கு இந்த ஒழுங்குபடுத்தலை வழங்குகிறது. ஒவ்வொரு AI மாடல் எதிர்பார்க்கும் தனிப்பட்ட குறியீடு எழுதுவதற்குப் பதிலாக, MCP ஒரு பொதுவான தொடர்பு முறையை உருவாக்குகிறது. இதனால் டெவலப்பர்கள் ஒருமுறை கருவிகளை உருவாக்கி அதனை பல்வேறு AI அமைப்புகளில் வேலை செய்யச் செய்யலாம்.

## 🧭 உங்கள் கற்றல் பாதை கண்ணோட்டம்

உங்கள் MCP பயணம் நம்பிக்கையுடன் மற்றும் திறமைகளுடன் கட்டமைக்கப்பட்டுள்ளது. ஒவ்வொரு கட்டமும் புதிய கருத்துக்களை அறிமுகப்படுத்தும் மற்றையவற்றைக் மனதில் வைக்கிறது.

### 🌱 அடித்தளம் கட்டம்: அடிப்படைகளை புரிந்துகொள்ளுதல் (தொடக்க தொகுதி 0-2)

இங்கு உங்கள் பயணம் துவங்குகிறது! MCP கருத்துக்களை பரிசின உணர்வுடன் மற்றும் எளிய உதாரணங்களுடன் அறிமுகப்படுத்துகிறோம். MCP என்பது என்ன, அது ஏன் அவசியம் மற்றும் அது AI வளர்ச்சியில் எவ்வாறு பங்கு வகிக்கிறதென புரிந்துகொள்ளலாம்.

• **தொடக்க தொகுதி 0 - MCP அறிமுகம்**: எமது பயணம் எதைப்பற்றிச் சொல்கிறது மற்றும் MCP இன் முக்கியத்துவத்தை எல்லாம் ஆராய்வோம். MCP செயலிலும் நடைமுறை எளிய உதாரணங்களையும் நம் அவர்கள் பார்க்கப்போகிறோம்.
• **அலகு 1 - முக்கியக் கருத்துக்கள் விளக்கம்**: இங்கு நீங்கள் MCP இன் அடிப்படைக் கட்டுமானங்களை கற்றுக்கொள்ளப்போகிறீர்கள். இந்தக் கருத்துக்கள் இயல்பானதும் புரிந்துகொள்ளக்கூடியதும் ஆக இருப்பதற்காக பலளவு ஒப்பமைகள் மற்றும் கண்ணோட்ட உதாரணங்களை பயன்படுத்துவோம்.

• **அலகு 2 - MCP இல் பாதுகாப்பு**: பாதுகாப்பு சற்று பயங்கரமாகத் தெரிந்தாலும், MCP இல் உள்ள கட்டமைக்கப்பட்ட பாதுகாப்பு அம்சங்களை எவ்வாறு கொண்டுள்ளது என்பதை நாங்கள் கற்றுத்தொடர்ந்து, உங்கள் பயன்பாடுகளை ஆரம்பத்திலேயே பாதுகாக்க சிறந்த நடைமுறைகளை அறிவோம்.

### 🔨 கட்டுமானப் பகுதி: உங்கள் முதல் செயலாக்கங்களை உருவாக்குதல் (அலகு 3)

இப்போது உண்மையான மகிழ்ச்சி தொடங்குகிறது! நீங்கள் MCP சேவையகங்களையும் கிளையன்ட்களையும் நேரடியாக உருவாக்க அனுபவிக்கும் வாய்ப்பு கிடைக்கும். கவலைப்பட வேண்டாம் - எளிமையாக தொடங்கி, ஒவ்வொரு படியிலும் வழிகாட்டுவோம்.

இந்த அலகில் பல கைபிடிக்கும் வழிகாட்டுதல்கள் உண்டு, அவற்றின் மூலம் உங்கள் விருப்பமாய்க் கோடிங் மொழியில் பயிற்சி பெற முடியும். உங்கள் முதல் சேவையகத்தை உருவாக்கி, அதோடு இணைக்க ஒரு கிளையன்டை தயார் செய்து, VS Code போன்ற பிரபலமான மேம்பாட்டு கருவிகளுடன் இணைக்கும் பணியையும் செய்வீர்கள்.

ஒவ்வொரு வழிகாட்டலும் முழுமையான குறியீடு உதாரணங்கள், பிரச்சனை தீர்க்கும் குறிப்புகள் மற்றும் குறிப்பிட்ட வடிவமைப்பு தேர்வுகள் ஏன் செய்தோம் என்பதற்கான விளக்கங்களைக் கொண்டுள்ளது. இந்த கட்டத்தை முடித்தவுடன், நீங்கள் பெருமையுடன் கொள்ளக்கூடிய செயல்படும் MCP செயலாக்கங்களை கைப்பற்றுவீர்கள்!

### 🚀 வளர்ச்சி பகுதி: மேம்பட்ட கருத்துக்கள் மற்றும் நிஜ உலகப் பயன்பாடு (அலகுகள் 4-5)

அடிப்படைகளை நன்கு கையாண்டபின், நீங்கள் மேலும் өрிஞ்சிய MCP அம்சங்களை ஆராயத் தயாராக இருக்கிறீர்கள். நாங்கள் நடைமுறை செயலாக்க முறைகள், பிழைத் திருத்தக் கலைகள் மற்றும் பல-முறை AI ஒருங்கிணைப்பு போன்ற மேம்பட்ட தலைவைகளை உள்ளடக்கியவைகளை கற்பிப்போம்.

மேலும், உற்பத்தி பயன்பாட்டுக்காக உங்கள் MCP செயலாக்கங்களை அளவீடு செய்வது மற்றும் Azure போன்ற மேக தளங்களுடன் ஒருங்கிணைப்பது எப்படி என்பதையும் கற்பீர்கள். இவை MCP தீர்வுகளை நிஜ உலகத் தேவைகளை சேகரிக்கும் நிலைக்கு உருவாக்கத் தயார் செய்யும்.

### 🌟 தேர்ச்சி பகுதி: சமுதாயம் மற்றும் நிபுணத்துவம் (அலகுகள் 6-11)

இறுதி பகுதி MCP சமுதாயத்தில் கலந்து கொள்ளுதல் மற்றும் உங்கள் ஆர்வமான பகுதிகளில் நிபுணத்துவம் பெறுவதை கவனிக்கும். நீங்கள் திறந்த மூல MCP திட்டங்களில் பங்களித்தல், மேம்பட்ட அங்கீகாரம் முறைகளை நிறைவேற்றல் மற்றும் விரிவான தரவுத்தளம் ஒருங்கிணைத்த தீர்வுகளை கட்டமைத்தல் ஆகியவற்றைப் பற்றிய கற்றல் பெறுவீர்கள்.

அலகு 11 சிறப்பு குறிப்பிடத்தக்கது - இது PostgreSQL ஒருங்கிணைப்புடன் உற்பத்திக்குத் தயாரான MCP சேவையகங்களை கட்டமைப்பதற்கான முழுமையான 13-லாப் கைதிறன் கற்றல் பாதை ஆகும். இது நீங்கள் கற்றதெல்லாம் ஒன்றாக இணைத்து காணும் ஓர் மேற்பார்வைத் திட்டம் போல் இருக்கும்!

### 💻 மாதிரி குறியீட்டு திட்டங்கள்

- குறைந்தது ஒரே மொத்தமுறை மொழிகளில் C#, Java, JavaScript, Python, அல்லது TypeScript ப்ரோக்ராமிங் அடிப்படை அறிவு
- கிளையண்ட்-சர்வர் மாதிரி மற்றும் APIs பற்றிய புரிதல்

- ( விருப்பம் ) AI/ML கருத்துக்களில் பின்னணி

- எங்கள் சமூகம் விவாதங்களில் பார்வையிடுதல் மற்றும் உதவி பெறுதல்

## 📚 படிப்பு வழிகாட்டி & வளங்கள்

இந்த தொகுப்பில் பல வளங்கள் உள்ளன, உங்களுக்கு வழிகாட்ட உதவவும் சக்தியும்:

### படிப்பு வழிகாட்டி

ஒரு விரிவான [படிப்பு வழிகாட்டி](./study_guide.md) வேறு பகுதி எவ்வாறு செயல்படும் என்பதைக் காட்சி முறை விரிவாக்குவதற்கும் உதவும். இது எல்லா தலைப்புகளும் எப்படி இணைக்கப்படுகின்றன என்பதையும் உதவிக்குறிப்புகளையும் அளிக்கும். குறிப்பாக, படங்களை பார்க்க விரும்பும் கண்களுக்கான பெரிய படிப்பாக்கம் அதிவெற்றி.

வழிகாட்டியில் உள்ளவை:
- எல்லா தலைப்புகளும் காட்சியளிக்கும் பாடத்திட்ட வரைபடம்
- ஒவ்வொரு தொகுப்பு பகுதியின் விரிவான படிவம்
- மாதிரி திட்டங்களை எப்படி பயன்படுத்துவது என்பதற்கான வழிகாட்டி
- வேறு திறன் நிலைகளுக்கான பரிந்துரைக்கப்பட்ட கற்றல் பாதைகள்
- உங்கள் கற்றல் பயணத்தை நிறைவு செய்ய கூடுதல் வளங்கள்

### மாற்றப்பட்ட கோப்பு

நாம் மிக விரிவாக [மாற்றப்பட்ட கோப்பு](./changelog.md) வைத்திருக்கிறோம், இது பாடத்திட்டப் பொருட்களில் எல்லா முக்கிய மாற்றங்களையும் தடுப்பதற்கும், அதனால் நீங்கள் இன்றைக்கு உள்ள முன்னேற்றங்களை தெளிவாக காணலாம்.
- புதிய உள்ளடக்கம் சேர்க்கைகள்
- கட்டமைப்பு மாற்றங்கள்
- அம்ச மேம்பாடுகள்
- ஆவண மேம்பாடுகள்

## 🎓 உங்கள் MCP பயணம் துவங்குகிறது

பாராட்டு! நீங்கள் ஒரு நெகிழ்வான பயணத்தின் முதல் படி எடுத்து, உங்கள் ப்ரோக்ராமிங் திறன்களை விரிவாக்கி, AI மேம்பாட்டின் முன்னணி முனையில் இணைகிறீர்கள்.

### நீங்கள் இதுவரை செய்த சாதனைகள்

இந்த அறிமுகத்தைப் படிப்பதன் மூலம், நீங்கள் MCP அறிவின் அடித்தளத்தை கட்டியுள்ளீர்கள். MCP என்ன, அது ஏன் முக்கியம் என்பதையும் மற்றும் இந்த பாடத்திட்டம் உங்கள் கற்றலை எப்படிச் சொல்கிறது என்பதையும் புரிந்து கொண்டுள்ளீர்கள். இது ஒரு முக்கிய சாதனை மற்றும் இதுபோன்ற தொழில்நுட்பத்தில் உங்கள் நிபுணத்துவத்தின் துவக்கம்.

### எதிர்கால சவால்

நீங்கள் பாடங்களை முறையாக வழிமொழிகிறபோது ஒவ்வொரு நிபுணரும் ஒரே முறையில் ஒரு தொடக்கநிலை மாணவர் தான் என்பது நினைவில் வையுங்கள். இப்போது சிக்கலானது போன்ற கருத்துக்கள் மேலதிக பயிற்சியால் எளிதாக மாறும். ஒவ்வொரு சிறிய படியும் உங்கள் மேம்பாட்டுப் பயணத்தில் சக்தி வாய்ந்த திறன்களை கட்டமைக்கும்.

### நீங்கள் சேர்ந்துள்ள ஆதரவு வலைப்பின்னல்

நீங்கள் MCP ஆர்வமுள்ள கற்றுக்கொள்ளும் மக்களும் நிபுணர்களும் சேர்ந்த ஒரு சமுதாயத்தில் சேர்ந்துள்ளீர்கள். நிரல் சிக்கல் வந்தாலும் அல்லது ஒரு கண்டுபிடிப்பை பகிர விரும்பினாலும், இந்த சமுதாயம் உங்கள் பயணத்திற்கு ஆதரவாக உள்ளது.
