# மாற்றம் எழுதுகோல்: MCP தொடக்கத்திற்கான பாடத்திட்டம்

இந்த ஆவணம் Model Context Protocol (MCP) தொடக்கக்கால பாடத்திட்டத்திற்கான அனைத்து முக்கிய மாற்றங்களின் பதிவாக செயல்படுகிறது. மாற்றங்கள் வரிசைப்படுத்தப்பட்டுள்ளன (புதிய மாற்றங்கள் முதலில்).

#### அடிப்படை கருத்துக்கள் மேம்பாடு (01-CoreConcepts/)  
- **புதிய கிளையன்ட் முனைமை - ரூட்ஸ்**: கோப்பு அமைப்பு எல்லைகள் மற்றும் அணுகல் அனுமதிகளை புரிந்துகொள்ள சர்வர்களுக்கு கண்ணோட்டமான ஆவணங்கள் சேர்க்கப்பட்டது  
- **கருவி குறிப்பு**: கருவி நடத்தை குறிப்பு (`readOnlyHint`, `destructiveHint`) விரிவாக சேர்க்கப்பட்டு சிறந்த கருவி செயல்பாட்டு முடிவுகளுக்கு உதவுகிறது  
- **சேர்க்கை கோரிக்கையில் கருவி அழைப்புகள்**: மாதிரி சார்ந்த கருவி அழைப்புக்களுக்கு `tools` மற்றும் `toolChoice` பரிமாணங்கள் சேர்க்கப்பட்டுள்ளது  
- **URL முறை ஊக்குவிப்பு**: சர்வர் தொடங்கும் வெளி வலை தொடர்புகளுக்கான URL அடிப்படையிலான ஊக்குவிப்பு ஆவணமாக்கப்பட்டது  
- **பணிகள் (சோதனை)**: நிலையான செயலாக்க உறைப்பு மற்றும் தணிக்கப்பட்ட முடிவுகள் மீட்டெடுக்கும் சோதனை அம்சம் சேர்க்கப்பட்டது  
- **ஐகான்கள் ஆதரவு**: கருவிகள், வளங்கள், வள வார்ப்புகள், உத்தரவுரைகள் இணை metadata க்கு ஐகான்கள் சேர்க்கக்கூடியதாக ஆனது

- **கட்டமைப்பு இணக்கம்**: இரண்டு அடுக்கு கட்டமைப்பு (தரவு அடுக்கு + பரிமாற்ற அடுக்கு) ஆவண சரியான இணக்கம் உறுதி செய்யப்பட்டது  
- **முதன்மை பொருள்கள் ஆவணம்**: சர்வர் முதன்மைகள் (வளங்கள், உத்தரவுரைகள், கருவிகள்) மற்றும் கிளையன்ட் முதன்மைகள் (சேர்க்கை, ஊக்குவிப்பு, பதிவேடு, ரூட்ஸ்) சரிபார்க்கப்பட்டது

  - **வழக்கமான மற்றும் குறைந்த மட்ட சர்வர்**: இரு அணுகுமுறைகளுக்குமான Python மற்றும் TypeScript உதாரணங்கள் உடன் விரிவான ஒப்பீடு
  - **ஹேண்ட்லர் அடிப்படையிலான வடிவமைப்பு**: மாண்பூட்டான, நெகிழ்வான சர்வர் நடைமுறைக்கு ஹேண்ட்லர் வகை கருவி/வள/மூலம் மேலாண்மை விளக்கம்
  - **நடைமுறை படைப்புகள்**: முன்னேற்ற அம்சங்கள் மற்றும் கட்டமைப்புகளுக்கான குறைந்த மட்ட சர்வர் படைப்புகளை நிஜ உலகுக்கான நிலைபாடுகளுடன் விளக்கம்

#### எளிய அங்கீகாரம் (03-GettingStarted/11-simple-auth)
- **புதிய அத்தியாயம் சேர்க்கப்பட்டது**: MCP சர்வர்களில் எளிய அங்கீகாரம் நடைமுறைகள் படி படியாக்கும் கையேடு
  - **அங்கீகார கருத்துக்கள்**: அங்கீகாரத்திற்கும் அனுமதிப்பற்றுடனும் தெளிவான விளக்கம் மற்றும் சான்று மேலாண்மை
  - **அடிப்படை அங்கீகாரம் நடைமுறை**: Python (Starlette) மற்றும் TypeScript (Express) இல் மிடில்வேர் அடிப்படையிலான அங்கீகாரம் படைப்புகள், குறியீட்டு உதாரணங்களுடன்
  - **முன்னேற்ற பாதுகாப்புக்கு முன்னேற்றம்**: எளிய அங்கீகாரத்திலிருந்து OAuth 2.1 மற்றும் RBAC ஆக முன்னேற வழிகாட்டி, முன்னேற்ற பாதுகாப்பு கூறுகள் குறித்த குறிப்புகளுடன்

இந்தச் சேர்க்கைகள் வலுவான, பாதுகாப்பான மற்றும் நெகிழ்வான MCP சர்வர் நடைமுறைகளை கட்டிடுவதற்கான நடைமுறை மேற்கோள்களை வழங்குகின்றன, அடிப்படைக் கருத்துக்களையும் முன்னேற்ற தயாரிப்பு படைப்புகளையும் இணைக்கின்றன.

    - PKCE மற்றும் பயனர் உறுதிப்படுத்தலை உட்படுத்தி Confused Deputy தாக்குதல் தடுப்பு
    - பார்வையாளர் சான்றிதழ் பாதுகாப்புடன் தொகம் பரப்பும் தடுப்பு
    - குறியாக்கப்பட்ட இணைப்பு மற்றும் பாவனை பகுப்பாய்வு உட்பட அமர்வு கைபற்றல் தடுப்பு
  - **என்டர் பிரைஸ் பாதுகாப்பு ஒருங்கிணைப்பு**: Azure Application Insights கண்காணிப்பு, அச்சுறுத்தல் கண்டறிதல் குழாய்கள் மற்றும் பொருள் சங்கிலி பாதுகாப்பு
  - **செயல்முறை சரிபார்ப்பு பட்டியல்கள்**: Microsoft பாதுகாப்பு சுற்றுச்சூழல் நன்மைகளுடன் கட்டாய மற்றும் பரிந்துரைக்கப்பட்ட பாதுகாப்பு கட்டுப்பாடுகள் தெளிவு

#### மூலக் கருத்துக்கள் (01-CoreConcepts/) - முழுமையான நவீனமயம்
- **நடவடிக்கை பதிப்பு புதுப்பிப்பு**: MCP குறிப்பு 2025-06-18 உடன் மாற்றப்பட்ட தேதி அடிப்படையிலான பதிப்பீடு (YYYY-MM-DD வடிவில்)
- **கட்டமைப்பு மேம்பாடு**: தற்போது MCP கட்டமைப்பு பாணிகளுக்கு ஏற்ப Hosts, Clients மற்றும் Servers விரிவான விளக்கங்கள்
  - Hosts இப்போது பல MCP கிளையன்ட் இணைப்புகளை ஒருங்கிணைக்கும் AI பயன்பாடுகளாக தெளிவாக வரையறுக்கப்பட்டன
  - Clients ஒன்று-மாத்திரம் சர்வர் உறவுகளை பராமரிக்கும் நடைமுறை இணைப்பாளர்களாக விவரிக்கப்பட்டன
  - Servers உள்ளூர் மற்றும் தொலைவில் அமைக்கும் சூழ்நிலைகள் விரிவுபடுத்தப்பட்டன
- **அடிப்படை மீட்பு**: சர்வர் மற்றும் கிளையன்ட் அடிப்படைகள் முழுமையாக மாற்றப்பட்டன
  - சர்வர் அடிப்படைகள்: வளங்கள் (தரவு மூலங்கள்), ப்ராம்ட் (வார்ப்பு வடிவங்கள்), கருவிகள் (செயலாக்கக் கூறுகள்) விரிவான விளக்கங்கள் மற்றும் உதாரணங்களுடன்
  - கிளையன்ட் அடிப்படைகள்: மாதிரிப்படுத்துதல் (LLM நிறைவு), கேள்வித்தாள் (பயனர் உள்ளீடு), பதிவு (துணைத்திருத்தம்/கண்காணிப்பு)
  - தற்போதைய கண்டுபிடி (`*/list`), மீள் பெறுதல் (`*/get`), மற்றும் செயல்படுத்தல் (`*/call`) முறைகள் புதுப்பிக்கப்பட்டன
- **நடவடிக்கை கட்டமைப்பு**: இரு அடுக்கு கட்டமைப்பு முறை அறிமுகம்
  - தரவு அடுக்கு: JSON-RPC 2.0 அடித்தளம் உடன் வாழ்க்கை நெறி மேலாண்மை மற்றும் அடிப்படைகள்
  - போக்குவரத்து அடுக்கு: STDIO (உள்ளூர்) மற்றும் HTTP தொடர் + SSE (தொலை) போக்குவரத்து முறைகள்
- **பாதுகாப்பு கட்டமைப்பு**: தெளிவான பயனர் ஒப்புதல், தரவு தனியுரிமை பாதுகாப்பு, கருவி செயல்பாடு பாதுகாப்பு, மற்றும் போக்குவரத்து பாதுகாப்பு
- **தொடர்பு பாணிகள்**: தொடக்கம், கண்டுபிடித்தல், செயல்படுத்தல் மற்றும் அறிவிப்பு ஓட்டங்களை காட்டும் நடைமுறைகளை புதுப்பித்தல்
- **குறியீடு உதாரணங்கள்**: தற்போதைய MCP SDK பாணி பிரதிநிதித்துவம் செய்வதற்காக .NET, Java, Python, JavaScript மொழி உதாரணங்கள் புதுப்பிக்கப்பட்டன

#### பாதுகாப்பு (02-Security/) - முழுமையான பாதுகாப்பு மறுசீரமைப்பு  
- **தரநிலைகள் இணைப்பு**: MCP குறிப்பு 2025-06-18 பாதுகாப்பு தேவைகளுடன் முழுமையான இணைப்பு
- **அங்கீகாரம் மேம்பாடு**: தனிப்பயன் OAuth சர்வரிலிருந்து வெளிப்புற அடையாள வழங்குநர் அனுமதி (Microsoft Entra ID) வரை வளர்ச்சி ஆவணம்
- **AI-சிறப்பு அச்சுறுத்தல் பகுப்பாய்வு**: நவீன AI தாக்குதல் திசைகள் விரிவடைந்த வரைபடம்
  - விரிவான ப்ராம்ட் இன்ஜெக்ஷன் தாக்குதல் நிலைப்பாடுகள் மற்றும் உண்மையான உலக உதாரணங்கள்
  - கருவி விஷப்பொருள் உற்பத்தி மற்றும் "அடி இழுத்தல்" தாக்குதல் முறைகள்
  - சூழல் ஜன்னல் விஷப்பொருள் மற்றும் மாதிரி குழப்ப தாக்குதல்கள்
- **Microsoft AI பாதுகாப்பு தீர்வுகள்**: Microsoft பாதுகாப்பு சுற்றுச்சூழல் விரிவான கையொப்பம்
  - AI Prompt Shields மேம்பட்ட கண்டறிதல், முக்கியத்துவம் பெருக்கல் மற்றும் பிரிவார எடுக்கும் நுட்பங்கள்
  - Azure Content Safety ஒருங்கிணைப்பு முறைகள்
  - GitHub முன்னேற்றப்பட்ட பாதுகாப்பு பொருள் சங்கிலி பாதுகாப்புக்கு
- **மேம்பட்ட அச்சுறுத்தல் தடுப்பு**: விரிவான பாதுகாப்பு கட்டுப்பாடுகள்
  - MCP-சார்ந்த அமர்வு கைபற்றல் தாக்குதல் மற்றும் குறியாக்கப்பூர்வ அமர்வு ID தேவைகள்
  - MCP பிரதிநிதி பிரச்சனைகள் மற்றும் தெளிவான ஒப்புதலுடன் Confused Deputy தாக்குதல் தடுப்பு
  - கட்டாய மதிப்பீடு கட்டுப்பாடுகளுடன் தொகம் பரப்புத் தவறுகள்
- **பொருள் சங்கிலி பாதுகாப்பு**: அடிப்படைக் மாதிரிகள், உருவாக்க சேவைகள், சூழல் வழங்கிகள் மற்றும் மூன்றாம் தரப்பு APIs உட்பட AI பொருள் சங்கிலி விரிவாக்கம்
- **அடித்தளம் பாதுகாப்பு**: பூஜ்ஜிய நம்பிக்கை கட்டமைப்பு மற்றும் Microsoft பாதுகாப்பு சுற்றுச்சூழல் ஒருங்கிணைப்பு மேம்படுத்தப்பட்டது
- **வளம் அமைப்பு**: வகைப்படுத்தப்பட்ட விரிவான வள இணைப்புகள் (அதிகாரப்பூர்வ ஆவணங்கள், தரநிலைகள், ஆராய்ச்சி, Microsoft தீர்வுகள், செயல்முறை கையேடுகள்)
