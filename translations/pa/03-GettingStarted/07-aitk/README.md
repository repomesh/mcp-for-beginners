# ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ ਲਈ AI ਟੂਲਕਿੱਟ ਐਕਸਟੇੰਸ਼ਨ ਤੋਂ ਸਰਵਰ ਦੀ ਵਰਤੋਂ ਕਰਨਾ

ਜਦੋਂ ਤੁਸੀਂ ਇੱਕ AI ਏਜੰਟ ਬਣਾ ਰਹੇ ਹੋ, ਤਾਂ ਇਹ ਸਿਰਫ ਸਮਾਰਟ ਜਵਾਬ ਬਣਾਉਣ ਬਾਰੇ ਨਹੀਂ ਹੈ; ਇਹ ਵੀ ਮਹੱਤਵਪੂਰਣ ਹੈ ਕਿ ਤੁਹਾਡੇ ਏਜੰਟ ਕੋਲ ਕਾਰਵਾਈ ਕਰਨ ਦੀ ਸਮਰੱਥਾ ਹੋਵੇ। ਇੱਥੇ ਮਾਡਲ ਸੰਦੇਸ਼ ਪ੍ਰੋਟੋਕੋਲ (MCP) ਅਹੰਕਾਰ ਹੈ। MCP ਏਜੰਟਾਂ ਲਈ ਬਾਹਰੀ ਟੂਲਾਂ ਅਤੇ ਸੇਵਾਵਾਂ ਤੱਕ ਪਹੁੰਚਨਾ ਸੌਖਾ ਬਣਾ ਦਿੰਦਾ ਹੈ। ਇਸ ਨੂੰ ਇਸ ਤਰ੍ਹਾਂ ਸੋਚੋ ਕਿ ਤੁਸੀਂ ਆਪਣੇ ਏਜੰਟ ਨੂੰ ਇੱਕ ਟੂਲਬਾਕਸ ਨਾਲ ਜੋੜ ਰਹੇ ਹੋ ਜੋ ਇਹ ਅਸਲ ਵਿੱਚ ਵਰਤ ਸਕਦਾ ਹੈ।

ਕਹੋ ਤੁਸੀਂ ਇੱਕ ਏਜੰਟ ਨੂੰ ਆਪਣੇ ਕੈਲਕੁਲੇਟਰ MCP ਸਰਵਰ ਨਾਲ ਜੋੜਦੇ ਹੋ। ਅਚਾਨਕ, ਤੁਹਾਡਾ ਏਜੰਟ ਸਿਰਫ “47 ਦਾ 89 ਗੁਣਾ ਕਿੰਨਾ ਹੈ?” ਵਰਗੇ ਪ੍ਰੰਪਟ ਨੂੰ ਪ੍ਰਾਪਤ ਕਰਕੇ ਗਣਿਤੀ ਕਾਰਵਾਈਆਂ ਕਰ ਸਕਦਾ ਹੈ—ਕੋਈ ਹਾਰਡਕੋਡ ਲਾਜਿਕ ਜਾਂ ਕਸਟਮ API ਬਣਾਉਣ ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ।

## ਜਾਇਜ਼ਾ

ਇਹ ਪਾਠ ਦੱਸਦਾ ਹੈ ਕਿ ਕਿਸ ਤਰ੍ਹਾਂ ਤੁਹਾਡੇ ਏਜੰਟ ਨੂੰ [AI ਟੂਲਕਿੱਟ](https://aka.ms/AIToolkit) ਐਕਸਟੇੰਸ਼ਨ ਨਾਲ ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ ਵਿੱਚ ਕੈਲਕੁਲੇਟਰ MCP ਸਰਵਰ ਜੋੜ ਕੇ ਪ੍ਰाकृतिक ਭਾਸ਼ਾ ਰਾਹੀਂ ਜੋੜ ਅਤੇ ਘਟਾ, ਗੁਣਾ ਅਤੇ ਭਾਗ ਵਰਗੀਆਂ ਗਣਤੀ ਕਾਰਵਾਈਆਂ ਕਰਵਾਈਆਂ ਜਾ ਸਕਦੀਆਂ ਹਨ।

AI ਟੂਲਕਿੱਟ ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ ਲਈ ਇੱਕ ਸ਼ਕਤੀਸ਼ਾਲੀ ਐਕਸਟੇੰਸ਼ਨ ਹੈ ਜੋ ਏਜੰਟ ਵਿਕਾਸ ਨੂੰ ਆਸਾਨ ਬਣਾਉਂਦਾ ਹੈ। AI ਇੰਜੀਨੀਅਰ ਸਥਾਨਕ ਜਾਂ ਕਲਾਊਡ ਵਿੱਚ ਜਨਰੇਟਿਵ AI ਮਾਡਲ ਵਿਕਸਤ ਅਤੇ ਟੈਸਟ ਕਰਕੇ ਆਸਾਨੀ ਨਾਲ AI ਐਪਲੀਕੇਸ਼ਨ ਬਣਾ ਸਕਦੇ ਹਨ। ਇਹ ਐਕਸਟੇੰਸ਼ਨ ਅੱਜ ਮੌਜੂਦ ਜ਼ਿਆਦਾਤਰ ਮੁੱਖ ਜਨਰੇਟਿਵ ਮਾਡਲਾਂ ਨਾਲ ਸਹਿਯੋਗ ਕਰਦਾ ਹੈ।

*ਨੋਟ*: AI ਟੂਲਕਿੱਟ ਇਸ ਵੇਲੇ Python ਅਤੇ TypeScript ਨੂੰ ਸਹਿਯੋਗ ਕਰਦਾ ਹੈ।

## ਸਿੱਖਣ ਦੇ ਉਦੇਸ਼

ਇਸ ਪਾਠ ਦੇ ਅੰਤ ਤੱਕ, ਤੁਸੀਂ ਸਮਰੱਥ ਹੋਵੋਗੇ:

- AI ਟੂਲਕਿੱਟ ਰਾਹੀਂ MCP ਸਰਵਰ ਨੂੰ ਵਰਤਣ ਲਈ।
- ਏਜੰਟ ਕੰਫਿਗਰੇਸ਼ਨ ਤਿਆਰ ਕਰਨ ਲਈ, ਜੋ MCP ਸਰਵਰ ਪ੍ਰਦਾਨ ਕੀਤੇ ਗਏ ਟੂਲਾਂ ਨੂੰ ਖੋਜ ਅਤੇ ਵਰਤਣ ਵਿੱਚ ਸਮਰੱਥ ਹੋਵੇ।
- ਪ੍ਰाकृतिक ਭਾਸ਼ਾ ਰਾਹੀਂ MCP ਟੂਲਾਂ ਦੀ ਵਰਤੋਂ ਕਰਨ ਲਈ।

### -2- ਏਜੰਟ ਲਈ ਸਿਸਟਮ ਪ੍ਰੰਪਟ ਬਣਾਓ

ਏਜੰਟ ਬਣਾਉਣ ਤੋਂ ਬਾਅਦ, ਇਸ ਦੀ ਵਿਅਕਤੀਗਤਤਾ ਅਤੇ ਮਕਸਦ ਨਿਰਧਾਰਿਤ ਕਰਨ ਦਾ ਸਮਾਂ ਹੈ। ਇਸ ਭਾਗ ਵਿੱਚ, ਤੁਸੀਂ **Generate system prompt** ਫੀਚਰ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਏਜੰਟ ਦੀ ਨਿਯਤ ਵਿਵਹਾਰ ਦੀ ਵਿਆਖਿਆ ਕਰੋਗੇ—ਇਸ ਮਾਮਲੇ ਵਿੱਚ ਕੈਲਕੁਲੇਟਰ ਏਜੰਟ—ਅਤੇ ਮਾਡਲ ਤੁਹਾਡੇ ਲਈ ਸਿਸਟਮ ਪ੍ਰੰਪਟ ਤਿਆਰ ਕਰੇਗਾ।

![Visual Studio Code ਲਈ AI ਟੂਲਕਿੱਟ ਦਾ "Calculator Agent" ਇੰਟਰਫੇਸ ਸਕ੍ਰੀਨਸ਼ਾਟ ਜਿਸ ਵਿੱਚ ਇੱਕ ਮੋਡਲ ਖਿੜਕੀ ਖੁੱਲੀ ਹੋਈ ਹੈ ਜਿਸਦਾ ਸਿਰਲੇਖ "Generate a prompt" ਹੈ। ਮੋਡਲ ਵਿੱਚ ਕਿਹਾ ਗਿਆ ਹੈ ਕਿ ਤੁਹਾਡੀ ਬੁਨਿਆਦੀ ਜਾਣਕਾਰੀ ਸਾਂਝੀ ਕਰਕੇ ਪ੍ਰੰਪਟ ਟੈਮਪਲੇਟ ਬਣਾਇਆ ਜਾ ਸਕਦਾ ਹੈ। ਇੱਕ ਟੈਕਸਟ ਬਾਕਸ ਵਿੱਚ ਨਮੂਨਾ ਸਿਸਟਮ ਪ੍ਰੰਪਟ ਹੈ: "ਤੁਸੀਂ ਇੱਕ ਮਦਦਗਾਰ ਅਤੇ ਪ੍ਰਭਾਵਸ਼ਾਲੀ ਗਣਿਤ ਸਹਾਇਕ ਹੋ। ਜਦੋਂ ਬੁਨਿਆਦੀ ਹਿਸਾਬ ਨਾਲ ਜੁੜੇ ਸਮੱਸਿਆ ਦਿੱਤੀ ਜਾਵੇ, ਤੁਸੀਂ ਸਹੀ ਨਤੀਜਾ ਦਿੰਦੇ ਹੋ।" ਟੈਕਸਟ ਬਾਕਸ ਥੱਲੇ "Close" ਅਤੇ "Generate" ਬਟਨ ਹਨ। ਪਿਛੋਕੜ ਵਿੱਚ, ਏਜੰਟ ਦੀ ਸੰਰਚਨਾ ਦਾ ਹਿੱਸਾ, ਚੁਣਿਆ ਹੋਇਆ ਮਾਡਲ "OpenAI GPT-4o (via GitHub)" ਅਤੇ ਸਿਸਟਮ ਅਤੇ ਯੂਜ਼ਰ ਪ੍ਰੰਪਟਾਂ ਦੇ ਖੇਤਰ ਦਿੱਖ ਰਹੇ ਹਨ।](../../../../translated_images/pa/aitk-generate-prompt.ba9e69d3d2bbe2a2.webp)

1. **Prompts** ਭਾਗ ਲਈ, **Generate system prompt** ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਇਹ ਬਟਨ ਪ੍ਰੰਪਟ ਬਿਲਡਰ ਖੋਲ੍ਹਦਾ ਹੈ ਜੋ ਏਜੰਟ ਲਈ ਸਿਸਟਮ ਪ੍ਰੰਪਟ ਤਿਆਰ ਕਰਨ ਲਈ AI ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ।
1. **Generate a prompt** ਵਿੰਡੋ ਵਿੱਚ ਹੇਠਾਂ ਦੇ ਬਿਆਨ ਨੂੰ ਟਾਈਪ ਕਰੋ: `You are a helpful and efficient math assistant. When given a problem involving basic arithmetic, you respond with the correct result.`
1. **Generate** ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਨੋਟੀਫਿਕੇਸ਼ਨ ਤੁਰੰਤ ਨੀਵੇਂ-ਸੱਜੇ ਕੌਰਨਰ ਵਿੱਚ ਦਿਖੇਗੀ ਕਿ ਸਿਸਟਮ ਪ੍ਰੰਪਟ ਬਣਾਇਆ ਜਾ ਰਿਹਾ ਹੈ। ਜਦੋਂ ਪ੍ਰੰਪਟ ਬਣਾਉਣਾ ਖਤਮ ਹੋ ਜਾਵੇਗਾ, ਉਹ ਪ੍ਰੰਪਟ **Agent (Prompt) Builder** ਵਿੱਚ **System prompt** ਖੇਤਰ ਵਿੱਚ ਦਿਖਾਈ ਦੇਵੇਗਾ।
1. **System prompt** ਦੀ ਸਮੀਖਿਆ ਕਰੋ ਅਤੇ ਜ਼ਰੂਰਤ ਪਵੇ ਤਾਂ ਇਸ ਵਿੱਚ ਬਦਲਾਅ ਕਰੋ।

### -3- MCP ਸਰਵਰ ਬਣਾਓ

ਹੁਣ ਜਦੋਂ ਤੁਸੀਂ ਏਜੰਟ ਦਾ ਸਿਸਟਮ ਪ੍ਰੰਪਟ ਨਿਰਧਾਰਿਤ ਕਰ ਲਿਆ ਹੈ—ਜੋ ਇਸਦੇ ਵਿਵਹਾਰ ਅਤੇ ਜਵਾਬਾਂ ਦੀ ਮਾਰਗਦਰਸ਼ਨ ਕਰਦਾ ਹੈ—ਫਿਰ ਸਮਾਂ ਆ ਗਿਆ ਹੈ ਕਿ ਏਜੰਟ ਨੂੰ ਵਾਸਤਵਿਕ ਸ਼ਕਤੀਆਂ ਨਾਲ ਸਾਜੋ। ਇਸ ਭਾਗ ਵਿੱਚ, ਤੁਸੀਂ ਕੈਲਕੁਲੇਟਰ MCP ਸਰਵਰ ਬਣਾਵੋਗੇ ਜੋ ਜੋੜ, ਘਟਾਉ, ਗੁਣਾ ਅਤੇ ਭਾਗ ਦੇ ਕੰਮ ਕਰਨ ਲਈ ਟੂਲ ਦਿੱਤੇ ਜਾਣਗੇ। ਇਹ ਸਰਵਰ ਤੁਹਾਡੇ ਏਜੰਟ ਨੂੰ ਪ੍ਰाकृतिक ਭਾਸ਼ਾ ਪ੍ਰੰਪਟਾਂ ਦੇ ਜਵਾਬ ਵਿੱਚ ਰੀਅਲ-ਟਾਈਮ ਗਣਿਤ ਕਾਰਵਾਈਆਂ ਕਰਨ ਯੋਗ ਬਣਾਏਗਾ।

!["Visual Studio Code ਲਈ AI ਟੂਲਕਿੱਟ ਐਕਸਟੇੰਸ਼ਨ ਵਿੱਚ Calculator Agent ਇੰਟਰਫੇਸ ਦੇ ਹੇਠਲੇ ਭਾਗ ਦੀ ਸਕ੍ਰੀਨਸ਼ਾਟ ਜੋ “Tools” ਅਤੇ “Structure output” ਲਈ ਵਧਾਈ ਯੋਗ ਮੀਨੂ ਦਰਸਾਉਂਦਾ ਹੈ, ਨਾਲ ਹੀ “Choose output format” ਡ੍ਰੌਪਡਾਊਨ ਜੋ “text” ਤੇ ਸੈੱਟ ਹੈ। ਸੱਜੇ ਵੱਲ, ਇੱਕ ਬਟਨ “+ MCP Server” ਦਿੱਤਾ ਹੈ Model Context Protocol ਸਰਵਰ ਸ਼ਾਮਲ ਕਰਨ ਲਈ। ਟੂਲਜ਼ ਭਾਗ ਦੇ ਉੱਪਰ ਇੱਕ ਚਿੱਤਰ ਆਈਕਨ ਪਲੇਸਹੋਲਡਰ ਦਿੱਤਾ ਹੈ।](../../../../translated_images/pa/aitk-add-mcp-server.9742cfddfe808353.webp)

AI ਟੂਲਕਿੱਟ ਆਪਣੇ MCP ਸਰਵਰ ਬਣਾਉਣ ਲਈ ਸਾਂਚੇ (ਟੈਂਪਲੇਟ) ਮੁਹੱਈਆ ਕਰਵਾਉਂਦਾ ਹੈ। ਅਸੀਂ ਕੈਲਕੁਲੇਟਰ MCP ਸਰਵਰ ਬਣਾਉਣ ਲਈ Python ਸਾਂਚਾ ਵਰਤਾਂਗੇ।

*ਨੋਟ*: AI ਟੂਲਕਿੱਟ ਇਸ ਵੇਲੇ Python ਅਤੇ TypeScript ਨੂੰ ਸਹਿਯੋਗ ਕਰਦਾ ਹੈ।

1. **Agent (Prompt) Builder** ਦੇ **Tools** ਖੰਡ ਵਿੱਚ, **+ MCP Server** ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ। ਇਹ ਕਮਾਂਡ ਪੈਲੇਟ ਰਾਹੀਂ ਸੈਟਅੱਪ ਵਿਜ਼ਾਰਡ ਚਲਾਏਗਾ।
1. **+ Add Server** ਚੁਣੋ।
1. **Create a New MCP Server** ਚੁਣੋ।
1. ਸਾਂਚਾ ਵਜੋਂ **python-weather** ਚੁਣੋ।
1. MCP ਸਰਵਰ ਟੈਂਪਲੇਟ ਸੇਵ ਕਰਨ ਲਈ **Default folder** ਚੁਣੋ।
1. ਸਰਵਰ ਦਾ ਨਾਂ ਦਿਓ: **Calculator**
1. ਇੱਕ ਨਵੀਂ ਵਿਜ਼ੂਅਲ ਸਟੂਡੀਓ ਕੋਡ ਵਿੰਡੋ ਖੁਲੇਗੀ। **Yes, I trust the authors** ਨੂੰ ਚੁਣੋ।
1. ਟਰਮੀਨਲ ਵਿੱਚ (**Terminal** > **New Terminal**) ਵਰਚੁਅਲ ਐਨਵਾਇਰਨਮੈਂਟ ਬਣਾਓ: `python -m venv .venv`
1. ਟਰਮੀਨਲ ਵਿੱਚ ਵਰਚੁਅਲ ਐਨਵਾਇਰਨਮੈਂਟ ਐਕਟੀਵੇਟ ਕਰੋ:
    1. Windows ਲਈ - `.venv\Scripts\activate`
    1. macOS/Linux ਲਈ - `source .venv/bin/activate`
1. ਟਰਮੀਨਲ ਵਿੱਚ ਡੀਪੈਂਡੈਂਸੀਜ਼ ਇੰਸਟਾਲ ਕਰੋ: `pip install -e .[dev]`
1. **Activity Bar** ਦੇ **Explorer** ਦ੍ਰਿਸ਼ ਵਿੱਚ, **src** ਡਾਇਰੈਕਟਰੀ ਨੂੰ ਵਧਾਓ ਅਤੇ **server.py** ਫ਼ਾਈਲ ਖੋਲ੍ਹੋ।
1. **server.py** ਵਿੱਚ ਦਿੱਤਾ ਕੋਡ ਹਟਾ ਕੇ ਨਵਾਂ ਕੋਡ ਸੇਵ ਕਰੋ:

    ```python
    """
    Sample MCP Calculator Server implementation in Python.

    
    This module demonstrates how to create a simple MCP server with calculator tools
    that can perform basic arithmetic operations (add, subtract, multiply, divide).
    """
    
    from mcp.server.fastmcp import FastMCP
    
    server = FastMCP("calculator")
    
    @server.tool()
    def add(a: float, b: float) -> float:
        """Add two numbers together and return the result."""
        return a + b
    
    @server.tool()
    def subtract(a: float, b: float) -> float:
        """Subtract b from a and return the result."""
        return a - b
    
    @server.tool()
    def multiply(a: float, b: float) -> float:
        """Multiply two numbers together and return the result."""
        return a * b
    
    @server.tool()
    def divide(a: float, b: float) -> float:
        """
        Divide a by b and return the result.
        
        Raises:
            ValueError: If b is zero
        """
        if b == 0:
            raise ValueError("Cannot divide by zero")
        return a / b
    ```

### -4- ਕੈਲਕੁਲੇਟਰ MCP ਸਰਵਰ ਨਾਲ ਏਜੰਟ ਚਲਾਓ

ਹੁਣ ਜਦੋਂ ਤੁਹਾਡੇ ਕੋਲ ਟੂਲ ਹਨ, ਦਾ ਸਮਾਂ ਹੈ ਉਹਨਾਂ ਦੀ ਵਰਤੋਂ ਕਰਨ ਦਾ! ਇਸ ਭਾਗ ਵਿੱਚ, ਤੁਸੀਂ ਏਜੰਟ ਨੂੰ ਪ੍ਰੰਪਟ ਸੌਂਪ ਕੇ ਇਹ ਪਰਖੋਗੇ ਕਿ ਕੀ ਏਜੰਟ ਕੈਲਕੁਲੇਟਰ MCP ਸਰਵਰ ਵਿੱਚੋਂ ਸਹੀ ਟੂਲ ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ ਜਾਂ ਨਹੀਂ।

![Visual Studio Code ਲਈ AI ਟੂਲਕਿੱਟ ਐਕਸਟੇੰਸ਼ਨ ਵਿੱਚ Calculator Agent ਇੰਟਰਫੇਸ ਦੀ ਸਕ੍ਰੀਨਸ਼ਾਟ। ਖੱਬੇ ਪੈਨਲ ਵਿੱਚ, “Tools” ਹੇਠਾਂ ਇੱਕ MCP ਸਰਵਰ ਨਾਮ ਨਾਲ local-server-calculator_server ਸ਼ਾਮਲ ਹੈ, ਜਿੱਥੇ ਚਾਰ ਉਪਲਬਧ ਟੂਲ ਹਨ: ਜੋੜ, ਵੰਡ, ਗੁਣਾ, ਅਤੇ ਘਟਾਓ। ਇੱਕ ਬੈਜ ਦਿਖਾ ਰਿਹਾ ਹੈ ਕਿ ਚਾਰ ਟੂਲ ਐਕਟਿਵ ਹਨ। ਹੇਠਾਂ ਸੰਕੁਚਿਤ “Structure output” ਖੰਡ ਹੈ ਅਤੇ ਨੀਲਾ “Run” ਬਟਨ ਹੈ। ਸੱਜੇ ਪੈਨਲ ਵਿੱਚ, “Model Response” ਹੇਠਾਂ, ਏਜੰਟ ਗੁਣਾ ਅਤੇ ਘਟਾਓ ਟੂਲਾਂ ਨੂੰ ਹੁਕਮ ਦੇ ਰਿਹਾ ਹੈ ਉਨ੍ਹਾਂ ਵਿੱਚ ਇਨਪੁੱਟ {"a": 3, "b": 25} ਅਤੇ {"a": 75, "b": 20} ਹਨ। ਆਖਰੀ “Tool Response” 75.0 ਹੈ। ਹੇਠਾਂ “View Code” ਬਟਨ ਹੈ।](../../../../translated_images/pa/aitk-agent-response-with-tools.e7c781869dc8041a.webp)

ਤੁਸੀਂ ਆਪਣੇ ਲੋਕਲ ਡਿਵੈਲਪਮੈਂਟ ਮਸ਼ੀਨ 'ਤੇ MCP ਕਲਾਇੰਟ ਵਜੋਂ **Agent Builder** ਰਾਹੀਂ ਕੈਲਕੁਲੇਟਰ MCP ਸਰਵਰ ਚਲਾਓਗੇ।

1. MCP ਸਰਵਰ ਦੀ ਡੀਬੱਗਿੰਗ ਸ਼ੁਰੂ ਕਰਨ ਲਈ `F5` ਦਬਾਓ। **Agent (Prompt) Builder** ਨਵੀਂ ਐਡੀਟਰ ਟੈਬ ਵਿੱਚ ਖੁਲੇਗਾ। ਟਰਮੀਨਲ ਵਿੱਚ ਸਰਵਰ ਦੀ ਸਥਿਤੀ ਵੇਖੀ ਜਾ ਸਕਦੀ ਹੈ।
1. **Agent (Prompt) Builder** ਵਿੱਚ **User prompt** ਖੇਤਰ ਵਿੱਚ ਇਹ ਪ੍ਰੰਪਟ ਦਿਓ: `I bought 3 items priced at $25 each, and then used a $20 discount. How much did I pay?`
1. ਏਜੰਟ ਦੇ ਜਵਾਬ ਨੂੰ ਬਣਾਉਣ ਲਈ **Run** ਬਟਨ 'ਤੇ ਕਲਿੱਕ ਕਰੋ।
1. ਏਜੰਟ ਆਉਟਪੁੱਟ ਦੀ ਸਮੀਖਿਆ ਕਰੋ। ਮਾਡਲ ਨੂੰ ਇਹ ਨਤੀਜਾ ਨਿਕਲਣਾ ਚਾਹੀਦਾ ਹੈ ਕਿ ਤੁਸੀਂ **$55** ਦੀ ਅਦਾਇਗੀ ਕੀਤੀ ਹੈ।
1. ਇਹ ਤੁਹਾਡੇ ਲਈ ਜਾਣਣਾ ਬਹਿਤਰੀਨ ਹੈ:
    - ਏਜੰਟ ਗੁਣਾ ਅਤੇ ਘਟਾਓ ਟੂਲ ਚੁਣਦਾ ਹੈ ਗਣਨਾ ਵਿੱਚ ਮਦਦ ਲਈ।
    - ਗੁਣਾ ਟੂਲ ਲਈ `a` ਅਤੇ `b` ਮੁੱਲ ਨਿਰਧਾਰਿਤ ਕੀਤੇ ਜਾਂਦੇ ਹਨ।
    - ਘਟਾਓ ਟੂਲ ਲਈ `a` ਅਤੇ `b` ਮੁੱਲ ਨਿਰਧਾਰਿਤ ਕੀਤੇ ਜਾਂਦੇ ਹਨ।
    - ਹਰ ਟੂਲ ਤੋਂ ਜਵਾਬ ਸਬੰਧਤ **Tool Response** ਵਿੱਚ ਦਿੱਤਾ ਜਾਂਦਾ ਹੈ।
    - ਮਾਡਲ ਤੋਂ ਆਖਰੀ ਆਉਟਪੁੱਟ ਅੰਤਿਮ **Model Response** ਵਿੱਚ ਹੁੰਦਾ ਹੈ।
1. ਐਜੰਟ ਦੀ ਹੋਰ ਪਰਖ ਕਰਨ ਲਈ ਹੋਰ ਪ੍ਰੰਪਟ ਭੇਜੋ। ਤੁਸੀਂ ਮੌਜੂਦਾ ਪ੍ਰੰਪਟ ਨੂੰ ਸੋਧ ਸਕਦੇ ਹੋ **User prompt** ਖੇਤਰ ਵਿੱਚ ਜਾ ਕੇ।
1. ਜਦੋਂ ਤੁਸੀਂ ਟੈਸਟਿੰਗ ਮੁਕੰਮਲ ਕਰ ਲੈੋ, ਤਾਂ ਟਰਮੀਨਲ ਵਿੱਚ **CTRL/CMD+C** ਦਬਾ ਕੇ ਸਰਵਰ ਨੂੰ ਰੋਕੋ।

## ਕੰਮ

ਆਪਣੀ **server.py** ਫਾਇਲ ਵਿੱਚ ਇੱਕ ਹੋਰ ਟੂਲ ਏਂਟਰੀ ਸ਼ਾਮਲ ਕਰਨ ਦੀ ਕੋਸ਼ਿਸ਼ ਕਰੋ (ਉਦਾਹਰਨ ਵਜੋਂ, ਕਿਸੇ ਨੰਬਰ ਦੀ ਵਰਗਮੂਲ ਵਾਪਸ ਕਰਨਾ)। ਹੋਰ ਪ੍ਰੰਪਟ ਭੇਜੋ ਜੋ ਤੁਹਾਨੂੰ ਤੁਹਾਡਾ ਨਵਾਂ ਟੂਲ (ਜਾਂ ਮੌਜੂਦਾ ਟੂਲ) ਵਰਤਣ ਲਈ ਲੋੜੀਂਦਾ ਹੈ। ਨਵੇਂ ਟੂਲ ਲੋਡ ਕਰਨ ਲਈ ਸਰਵਰ ਨੂੰ ਮੁੜ ਚਲਾਓ।

- ਤੁਸੀਂ MCP ਸਰਵਰਾਂ ਵਿੱਚ ਨਵੇਂ ਟੂਲ ਸ਼ਾਮਲ ਕਰ ਸਕਦੇ ਹੋ, ਜਿਸ ਨਾਲ ਏਜੰਟ ਦੀ ਸਮਰੱਥਾ ਵਿਕਸਿਤ ਹੁੰਦੀ ਹੈ।
- AI ਟੂਲਕਿੱਟ ਵਿੱਚ ਸਾਂਚੇ ਸ਼ਾਮਲ ਹਨ (ਜਿਵੇਂ Python MCP ਸਰਵਰ ਸਾਂਚੇ) ਜੋ ਕਸਟਮ ਟੂਲ ਬਣਾਉਣ ਨੂੰ ਸੌਖਾ ਬਣਾਉਂਦੇ ਹਨ।
