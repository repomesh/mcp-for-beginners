# سیمپل چلانا

یہاں ہم فرض کرتے ہیں کہ آپ کے پاس پہلے سے ایک کام کرنے والا سرور کوڈ موجود ہے۔ براہ کرم پہلے کے ابواب میں سے کسی سرور کو تلاش کریں۔

## mcp.json سیٹ اپ کریں

یہاں ایک فائل ہے جسے آپ حوالہ کے طور پر استعمال کرتے ہیں، [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json).

سرور اندراج کو اس طرح تبدیل کریں کہ آپ کے سرور کے مطلق راستے کی نشاندہی ہو، بشمول وہ مکمل کمانڈ جو چلانے میں ضروری ہو۔

مثال کے طور پر اوپر دی گئی فائل میں سرور اندراج اس طرح دکھائی دیتا ہے:

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

آپ کو GitHub ریپوزیٹری کے روٹ کو بھی درج کرنا پڑ سکتا ہے، جسے کمانڈ `git rev-parse --show-toplevel` سے حاصل کیا جا سکتا ہے۔

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

یہ اس طرح کمانڈ چلانے کے برابر ہے: `node build/index.js`.

- اس سرور اندراج کو اس طرح تبدیل کریں کہ یہ آپ کے سرور فائل کی جگہ یا آپ کے منتخب کردہ رن ٹائم اور سرور مقام کے لحاظ سے آپ کے سرور کو اسٹارٹ اپ کرنے کے لئے درکار ہو۔

## سرور میں فیچرز استعمال کریں

- ایک بار جب آپ *mcp.json* کو *./vscode* فولڈر میں شامل کر لیتے ہیں، تو `play` آئیکن پر کلک کریں،

    ٹولنگ آئیکن کے بدلنے کو دیکھیں جو دستیاب ٹولز کی تعداد میں اضافہ کرے گا۔ ٹولنگ آئیکن GitHub Copilot میں چیٹ فیلڈ کے بالکل اوپر واقع ہے۔

## ٹول چلائیں

- اپنی چیٹ ونڈو میں اپنے ٹول کی وضاحت سے میل کھاتا کوئی پرامپٹ ٹائپ کریں۔ مثلاً ٹول `add` کو چلانے کے لیے کچھ اس طرح ٹائپ کریں "add 3 to 20"۔

    آپ کو چیٹ ٹیکسٹ باکس کے اوپر ایک ٹول دکھائی دے گا جو آپ کو اس ٹول کو چلانے کے لیے منتخب کرنے کا اشارہ دے گا، جیسا کہ اس ویژول میں:

    ![VS Code indicating it wanting to run a tool](../../../../../translated_images/ur/vscode-agent.d5a0e0b897331060.webp)

    ٹول منتخب کرنے پر ایک عددی نتیجہ "23" نظر آنا چاہیے اگر آپ کا پرامپٹ جیسا کہ ہم نے اوپر بتایا تھا۔
