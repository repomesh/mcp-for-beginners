# MCP میں جدید موضوعات

[![جدید MCP: محفوظ، قابل توسع، اور ملٹی موڈل AI ایجنٹس](../../../translated_images/ur/06.42259eaf91fccfc6.webp)](https://youtu.be/4yjmGvJzYdY)

_(اس سبق کی ویڈیو دیکھنے کے لیے اوپر تصویر پر کلک کریں)_

یہ باب ماڈل کانٹیکسٹ پروٹوکول (MCP) کے نفاذ میں متعدد جدید موضوعات کا احاطہ کرتا ہے، جن میں ملٹی موڈل انٹیگریشن، توسیع پذیری، سلامتی کی بہترین طریقہ کار، اور انٹرپرائز انٹیگریشن شامل ہیں۔ یہ موضوعات مضبوط اور پیداوار کے لیے تیار MCP ایپلی کیشنز بنانے کے لیے انتہائی اہم ہیں جو جدید AI نظاموں کی ضروریات کو پورا کر سکیں۔

## جائزہ

یہ سبق ماڈل کانٹیکسٹ پروٹوکول کے نفاذ میں جدید تصورات کو دریافت کرتا ہے، خاص طور پر ملٹی موڈل انٹیگریشن، توسیع پذیری، سلامتی کی بہترین طریقہ کار، اور انٹرپرائز انٹیگریشن پر توجہ مرکوز کرتا ہے۔ یہ موضوعات پیداواری معیار کی MCP ایپلی کیشنز بنانے کے لیے ضروری ہیں جو انٹرپرائز ماحول میں پیچیدہ تقاضوں کو سنبھال سکیں۔

> **آگے کی جھلک:** نیچے متعدد موضوعات `2026-07-28` MCP وضاحت ریلیز امیدوار سے متاثر ہیں — روٹ کانٹیکسٹ (5.4) اور سیمپلنگ (5.6) ان پرمیٹیوز پر مبنی ہیں جنہیں ریلیز امیدوار نے غیر مستعمل قرار دیا ہے، اور پروٹوکول فیچرز (5.16) میں حوالہ دی گئی تجرباتی ٹاسکس خصوصیت ایک مخصوص ٹاسکس ایکسٹینشن میں منتقل ہو گئی ہے۔ تفصیلات کے لیے دیکھیں [MCP میں کیا بدل رہا ہے: 2026-07-28 ریلیز امیدوار](../01-CoreConcepts/mcp-2026-07-28-release-candidate.md)۔

## سیکھنے کے مقاصد

اس سبق کے اختتام تک، آپ قابل ہوں گے کہ:

- MCP فریم ورکس میں ملٹی موڈل صلاحیتوں کا نفاذ کریں
- زیادہ مطالبہ والے منظرناموں کے لیے MCP کے توسیع پذیر فن تعمیرات ڈیزائن کریں
- MCP کی سلامتی اصولوں کے مطابق سلامتی کی بہترین طریقہ کار اپنائیں
- MCP کو انٹرپرائز AI نظاموں اور فریم ورکس کے ساتھ مربوط کریں
- پیداواری ماحول میں کارکردگی اور اعتبار کو بہتر بنائیں

## کلیدی نکات

- ملٹی موڈل MCP نفاذ AI صلاحیتوں کو متن کی پراسیسنگ سے باہر بڑھاتے ہیں
- توسیع پذیری انٹرپرائز تعیناتیوں کے لیے ضروری ہے اور افقی اور عمودی اسکیلنگ کے ذریعے حل کی جا سکتی ہے
- جامع سلامتی کے اقدامات ڈیٹا کی حفاظت کرتے ہیں اور مناسب رسائی کنٹرول کو یقینی بناتے ہیں
- Azure OpenAI اور Microsoft AI Foundry جیسے پلیٹ فارمز کے ساتھ انٹرپرائز انٹیگریشن MCP صلاحیتوں کو بڑھاتا ہے
- جدید MCP نفاذ کو بہتر فن تعمیر اور محتاط وسائل کے انتظام سے فائدہ حاصل ہوتا ہے

## مشق

ایک مخصوص استعمال کے لیے انٹرپرائز گریڈ MCP نفاذ ڈیزائن کریں:

1. اپنے استعمال کے کیس کے لیے ملٹی موڈل ضروریات کی شناخت کریں
2. حساس ڈیٹا کی حفاظت کے لیے درکار سلامتی کنٹرولز کا خاکہ تیار کریں
3. ایک ایسا اسکیل ایبل فن تعمیر ڈیزائن کریں جو مختلف بوجھ کو سنبھال سکے
4. انٹرپرائز AI نظاموں کے ساتھ انٹیگریشن پوائنٹس کی منصوبہ بندی کریں
5. ممکنہ کارکردگی کی رکاوٹوں اور ان کے حل کی حکمت عملی دستاویز کریں
