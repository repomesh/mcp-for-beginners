# 🚀 موڈل کانٹیکسٹ پروٹوکول (MCP) نصاب برائے مبتدی

## **C#, Java, JavaScript, Rust, Python، اور TypeScript میں عملی کوڈ کے مثالوں کے ساتھ MCP سیکھیں**

## 🧠 موڈل کانٹیکسٹ پروٹوکول نصاب کا جائزہ
موڈل کانٹیکسٹ پروٹوکول کی آپ کی سفر میں خوش آمدید! اگر آپ نے کبھی یہ سوچا ہے کہ AI ایپلیکیشنز مختلف ٹولز اور سروسز سے کیسے بات چیت کرتے ہیں، تو آپ ایک شاندار حل دریافت کرنے جا رہے ہیں جو ڈویلپرز کے لیے ذہین نظام بنانے کے طریقہ کار کو بدل رہا ہے۔

MCP کو AI ایپلیکیشنز کے لیے ایک عالمگیر مترجم سمجھیں - جیسے USB پورٹس آپ کو کسی بھی ڈیوائس کو کمپیوٹر سے منسلک کرنے کی اجازت دیتی ہیں، MCP بھی AI ماڈلز کو کسی بھی ٹول یا سروس سے معیاری طریقے سے جڑنے دیتا ہے۔ چاہے آپ اپنا پہلا چیٹ بوٹ بنا رہے ہوں یا پیچیدہ AI ورک فلو پر کام کر رہے ہوں، MCP کو سمجھنا آپ کو زیادہ قابل اور لچکدار ایپلیکیشنز بنانے کی طاقت دے گا۔

یہ نصاب آپ کے سیکھنے کے سفر کے لیے صبر اور توجہ کے ساتھ تیار کیا گیا ہے۔ ہم آسان تصورات سے شروع کریں گے جو آپ پہلے ہی سمجھتے ہیں اور تدریجاً عملی مشق کے ذریعے آپ کی مہارت کو بڑھائیں گے، آپ کی پسندیدہ پروگرامنگ زبان میں۔ ہر قدم میں واضح وضاحتیں، عملی مثالیں، اور بھرپور حوصلہ افزائی شامل ہوگی۔

جب آپ یہ سفر مکمل کریں گے تو آپ کو اپنی MCP سرورز بنانے، انہیں مقبول AI پلیٹ فارمز کے ساتھ منسلک کرنے، اور سمجھنے کا اعتماد حاصل ہوگا کہ یہ ٹیکنالوجی AI ترقی کے مستقبل کو کیسے تشکیل دے رہی ہے۔ آئیے اس دلچسپ مہم کا آغاز ایک ساتھ کرتے ہیں!

### سرکاری دستاویزات اور مواصفات

یہ نصاب **MCP Specification 2025-11-25** کے مطابق ہے (تازہ ترین مستحکم ریلیز)۔ MCP مواصفات تاریخ پر مبنی ورژننگ (YYYY-MM-DD فارمیٹ) استعمال کرتی ہے تاکہ پروٹوکول ورژن کو واضح طور پر ٹریک کیا جا سکے۔

> **آئندہ کے لیے دیکھیں:** اگلے مواصفات ورژن، **2026-07-28** کے ریلیز امیدوار کی شپمنٹ 28 جولائی 2026 کو شیڈول ہے۔ یہ پروٹوکول کو ٹرانسپورٹ لیئر پر بغیر ریاست بنا دیتا ہے، ایکسٹینشنز فریم ورک (MCP Apps, Tasks) کو رسمی طور پر متعارف کراتا ہے، اجازت کو سخت کرتا ہے، اور Roots، Sampling، اور Logging کو ختم کرتا ہے۔ مکمل وضاحت کے لیے دیکھیں [What's Changing in MCP: The 2026-07-28 Release Candidate](./01-CoreConcepts/mcp-2026-07-28-release-candidate.md)۔

## سیکھنے کے مقاصد

اس نصاب کے اختتام تک، آپ اپنے نئے صلاحیتوں کے بارے میں پر اعتماد اور پُرجوش ہوں گے۔ یہاں وہ چیزیں ہیں جو آپ حاصل کریں گے:

• **MCP کی بنیادی باتیں سمجھیں**: آپ سمجھ جائیں گے کہ موڈل کانٹیکسٹ پروٹوکول کیا ہے اور یہ AI ایپلیکیشنز کے ساتھ کام کرنے کے طریقے کو کیسے بدل رہا ہے، ایسی مثالوں اور تشبیہوں کے ساتھ جو آپ کو سمجھ آ سکیں۔

• **اپنا پہلا MCP سرور بنائیں**: اپنی پسند کی پروگرامنگ زبان میں ایک کام کرنے والا MCP سرور بنائیں گے، آسان مثالوں سے شروع کرتے ہوئے اور آہستہ آہستہ اپنی مہارت کو بڑھائیں گے۔

• **AI ماڈلز کو حقیقی آلات سے جوڑیں**: یہ سیکھیں گے کہ AI ماڈلز اور حقیقی سروسز کے درمیان کیسے رابطہ قائم کیا جائے، تاکہ آپ کی ایپلیکیشنز کو طاقتور نئی صلاحیتیں ملیں۔

• **سیکیورٹی کی بہترین مشقیں نافذ کریں**: سمجھیں گے کہ MCP پر نفاذ کو کیسے محفوظ اور محفوظ رکھا جائے، تاکہ آپ کی ایپلیکیشنز اور صارفین دونوں کی حفاظت ہو۔

• **اعتماد کے ساتھ تعینات کریں**: جانیں گے کہ اپنے MCP منصوبوں کو ترقی سے پیداوار تک کیسے لایا جائے، عملی تعیناتی کی حکمت عملیوں کے ساتھ جو حقیقی دنیا میں کام کرتی ہیں۔

• **MCP کمیونٹی میں شامل ہوں**: آپ ایسی بڑھتی ہوئی کمیونٹی کا حصہ بنیں گے جو AI ایپلیکیشن کی ترقی کے مستقبل کو تشکیل دے رہی ہے۔

## ضروری پس منظر

MCP کی تفصیلات میں جانے سے پہلے، آئیے یقینی بنائیں کہ آپ کچھ بنیادی تصورات سے واقف ہیں۔ اگر آپ ان میں ماہر نہیں ہیں تو پریشان ہونے کی ضرورت نہیں - ہم آپ کو ہر چیز سمجھائیں گے جیسے جیسے آگے بڑھیں گے!

### پروٹوکولز کو سمجھنا (بنیاد)

پروٹوکول کو ایسے سمجھیں جیسے بات چیت کے اصول۔ جب آپ کسی دوست کو کال کرتے ہیں، تو آپ دونوں جانتے ہیں کہ جواب میں "ہیلو" کہنا، باری باری بات کرنا، اور اختتام پر "خدا حافظ" کہنا ضروری ہے۔ کمپیوٹر پروگرامز کو مؤثر رابطے کے لیے اسی طرح کے اصولوں کی ضرورت ہوتی ہے۔

MCP ایک پروٹوکول ہے - ایک متفقہ اصولوں کا مجموعہ جو AI ماڈلز اور ایپلیکیشنز کو ٹولز اور سروسز کے ساتھ مؤثر "بات چیت" کرنے میں مدد دیتا ہے۔ جیسے بات چیت کے اصول انسانوں کے رابطے کو آسان بناتے ہیں، ویسے ہی MCP AI ایپلیکیشنز کی بات چیت کو زیادہ قابل اعتماد اور طاقتور بناتا ہے۔

### کلائنٹ-سرور تعلقات (پروگرامز کیسے مل کر کام کرتے ہیں)

آپ روزانہ کلائنٹ-سرور تعلقات استعمال کرتے ہیں! جب آپ ویب براؤزر (کلائنٹ) استعمال کرتے ہوئے کسی ویب سائٹ پر جاتے ہیں، تو آپ ایک ویب سرور سے جڑتے ہیں جو آپ کو صفحہ کا مواد بھیجتا ہے۔ براؤزر جانتا ہے کہ معلومات کیسے طلب کرنی ہے، اور سرور جانتا ہے کہ کیسے جواب دینا ہے۔

MCP میں بھی ایسا ہی رشتہ ہوتا ہے: AI ماڈلز کلائنٹس کی طرح ہوتے ہیں جو معلومات یا کارروائیاں طلب کرتے ہیں، جبکہ MCP سرورز وہ صلاحیتیں فراہم کرتے ہیں۔ یہ ایسے ہے جیسے ایک مددگار معاون (سرور) ہو جس سے AI مخصوص کام کرنے کو کہہ سکتا ہے۔

### معیاری بنانا کیوں اہم ہے (چیزوں کو ساتھ کام کرنے کے قابل بنانا)

تصور کریں اگر ہر کار ساز مختلف شکل کے گیس پمپ استعمال کرے - آپ کو ہر کار کے لیے مختلف اڈاپٹر کی ضرورت ہوگی! معیاری بنانا یعنی ایک مشترکہ طریقہ کار پر اتفاق کرنا تاکہ چیزیں بغیر رکاوٹ کے کام کریں۔

MCP AI ایپلیکیشنز کے لیے یہ معیاری بنانا فراہم کرتا ہے۔ ہر AI ماڈل کو ہر ٹول کے ساتھ کام کرنے کے لیے الگ کوڈ بنانے کی بجائے، MCP ایک یونیورسل طریقہ فراہم کرتا ہے کہ وہ کیسے بات چیت کریں۔ اس کا مطلب ہے کہ ڈویلپرز ایک بار ٹول بنا سکتے ہیں اور وہ مختلف AI سسٹمز کے ساتھ کام کرے گا۔

## 🧭 آپ کا تعلیمی راستہ کا خاکہ

آپ کا MCP سفر سوچ سمجھ کر ترتیب دیا گیا ہے تاکہ آپ کا اعتماد اور مہارت بتدریج بڑھے۔ ہر مرحلہ نئے تصورات متعارف کراتا ہے اور آپ کی پہلے سے سیکھے ہوئے علم کو مضبوط کرتا ہے۔

### 🌱 بنیادی مرحلہ: بنیادی باتوں کو سمجھنا (ماڈیولز 0-2)

یہاں سے آپ کا سفر شروع ہوتا ہے! ہم آپ کو MCP کے تصورات متعارف کرائیں گے جو آپ کے جانے پہچانے تشبیہوں اور آسان مثالوں کے ذریعے ہوں گے۔ آپ سمجھیں گے کہ MCP کیا ہے، یہ کیوں موجود ہے، اور AI ترقی کی بڑی دنیا میں اس کا مقام کیا ہے۔

• **ماڈیول 0 - MCP کا تعارف**: ہم شروع کریں گے MCP کی اہمیت اور جدید AI ایپلیکیشنز کے لیے اس کی اہمیت کو سمجھنے سے۔ آپ MCP کے عملی استعمال کی حقیقی مثالیں دیکھیں گے اور سمجھیں گے کہ یہ ڈویلپرز کو درپیش عام مسائل کو کیسے حل کرتا ہے۔


• **ماڈیول 1 - بنیادی تصورات کی وضاحت**: یہاں آپ MCP کے اہم بنیادی اصول سیکھیں گے۔ ہم بہت سارے تشابہات اور بصری مثالیں استعمال کریں گے تاکہ یہ تصورات فطری اور قابل فہم محسوس ہوں۔

• **ماڈیول 2 - MCP میں سیکیورٹی**: سیکیورٹی سننے میں خوفناک لگ سکتی ہے، لیکن ہم آپ کو دکھائیں گے کہ MCP میں تعمیر شدہ حفاظتی خصوصیات شامل ہیں اور آپ کو بہترین طریقے سکھائیں گے جو آپ کی ایپلیکیشنز کی ابتدا سے حفاظت کرتے ہیں۔

### 🔨 تعمیراتی مرحلہ: اپنی پہلی عمل درآمدات بنانا (ماڈیول 3)

اب اصل مزہ شروع ہوتا ہے! آپ کو MCP سرورز اور کلائنٹس بنانے کا عملی تجربہ ملے گا۔ پریشان نہ ہوں - ہم سادہ سے شروع کریں گے اور ہر قدم پر رہنمائی کریں گے۔

اس ماڈیول میں متعدد عملی رہنما شامل ہیں جو آپ کو آپ کی پسندیدہ پروگرامنگ زبان میں مشق کرنے دیتے ہیں۔ آپ اپنا پہلا سرور بنائیں گے، اس سے کنیکٹ کرنے کے لیے کلائنٹ بنائیں گے، اور یہاں تک کہ VS Code جیسے مشہور ترقیاتی ٹولز کے ساتھ انٹیگریٹ کریں گے۔

ہر رہنما مکمل کوڈ کی مثالیں، مسئلہ حل کرنے کے نکات، اور وضاحتیں شامل ہے کہ ہم مخصوص ڈیزائن کے انتخاب کیوں کرتے ہیں۔ اس مرحلے کے آخر تک، آپ کے پاس کام کرنے والی MCP عمل درآمدات ہوں گی جن پر آپ فخر کر سکتے ہیں!

### 🚀 نمو کا مرحلہ: جدید تصورات اور عملی اطلاق (ماڈیولز 4-5)

بنیادی باتوں میں مہارت حاصل کرنے کے بعد، آپ تیار ہیں کہ MCP کی مزید پیچیدہ خصوصیات دریافت کریں۔ ہم عملی عمل درآمد کی حکمت عملی، بگ کو درست کرنے کی تکنیک، اور جدید موضوعات جیسے ملٹی موڈل AI انٹیگریشن پر بات کریں گے۔

آپ یہ بھی سیکھیں گے کہ اپنی MCP عمل درآمدات کو پیداوار کے لیے کس طرح اسکیل کریں اور Azure جیسے کلاؤڈ پلیٹ فارمز کے ساتھ انٹیگریٹ کریں۔ یہ ماڈیولز آپ کو ایسے MCP حل بنانے کے لیے تیار کرتے ہیں جو حقیقی دنیا کی ضروریات کو سنبھال سکیں۔

### 🌟 مہارت کا مرحلہ: کمیونٹی اور تخصص (ماڈیولز 6-11)

آخری مرحلہ MCP کمیونٹی میں شامل ہونے اور ان علاقوں میں تخصص پر توجہ دیتا ہے جو آپ کی سب سے زیادہ دلچسپی رکھتے ہیں۔ آپ سیکھیں گے کہ اوپن سورس MCP پروجیکٹس میں کیسے حصہ ڈالنا ہے، جدید توثیق کے پیٹرنز کو نافذ کرنا ہے، اور مکمل ڈیٹا بیس یکجا حل تیار کرنا ہے۔

ماڈیول 11 خاص ذکر کا مستحق ہے - یہ مکمل 13-لیب عملی سیکھنے کا راستہ ہے جو آپ کو PostgreSQL انٹیگریشن کے ساتھ پیداوار کے قابل MCP سرورز بنانے کی تعلیم دیتا ہے۔ یہ ایک کیپ اسٹون پروجیکٹ کی طرح ہے جو آپ نے جو کچھ سیکھا ہے اسے ایک ساتھ لاتا ہے!

### 💻 نمونہ کوڈ پروجیکٹس

MCP سیکھنے کے سب سے دلچسپ حصوں میں سے ایک آپ کی کوڈ ہنر کی تدریجی ترقی کو دیکھنا ہے۔ ہم نے اپنے کوڈ کی مثالوں کو اس طرح ڈیزائن کیا ہے کہ وہ آسان سے شروع ہوں اور جیسے جیسے آپ کی سمجھ بڑھتی جائے، زیادہ پیچیدہ ہوں۔ یہاں ہم تصورات کس طرح متعارف کراتے ہیں - ایسے کوڈ کے ساتھ جو سمجھنے میں آسان ہے لیکن اصلی MCP اصولوں کو ظاہر کرتا ہے، آپ نہ صرف سمجھیں گے کہ یہ کوڈ کیا کرتا ہے بلکہ کیوں یہ اس طرح مرتب کیا گیا ہے اور یہ بڑے MCP ایپلیکیشنز میں کیسے فٹ بیٹھتا ہے۔

## 🎓 آپ کا MCP سفر شروع ہوتا ہے

مبارک ہو! آپ نے ابھی ایک دلچسپ سفر کا پہلا قدم اٹھایا ہے جو آپ کی پروگرامنگ صلاحیتوں کو وسیع کرے گا اور آپ کو AI کی ترقی کی جدید سطح سے جوڑے گا۔

### آپ نے کیا حاصل کر لیا ہے

اس تعارف کو پڑھ کر، آپ نے پہلے ہی MCP علم کی بنیاد بنانا شروع کر دی ہے۔ آپ سمجھ چکے ہیں کہ MCP کیا ہے، کیوں اہم ہے، اور یہ نصاب آپ کے سیکھنے کے سفر کی حمایت کیسے کرے گا۔ یہ ایک اہم کامیابی ہے اور اس ٹیکنالوجی میں مہارت کی ابتدا ہے۔

### آگے کا سفر

جیسے جیسے آپ ماڈیولز میں ترقی کرتے جائیں گے، یاد رکھیں کہ ہر ماہر کبھی مبتدی تھا۔ وہ تصورات جو اب پیچیدہ لگتے ہیں، مشق اور اطلاق سے فطری بن جائیں گے۔ ہر چھوٹا قدم آپ کی ترقی کیریئر میں طاقتور صلاحیتوں کی طرف بڑھتا ہے۔

### آپ کا سپورٹ نیٹ ورک

آپ ایک ایسی کمیونٹی میں شامل ہو رہے ہیں جہاں سیکھنے والے اور ماہرین MCP کے بارے میں پرجوش ہیں اور دوسروں کی کامیابی کے لیے مددگار ہیں۔ چاہے آپ کسی کوڈنگ چیلنج میں پھنس جائیں یا کوئی نیا کامیابی شیئر کرنا چاہتے ہوں، کمیونٹی آپ کے سفر کی حمایت کے لیے یہاں ہے۔

اگر آپ پھنس جائیں یا AI ایپس بنانے کے حوالے سے کوئی سوال ہو، تو MCP کے بارے میں مباحثوں میں شامل ہوں۔ یہ ایک معاون کمیونٹی ہے جہاں سوالات خوش آمدید کہے جاتے ہیں اور علم بلا جھجک شیئر کیا جاتا ہے۔

[![Microsoft Foundry Discord](https://dcbadge.limes.pink/api/server/nTYy5BXMWG)](https://discord.gg/nTYy5BXMWG)

اگر آپ کے پاس پروڈکٹ کا فیڈ بیک یا تعمیر کے دوران غلطیاں ہوں تو یہاں دیکھیں:

[![Microsoft Foundry Developer Forum](https://img.shields.io/badge/GitHub-Microsoft_Foundry_Developer_Forum-blue?style=for-the-badge&logo=github&color=000000&logoColor=fff)](https://aka.ms/foundry/forum)

### شروع کرنے کے لیے تیار ہیں؟

آپ کا MCP کا سفر ابھی شروع ہوتا ہے! ماڈیول 0 سے شروع کریں تاکہ آپ اپنے پہلے عملی MCP تجربات میں داخل ہوں، یا نمونہ پراجیکٹس کو دیکھیں کہ آپ کیا بنانے جا رہے ہیں۔ یاد رکھیں - ہر ماہر نے بالکل آپ کی جگہ سے شروع کیا تھا، اور صبر و مشق کے ساتھ، آپ حیرت انگیز کامیابیاں حاصل کریں گے۔

ماڈل کانٹیکسٹ پروٹوکول کی ترقی کی دنیا میں خوش آمدید۔ آئیے مل کر کچھ شاندار بنائیں!
