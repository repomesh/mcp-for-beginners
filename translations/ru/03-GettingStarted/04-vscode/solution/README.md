# Запуск примера

Здесь предполагается, что у вас уже есть рабочий код сервера. Пожалуйста, найдите сервер из одной из предыдущих глав.

## Настройка mcp.json

Вот файл, который вы используете для справки, [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json).

Измените запись сервера при необходимости, указав абсолютный путь к вашему серверу, включая нужную полную команду для запуска.

В примере файла, упомянутом выше, запись сервера выглядит так:

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

Возможно, вам придется указать корневой каталог репозитория GitHub, который можно получить с помощью команды `git rev-parse --show-toplevel`.

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

Это соответствует выполнению команды следующего вида: `node build/index.js`.

- Измените эту запись сервера так, чтобы она соответствовала расположению вашего серверного файла или требованиям для запуска вашего сервера в зависимости от выбранного времени выполнения и местоположения сервера.

## Использование функций на сервере

- Нажмите на значок `play`, как только добавите *mcp.json* в папку *./vscode*,

    Обратите внимание, что значок инструментов изменится, увеличив количество доступных инструментов. Значок инструментов находится прямо над полем чата в GitHub Copilot.

## Запуск инструмента

- Введите запрос в окно чата, соответствующий описанию вашего инструмента. Например, чтобы вызвать инструмент `add`, введите что-то вроде "add 3 to 20".

    Вы увидите отображение инструмента над текстовым полем чата с предложением выбрать запуск инструмента, как показано на этой картинке:

    ![VS Code indicating it wanting to run a tool](../../../../../translated_images/ru/vscode-agent.d5a0e0b897331060.webp)

    При выборе инструмента вы должны получить числовой результат "23", если ваш запрос был такой, как указано выше.
