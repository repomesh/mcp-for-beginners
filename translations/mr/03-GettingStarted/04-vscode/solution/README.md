# नमुना चालवत आहे

येथे आपण असे गृहित धरतो की आपल्याकडे आधीच कार्यरत सर्व्हर कोड आहे. कृपया पूर्वीच्या अध्यायांपैकी एखाद्या सर्व्हरचा शोध घ्या.

## mcp.json सेट करा

हे संदर्भासाठी वापरायचे फायली आहे, [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json).

आवश्यकतेनुसार सर्व्हर नोंद बदला जेणेकरून आपल्या सर्व्हरचे पूर्ण मार्ग आणि चालविण्याचा पूर्ण आदेश दाखवेल.

वरील उदाहरण फाइलमध्ये सर्व्हर नोंद अशी दिसते:

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

आपण GitHub रिपॉजिटरी मूळ स्थान प्रविष्ट करावे लागेल, जे `git rev-parse --show-toplevel` या आदेशाने मिळवता येते.

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

हे अशी कमांड चालवण्यासारखे आहे: `node build/index.js`.

- आपला सर्व्हर फाइल कुठे आहे किंवा आपल्या निवडलेल्या रनटाइम आणि सर्व्हर स्थानानुसार सर्व्हर सुरू करण्यासाठी काय आवश्यक आहे, त्यानुसार ही सर्व्हर नोंद बदला.

## सर्व्हरमधील वैशिष्ट्ये वापरा

- एकदा आपण *mcp.json* *./vscode* फोल्डरमध्ये जोडल्यावर `play` आयकॉनवर क्लिक करा,

    प्राप्त उपकरणांचा नंबर वाढत असल्याचे टूलिंग आयकॉन बघा. टूलिंग आयकॉन GitHub Copilot मध्ये चॅट फील्डच्या वर आहे.

## एक साधन चालवा

- आपल्या चॅट विंडोमध्ये आपल्या उपकरणाचे वर्णन जुळणारा प्रॉम्प्ट टाइप करा. उदाहरणार्थ, टूल `add` चालवण्यासाठी काहीसे "add 3 to 20" लिहा.

    आपल्याला चॅट टेक्स्ट बॉक्सच्या वर एक साधन दिसेल ज्याला चालवण्यासाठी निवडू शकता, जसे खाली ऍनिमेशनमध्ये आहे:

    ![VS Code indicating it wanting to run a tool](../../../../../translated_images/mr/vscode-agent.d5a0e0b897331060.webp)

    जर आपणास रिप्रॉम्प्ट प्रमाणे टूल निवडले तर संदर्भ परिणाम "23" असा संख्यात्मक दिसेल.
