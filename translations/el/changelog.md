  - Οι Hosts ορίζονται πλέον σαφώς ως εφαρμογές AI που συντονίζουν πολλαπλές συνδέσεις πελατών MCP
  - Οι Clients περιγράφονται ως σύνδεσμοι πρωτοκόλλου που διατηρούν σχέσεις ένας-προς-έναν με τους servers
  - Οι Servers βελτιώνονται με σενάρια τοπικής έναντι απομακρυσμένης εγκατάστασης
- **Αναδιάρθρωση Αρχέγονων Στοιχείων**: Πλήρης αναθεώρηση των αρχέγονων στοιχείων server και client
  - Αρχέγονα Server: Πόροι (πηγές δεδομένων), Προτροπές (πρότυπα), Εργαλεία (εκτελέσιμες λειτουργίες) με λεπτομερείς εξηγήσεις και παραδείγματα
  - Αρχέγονα Client: Δειγματοληψία (LLM completions), Προτροπή (εισροή χρήστη), Καταγραφή (αποσφαλμάτωση/παρακολούθηση)
  - Ενημερωμένα με τρέχοντα πρότυπα μεθόδων ανακάλυψης (`*/list`), ανάκτησης (`*/get`), και εκτέλεσης (`*/call`)
- **Αρχιτεκτονική Πρωτοκόλλου**: Εισαγωγή μοντέλου αρχιτεκτονικής δύο επιπέδων
  - Επίπεδο Δεδομένων: Βάση JSON-RPC 2.0 με διαχείριση κύκλου ζωής και αρχέγονα στοιχεία
  - Επίπεδο Μεταφοράς: STDIO (τοπικό) και Streamable HTTP με SSE (απομακρυσμένο) μηχανισμοί μεταφοράς
- **Πλαίσιο Ασφαλείας**: Ολοκληρωμένες αρχές ασφαλείας που περιλαμβάνουν ρητή συγκατάθεση χρήστη, προστασία απορρήτου δεδομένων, ασφάλεια εκτέλεσης εργαλείων, και ασφάλεια στρώματος μεταφοράς
- **Πρότυπα Επικοινωνίας**: Ενημέρωση μηνυμάτων πρωτοκόλλου για εμφάνιση ροών αρχικοποίησης, ανακάλυψης, εκτέλεσης, και ειδοποίησης
- **Παραδείγματα Κώδικα**: Ανανέωση παραδειγμάτων σε πολλές γλώσσες (.NET, Java, Python, JavaScript) ώστε να αντανακλούν τα τρέχοντα πρότυπα MCP SDK
