# Εκτέλεση του παραδείγματος

Εδώ υποθέτουμε ότι ήδη έχετε έναν λειτουργικό κώδικα διακομιστή. Παρακαλούμε εντοπίστε έναν διακομιστή από ένα από τα προηγούμενα κεφάλαια.

## Ρύθμιση του mcp.json

Εδώ είναι ένα αρχείο που χρησιμοποιείτε για αναφορά, [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json).

Αλλάξτε την εγγραφή διακομιστή όπως χρειάζεται για να δείξετε το απόλυτο μονοπάτι προς τον διακομιστή σας, συμπεριλαμβανομένης της πλήρους εντολής που απαιτείται για την εκτέλεση.

Στο αρχείο παραδείγματος που αναφέρθηκε παραπάνω, η εγγραφή διακομιστή φαίνεται ως εξής:

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

Ίσως χρειαστεί να εισάγετε τη ρίζα του αποθετηρίου GitHub, η οποία μπορεί να ληφθεί από την εντολή, `git rev-parse --show-toplevel`.

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

Αυτό αντιστοιχεί στην εκτέλεση μιας εντολής όπως: `node build/index.js`.

- Αλλάξτε αυτή την εγγραφή διακομιστή ώστε να ταιριάζει με το που βρίσκεται το αρχείο του διακομιστή σας ή με ό,τι χρειάζεται για να ξεκινήσετε το διακομιστή σας ανάλογα με το επιλεγμένο runtime και την τοποθεσία του διακομιστή.

## Χρήση των λειτουργιών στον διακομιστή

- Κάντε κλικ στο εικονίδιο `play`, αφού έχετε προσθέσει το *mcp.json* στο φάκελο *./vscode*,

    Παρατηρήστε την αλλαγή του εικονιδίου εργαλείων για να αυξηθεί ο αριθμός των διαθέσιμων εργαλείων. Το εικονίδιο εργαλείων βρίσκεται ακριβώς πάνω από το πεδίο συνομιλίας στο GitHub Copilot.

## Εκτέλεση ενός εργαλείου

- Πληκτρολογήστε μια ερώτηση στο παράθυρο συνομιλίας σας που ταιριάζει με την περιγραφή του εργαλείου σας. Για παράδειγμα, για να ενεργοποιήσετε το εργαλείο `add` πληκτρολογήστε κάτι σαν "add 3 to 20".

    Θα πρέπει να δείτε να παρουσιάζεται ένα εργαλείο πάνω από το πλαίσιο κειμένου συνομιλίας που σας ζητά να επιλέξετε να εκτελέσετε το εργαλείο όπως σε αυτή την εικόνα:

    ![VS Code indicating it wanting to run a tool](../../../../../translated_images/el/vscode-agent.d5a0e0b897331060.webp)

    Η επιλογή του εργαλείου θα πρέπει να παράγει ένα αριθμητικό αποτέλεσμα που λέει "23" αν η ερώτησή σας ήταν όπως αναφέραμε προηγουμένως.
