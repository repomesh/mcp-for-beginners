# การรันตัวอย่าง

ที่นี่เราสมมติว่าคุณมีโค้ดเซิร์ฟเวอร์ที่ใช้งานได้แล้ว โปรดค้นหาเซิร์ฟเวอร์จากบทตอนก่อนหน้านี้

## ตั้งค่า mcp.json

นี่คือไฟล์ที่คุณใช้เป็นข้อมูลอ้างอิง, [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json)

เปลี่ยนรายการเซิร์ฟเวอร์ตามที่จำเป็นเพื่อชี้ไปยังเส้นทางสัมบูรณ์ไปยังเซิร์ฟเวอร์ของคุณรวมถึงคำสั่งเต็มที่จำเป็นสำหรับการรัน

ในไฟล์ตัวอย่างที่อ้างอิงข้างต้น รายการเซิร์ฟเวอร์จะมีลักษณะดังนี้:

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

คุณอาจต้องป้อนรากของที่เก็บ GitHub ซึ่งสามารถดึงมาจากคำสั่ง, `git rev-parse --show-toplevel`

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

นี่สอดคล้องกับการรันคำสั่งประมาณนี้: `node build/index.js`

- เปลี่ยนรายการเซิร์ฟเวอร์นี้ให้เหมาะสมกับที่ตั้งไฟล์เซิร์ฟเวอร์ของคุณหรือสิ่งที่จำเป็นสำหรับการเริ่มต้นเซิร์ฟเวอร์ของคุณขึ้นอยู่กับ runtime และที่ตั้งเซิร์ฟเวอร์ที่คุณเลือก

## ใช้งานฟีเจอร์ในเซิร์ฟเวอร์

- คลิกไอคอน `play` เมื่อคุณเพิ่ม *mcp.json* ลงในโฟลเดอร์ *./vscode* แล้ว

    สังเกตว่าไอคอนเครื่องมือจะเปลี่ยนเพื่อเพิ่มจำนวนเครื่องมือที่สามารถใช้ได้ ไอคอนเครื่องมือนั้นจะอยู่เหนือช่องแชทใน GitHub Copilot

## รันเครื่องมือ

- พิมพ์คำสั่งในหน้าต่างแชทของคุณที่ตรงกับคำอธิบายของเครื่องมือของคุณ เช่น เพื่อเรียกใช้เครื่องมือ `add` ให้พิมพ์ว่า "add 3 to 20"

    คุณควรจะเห็นเครื่องมือแสดงอยู่เหนือกล่องข้อความแชทเพื่อให้คุณเลือกใช้งานเครื่องมือนั้น เช่นในภาพนี้:

    ![VS Code indicating it wanting to run a tool](../../../../../translated_images/th/vscode-agent.d5a0e0b897331060.webp)

    การเลือกเครื่องมือควรจะได้ผลลัพธ์เป็นตัวเลข "23" หากคำสั่งของคุณเหมือนที่เราแนะนำไปก่อนหน้านี้
