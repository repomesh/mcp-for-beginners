# മാറ്റങ്ങളുടെ രേഖ: MCP ബിഗ്‌ളിനേർസ് കരിക്കുലം

മോഡൽ കോൺടെക്‌സ് പ്രോട്ടോക്കോൾ (MCP) ബിഗ്‌ളിനേർസ് കരിക്കുലത്തിൽ സംഭവിച്ച എല്ലാ പ്രധാന മാറ്റങ്ങളുടെയും രേഖ ഇതാണ്. മാറ്റങ്ങൾ തിരുമാനാനുസരണിബന്ധനയിൽ രേഖപ്പെടുത്തിയിരിക്കുന്നു (പുതിയത് ആദ്യം).

- **പ്രിമിറ്റീവുകളുടെ ഡോക്യുമെന്റേഷൻ**: സെർവർ പ്രിമിറ്റീവുകളും (റിസോഴ്‌സുകൾ, പ്രൊംപ്റ്റുകൾ, ടൂളുകൾ), ക്ലയന്റ് പ്രിമിറ്റീവുകൾ (സാമ്പ്ലിംഗ്, elicitation, ലോഗിംഗ്, റൂട്ട്സ്) ശരിയായി പരിശോധിച്ചു
- **ട്രാൻസ്പോർട്ട് മെക്കാനിസങ്ങൾ**: STDIO, Streamable HTTP ട്രാൻസ്പോർട്ട് ഡോക്യുമെന്റേഷൻ ശരിയാണ് എന്ന് സ്ഥിരീകരിച്ചു
- **സുരക്ഷാമാർഗ്ഗ നിർദേശങ്ങൾ**: നിലവിലെ MCP സുരക്ഷാ മികച്ച പ്രവർത്തനങ്ങൾ എന്നിവയുമായി പൊരുത്തപ്പെട്ടു

#### ലളിതമായ ഓത്തന്റിക്കേഷൻ (03-GettingStarted/11-simple-auth)
- **പുതിയത് ചേർത്തു**: MCP സെർവറുകളിൽ ലളിതമായ ഓത്തന്റിക്കേഷൻ നടപ്പാക്കുന്നതിന് പടിപ്പടിയായ ഗൈഡ്

  - **ബേസിക് ഓത്ത് നടപ്പാക്കൽ**: Python (Starlette), TypeScript (Express) എന്നിവയിൽ മിഡിൽവെയർ അടിസ്ഥാന ഓത്തന്റിക്കേഷൻ മാതൃകകൾ കോഡ് സാമ്പിളുകളോടെ

    - PKCE ഉപയോഗിച്ചുള്ള Confused Deputy ആക്രമണ നിരോധനവും ഉപയോക്തൃ സമ്മത വാലിഡേഷനും
    - ഓഡിയൻസിന്റെ വാലിഡേഷനോടും സുരക്ഷിത ടോക്കൺ മാനേജ്മെന്റോടും കൂടിയ ടോക്കൺ പാസ്സ്‌ത്രൂ തടയൽ
    - ക്രിപ്‌റ്റോഗ്രാഫിക് ബൈൻഡിങ്ങും ബീഹേവിയറൽ അനലിസിസും ഉൾപ്പെടെ സെഷൻ ഹൈജാക്കിംഗ് തടയൽ
  - **എന്റർപ്രൈസ് സെക്യൂരിറ്റി സംയോജനം**: ആസ്യൂർ അപ്ലിക്കേഷൻ ഇൻസൈറ്റ്സ് നിരീക്ഷണം, ത്രേറ്റ് കണ്ടെത്തൽ പൈപ്പ്‌ലൈനുകൾ, സപ്ലൈ ചെയിൻ സെക്യൂരിറ്റി
  - **നടപ്പിലാക്കൽ ചെക്ക്ലിസ്‌റ്റ്**: നിർബന്ധമായും ശിപാർശചെയ്യപ്പെടുന്ന സുരക്ഷാ കൺട്രോൾസുകളുടെ വ്യക്തമായ തിരിച്ചറിയലും മൈക്രോസോഫ്റ്റ് സെക്യൂരിറ്റി ഇക്കോസിസ്റ്റത്തിന്റെ ഗുണങ്ങളും

### ഡോക്യുമെന്റേഷൻ ഗുണനിലവാരവും സ്റ്റാൻഡേഡ്സ് പൊരുത്തത്തോടെയും
- **സ്പെസിഫിക്കേഷൻ റഫറൻസുകൾ**: എല്ലാ റഫറൻസുകളും MCP സ്‌പെസിഫിക്കേഷൻ 2025-06-18 ആക്കി പുതുക്കിയതായി
- **Microsoft Security Ecosystem**: എല്ലാ സുരക്ഷാ ഡോക്യുമെന്റേഷനിലും വർദ്ധിച്ച സംയോജന മാർഗ്ഗനിർദേശം
- **പ്രായോഗിക നടപ്പിലാക്കൽ**: .NET, Java, Python ഭാഷകളിൽ ഡീറ്റെയിൽഡ് കോഡ് ഉദാഹരണങ്ങൾ ചേർത്തു, എന്റർപ്രൈസ് മാതൃകകളോടെ
- **സ്രോതസ്സ് സംഘടിപ്പിക്കൽ**: ഔദ്യോഗിക ഡോക്യുമെന്റേഷൻ, സുരക്ഷാ സ്റ്റാൻഡേർഡുകൾ, നടപ്പിലാക്കൽ മാർഗ്ഗനിർദേശങ്ങളായി സമഗ്ര വർഗ്ഗീകരണം
- **വിഷ്വൽ ഇതരങ്ങൾ**: നിർബന്ധമായ ആവശ്യങ്ങളും ശിപാർശചെയ്യുന്ന പ്രാക്ടീസുകളും വ്യക്തമായി അടയാളപ്പെടുത്തിയത്


#### കോർ കോൺസെപ്റ്റുകൾ (01-CoreConcepts/) - പൂർണ്ണ നവീകരണം
- **പ്രോട്ടോക്കോൾ പതിപ്പ് അപ്‌ഡേറ്റ്**: MCP സ്‌പെസിഫിക്കേഷൻ 2025-06-18 നിലവിലെത്തി, തീയതി അടിസ്ഥാനത്തിലുള്ള പതിപ്പൊദ്യോഗികമാക്കി (YYYY-MM-DD ഫോർമാറ്റ്)
- **ആർക്കിടെക്ചർ മെച്ചപ്പെടുത്തൽ**: ഹോസ്റ്റുകൾ, ക്ലയന്റുകൾ, സർവറുകൾ MCP ആർക്കിടെക്ചർ മാതൃകകൾ പ്രകാരം മെച്ചപ്പെടുത്തിയ വിവരണങ്ങൾ
  - ഹോസ്റ്റുകൾ ഇപ്പോൾ MCP ക്ലയന്റ് കണക്ഷനുകൾ കോർഡിനേറ്റ് ചെയ്യുന്ന AI ഐപ്ലിക്കേഷനായി വ്യക്തമാക്കി
  - ക്ലയന്റുകൾ ഒന്നിനൊന്ന് സർവറുമായി ബന്ധിപ്പിക്കുന്ന പ്രോട്ടോക്കോൾ കണക്റ്ററുകളായി വിവരണം
  - സർവറുകൾ ലോക്കൽ, റിമോട്ട് ഡിപ്ലോയ്‌മെന്റ് സാഹചര്യങ്ങളോടുകൂടി മെച്ചപ്പെടുത്തി
- **പ്രിമിറ്റീവ് പുനഃസംഘടന**: സർവർ, ക്ലയന്റ് പ്രിമിറ്റീവ് പൂർണമായി പുനഃസംഘടിപ്പിച്ചു
  - സർവർ പ്രിമിറ്റീവ്: റിസോഴ്‌സുകൾ (ഡേറ്റാ സോഴ്സുകൾ), പ്രോപ്റ്റുകൾ (ടെംപ്ലേറ്റുകൾ), ടൂളുകൾ (പ്രവർത്തന ഫունക്ഷനുകൾ) വിശദമായ വിശദീകരണവും ഉദാഹരണങ്ങളുമായി
  - ക്ലയന്റ് പ്രിമിറ്റീവ്: സാമ്പ്ലിംഗ് (LLM പൂർത്തീകരണങ്ങൾ), elicitation (ഉപയോക്തൃ ഇൻ‌പുട്ട്), ലോഗിംഗ് (ഡീബഗ്ഗിംഗ്/നിരീക്ഷണം)
  - നിലവിലെ disco൪വറി (`*/list`), റിട്ട്രീവൽ (`*/get`), എക്‌സിക്യൂഷൻ (`*/call`) രീതിപ്രകാരം അപ്‌ഡേറ്റ് ചെയ്തിട്ടുണ്ട്
- **പ്രോട്ടോക്കോൾ ആർക്കിടെക്ചർ**: രണ്ട് തരം ആർക്കിടെക്ചർ മോഡൽ അവതരിപ്പിച്ചു
  - ഡാറ്റ ലെയർ: JSON-RPC 2.0 അടിസ്ഥാനവും ലൈഫ്‌സൈക്കിൾ മാനേജ്മെന്റും പ്രിമിറ്റീവുകളും
  - ട്രാൻസ്പോർട്ട് ലെയർ: STDIO (ലോകൽ) & സ്ട്രീമബിൾ HTTP SSE (റിമോട്ട്) ട്രാൻസ്പോർട്ട് മേധോദുകൾ
- **സുരക്ഷാ ഫ്രെയിംവർക്ക്**: വ്യക്തമായ ഉപയോക്തൃ സമ്മതി, ഡാറ്റ പ്രൈവസിയുടെ സംരക്ഷണം, ടൂൾ എക്സിക്യൂഷൻ സുരക്ഷ, ട്രാൻസ്പോർട്ട് ലെയർ സെക്യൂരിറ്റി തുടങ്ങിയ സമഗ്ര സുരക്ഷാ വിഷയങ്ങൾ
- **കമ്മ്യൂണിക്കേഷൻ മാതൃകകൾ**: ഇൻഷിയലൈസേഷൻ, ഡിസ്കവറി, എക്സിക്യൂഷൻ, നോട്ടിഫിക്കേഷൻ ഫ്ലോകൾ മոզയിലൂടെ പ്രോട്ടോക്കോൾ സന്ദേശങ്ങൾ അപ്‌ഡേറ്റ് ചെയ്തിട്ടുണ്ട്
- **കോഡ് ഉദാഹരണങ്ങൾ**: .NET, Java, Python, JavaScript തുടങ്ങിയ ഭാഷകളിലെ MCP SDK നിലവിലെ മാതൃകകൾ പ്രതിഫലിക്കുന്ന വിധം പുതുക്കി

#### സെക്യൂരിറ്റി (02-Security/) - സമഗ്ര സുരക്ഷാ പുനഃസംഘടനം  
- **സ്റ്റാൻഡേർഡ് പൊരുത്തം**: MCP സ്‌പെസിഫിക്കേഷൻ 2025-06-18 സുരക്ഷാ ആവശ്യകതകളുമായി പൂർണ്ണ പൊരുത്തം
- **ഔത്തന്റിക്കേഷൻ പരിണാമം**: കസ്റ്റം OAuth സെർവറുകളിൽ നിന്ന് പുറം തിരിച്ചറിയൽ ദാതാവ് ഡെലിഗേഷനിലേക്ക് (Microsoft Entra ID) പരിണമനം രേഖപ്പെടുത്തിയിരിക്കുന്നു
- **AI-സ്പെസിഫിക് ത്രേറ്റ് അനലിസിസ്**: ആധുനിക AI ആക്രമണവകുപ്പ് വ്യാപൃതമായ വിവരങ്ങൾ ഉൾപ്പെടുത്തി
  - റിയൽ വേൾഡ് ഉദാഹരണങ്ങളോടെ പ്രോപ്റ്റ് ഇൻജക്ഷൻ ആക്രമണ പരിസർഗ്ഗങ്ങൾ
  - ടൂൾ പവൈസണിംഗ് മെക്കാനിസങ്ങൾ, "റഗ് 풀" ആക്രമണ മാതൃകകൾ
  - കോൺടെക്‌സ്‌റ്റ് വിൻഡോ പവൈസണിംഗ്, മോഡൽ കൺഫ്യൂഷൻ ആക്രമണങ്ങൾ
- **Microsoft AI Security Solutions**: മൈക്രോസോഫ്റ്റ് സെക്യൂരിറ്റി ഇക്കോസിസ്റ്റം സമഗ്ര പരിധി
  - AI Prompt Shields ഉയർന്ന തിരിച്ചറിയൽ, സ്പോട്ട്‌ലൈറ്റിങ്, ഡെലിമിറ്റർ സാങ്കേതികവിദ്യകൾ
  - ആസ്യൂർ കോൺടെന്റ് സേഫ്റ്റി സംയോജന മാതൃകകൾ
  - GitHub ആഡ്വാൻസ്ഡ് സെക്യൂരിറ്റി സപ്ലൈ ചെയിൻ സംരക്ഷണത്തിന്
- **ഉന്നത ത്രേറ്റ് നിവാരണ**: സൂക്ഷ്മ സുരക്ഷാ നിയന്ത്രണങ്ങൾ
  - MCP-നിർദ്ദಿಷ್ಟ ആക്രമണ രംഗങ്ങളിൽ സെഷൻ ഹൈജാക്കിംഗ്, ക്രിപ്‌റ്റോഗ്രാഫിക് സെഷൻ ഐഡി ആവശ്യകതകൾ
  - MCP പ്രോക്സി സാഹചര്യങ്ങളിൽ Confused Deputy പ്രശ്നങ്ങൾ വ്യക്തമായ സമ്മതി ആവശ്യകതകളോടെ
  - നിർബന്ധമായ വാലിഡേഷൻ കൺട്രോൾസ് ഉൾപ്പെടുന്ന ടോക്കൺ പാസ്സ്‌ത്രൂ ത്വരിതം
- **സപ്ലൈ ചെയിൻ സെക്യൂരിറ്റി**: ഫൗണ്ടേഷൻ മോഡലുകൾ, എംബെഡിംഗ്സ് സർവീസുകൾ, കോൺടെക്‌സ്‌റ്റ് പ്രൊവൈഡേഴ്‌സ്, തൃतीयപക്ഷ API-കൾ ഉൾപ്പെടെയുള്ള AI സപ്ലൈ ചെയിൻ വ്യാപ്തി വിപുലപ്പെടുത്തി
- **ഫൗണ്ടേഷൻ സെക്യൂരിറ്റി**: സീറോ ട്രസ്റ്റ് ആർക്കിടെക്ചർ, മൈക്രോസോഫ്റ്റ് സെക്യൂരിറ്റി ഇക്കോസിസ്റ്റം ഉൾപ്പെടെയുള്ള എൻ‌റ്റർപ്രൈസ് സുരക്ഷാ മാതൃകകളുമായി മെച്ചപ്പെട്ട സംയോജനം

- README.md ൽ പഠനപരിക्रम നാവിഗേഷൻ പൂർണ്ണമായും പുനർരൂപപ്പെടുത്തി

- എല്ലാ പുതിയ ഫയലുകളും ഉൾപ്പെടുന്ന റീപോസിറ്ററി ഘടന വിഭാഗം അപ്‌ഡേറ്റ് ചെയ്തു
- "How to Use This Curriculum" വിഭാഗം വ്യക്തമായ ശിപാർശകൾക്കൊപ്പം മെച്ചപ്പെടുത്തി
- MCP സ്‌പെസിഫിക്കേഷൻ ലിങ്കുകൾ ശരിയായ URLs ലേക്ക് പുതുക്കി
- പഠനപരിക്രമ ഘടനയിൽ Context Engineering വിഭാഗം (5.14) ചേർത്തു

### സ്റ്റഡി ഗൈഡ് അപ്‌ഡേറ്റുകൾ
- നിലവിലുള്ള റീപോസിറ്ററി ഘടനയ്ക്ക് അനുയോജിച്ചു പഠനഗൈഡ് പൂർണ്ണമായും പുനഃസംഘടിപ്പിച്ചു
- MCP ക്ലയന്റുകളും ടൂളുകളും, ജനപ്രിയ MCP സർവറുകളും ഉൾപ്പെടുന്ന പുതിയ വിഭാഗങ്ങൾ ചേർത്തു
- വിഷ്വൽ പഠനമാപ്പ് എല്ലാ വിഷയങ്ങളും കൃത്യമായി പ്രതിഫലിപ്പിക്കുന്ന വിധം പുതുക്കി
- എല്ലാ പ്രത്യേക മേഖലകളും ഉൾക്കൊള്ളുന്ന അഡ്വാൻസ്ഡ് ടോപിക്സിന്റെ വിവരണങ്ങൾ മെച്ചപ്പെടുത്തി
- യഥാർത്ഥ ഉദാഹരണങ്ങൾ ഉൾപ്പെടുത്തിയ കേസ്സ്റ്റഡീസ് സെക്ഷൻ അപ്‌ഡേറ്റ് ചെയ്തു
- ഈ സമഗ്ര മാറ്റലോഗ് ചേർത്തു

### കമ്മ്യൂണിറ്റി സംഭാവനകൾ (06-CommunityContributions/)

- MCP ക്ലയന്റ് വിഭാഗം എല്ലാ ജനപ്രിയ ക്ലയന്റ് ഓപ്ഷനുകളും ഉൾപ്പടെ അപ്‌ഡേറ്റ് ചെയ്തു
