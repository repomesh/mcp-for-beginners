# ಚೇಂಜ್‌ಲಾಗ್: MCP ಶುಭಾರಂಭಿಕರಿಗೆ ಪಾಠ್ಯಕ್ರಮ

ಈ ದಸ್ತಾವೇಜು ಮಾದರಿ ಕಾಂಟೆಕ್ಸ್ಟ್ ಪ್ರೋಟೋಕಾಲ್ (MCP) ಶುಭಾರಂಭಿಕರಿಗೆ ಪಾಠ್ಯಕ್ರಮಕ್ಕೆ ಸಂಬಂಧಿಸಿದ ಎಲ್ಲಾ ಪ್ರಮುಖ ಬದಲಾವಣೆಗಳನ್ನು ದಾಖಲಿಸುವುದಾಗಿ ಸೇವೆ ನೀಡುತ್ತದೆ. ಬದಲಾವಣೆಗಳನ್ನು ವಿರುದ್ಧ ಕಾಲಕ್ರಮದಲ್ಲಿ ದಾಖಲಾಗಿದೆ (ಹೊಸತಾದ ಬದಲಾವಣೆಗಳು ಮೊದಲು ಇರುತ್ತವೆ).

## 2ನೇ ಜುಲೈ, 2026

### ಹೊಸ ಪಾಠ: 2026-07-28 MCP ರೂಪುರೇಷೆ ಬಿಡುಗಡೆಯ ಅಭ್ಯರ್ಥಿ

ಬರುವ `2026-07-28` MCP ರೂಪುರೇಷೆ ಬಿಡುಗಡೆಯ ಅಭ್ಯರ್ಥಿಯ (ಮೇ 21, 2026 ರಂದು ಘೋಷிக்கப்பட்டಿತ್ತು; ಅಂತಿಮ ಬಿಡುಗಡೆ ಜುಲೈ 28, 2026 ರಿಗೆ ನಿಯೋಜಿಸಲಾಗಿದೆ) ಕುರಿತ ಸಂಕಲನವನ್ನು ಸೇರಿಸಲಾಗಿದೆ, ಇದು [ಅಧಿಕೃತ ಘೋಷಣೆ ಬ್ಲಾಗ್ ಪೋಸ್ಟ್](https://blog.modelcontextprotocol.io/posts/2026-07-28-release-candidate/) ನಿಂದ ಸಂಕ್ಷಿಪ್ತವಾಗಿದೆ. ಪಾಠ್ಯಕ್ರಮದ ಮೂಲಭೂತ ಪ್ಯಾಥರ್ ಮುಂದುವರೆದಂತೆ **MCP ರೂಪುರೇಷೆ 2025-11-25** ಆಗಿದೆ ಹೊಸ ಆವೃತ್ತಿ ಹಡಗಿನವರೆಗೂ, ಆದಕಾರಣ ಇದನ್ನು ಈಗಿನ ಪಾಠ್ಯಗಳನ್ನು ಮರುಲೇಖನ ಮಾಡದೇ ಮುಂದಿನ ಮಾರ್ಗದರ್ಶನವಾಗಿ ಪ್ರಸ್ತುತಪಡಿಸಲಾಗಿದೆ.

- **ಹೊಸದು**: [01-CoreConcepts/mcp-2026-07-28-release-candidate.md](./01-CoreConcepts/mcp-2026-07-28-release-candidate.md) — ಸ್ಥಿತರಹಿತ ಪ್ರೋಟೋಕಾಲ್ ಕೋರ್ ಅನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಆವರಿಸುವ ಪಾಠ ( `initialize` ಹ್ಯಾಂಡ್‌ಶೇಕ್ ಮತ್ತು `Mcp-Session-Id` ನ್ನು ತೆಗೆದುಹಾಕುವುದು), ಹೊಸ `Mcp-Method`/`Mcp-Name` ರೌಟಿಂಗ್ ಹೆಡರ್ಸ್, `ttlMs`/`cacheScope` ಕ್ಯಾಶಿಂಗ್ ಮೆಟಾಡೇಟಾ, `_meta` ನಲ್ಲಿ W3C ಟ್ರೇಸ್ ಕಾಂಟೆಕ್ಸ್ಟ್, ಅಧಿಕೃತ ವಿಸ್ತರಣೆಗಳ ಚಟುವಟಿಕೆ (MCP ಅಪ್ಲಿಕೇಶನ್ಗಳು ಮತ್ತು ಹೊಸ ಟಾಸ್ಕ್ಸ್ ವಿಸ್ತರಣೆ), ಆರು ಸುತ್ತು ಸಂರಕ್ಷಣೆಯ SEPs, ರೂಟ್/ಸ್ಯಾಂಪ್ಲಿಂಗ್/ಲಾಗಿಂಗ್ ನ ನಿಯೋಜನೆ ಕಡಿತ, ಮತ್ತು ಸಂಪೂರ್ಣ JSON Schema 2020-12 ಗಾಗಿ ಉಪಕರಣ.schemas ಗೆ ಸ್ಥಳಾಂತರ.

## 16 ಜೂನ್, 2026

### MCP ರೂಪುರೇಷೆ ಹೊಂದಾಣಿಕೆ ಮತ್ತು ಮಾದರಿ ಪರಿಶೀಲನೆ

ಪಠ್ಯಕ್ರಮವನ್ನು ಪ್ರಸ್ತುತ **MCP ರೂಪುರೇಷೆ 2025-11-25** ಮತ್ತು ಇತ್ತೀಚಿನ ಅಧಿಕೃತ SDK ಗಳೊಂದಿಗೆ ಪರಿಶೀಲಿಸಿಕೊಂಡು, ಹಳೆಯ ರೂಪುರೇಷೆ ಉಲ್ಲೇಖಗಳನ್ನು ಸರಿಪಡಿಸಿ ಮತ್ತು ಕೋರ್ ಮಾದರಿಗಳು ಇನ್ನೂ ನಿರ್ಮಾಣ ಮತ್ತು ಚಾಲನೆ ಮಾಡುತ್ತಿರುವುದನ್ನು ದೃಢೀಕರಿಸಲಾಯಿತು.

#### ರೂಪುರೇಷೆ ಆವೃತ್ತಿ ತಿದ್ದುಪಡಿ (2025-06-18 / 2025-03-26 → 2025-11-25)

ಹಳೆಯ ರೂಪುರೇಷೆಯು *ಪ್ರಸ್ತುತ/ಇತ್ತೀಚಿನ* ಮಾನದಂಡ ಎಂದು ತಿಳಿಸಿದ್ದೆಲ್ಲಾ ಇಂಗ್ಲಿಷ್ ವಿಷಯಗಳನ್ನು ನವೀಕರಿಸಿ, ಲಿಂಕ್‌ಗಳನ್ನು `modelcontextprotocol.io` ರೂಪುರೇಷೆ canonical ಮಾರ್ಗಗಳಿಗೆ ಮರುಲಕ್ಷಣಗೊಳಿಸಲಾಗಿದೆ:
- **05-AdvancedTopics/mcp-security/README.md**: "ಪ್ರಸ್ತುತ ಮಾನದಂಡ" ಬ್ಯಾನರ್, ಪರಿಚಯ, ಕೋರ್ ಸುರಕ್ಷತಾ ತತ್ವಗಳು, ಅಗತ್ಯವಿರುವ ಅಗತ್ಯಗಳು, Microsoft Entra ID ವಿಭಾಗ, ಉಲ್ಲೇಖಗಳು ಮತ್ತು ಸಂಪನ್ಮೂಲಗಳ ಲಿಂಕ್‌ಗಳು, ಮತ್ತು ಮುಕ್ತಾಯ ಸುರಕ್ಷತಾ ಸೂಚನೆ (8 ಉಲ್ಲೇಖಗಳು) ನವೀಕರಿಸಲಾಯಿತು 2025-11-25 ಗೆ
- **05-AdvancedTopics/mcp-transport/README.md**: ಹೆಚ್ಚುವರಿ ಸಂಪನ್ಮೂಲಗಳ ರೂಪುರೇಷೆ ಲಿಂಕ್ ಮತ್ತು "ಪ್ರಸ್ತುತ ಮಾನದಂಡ" ಬ್ಯಾನರ್ ನವೀಕರಿಸಲಾಯಿತು 2025-11-25 ಗೆ
- **05-AdvancedTopics/mcp-realtimesearch/README.md**: ಹಳೇ `2025-03-26` ಸುರಕ್ಷತಾ-ಮತ್ತು-ವಿಶ್ವಾಸ ಲಿಂಕ್ ನ 2025-11-25 ಸುರಕ್ಷತಾ ಉತ್ತಮ ಅಭ್ಯಾಸ ಪುಟದಿಂದ ಬದಲಿಸಲಾಗಿದೆ
- **03-GettingStarted/14-sampling/README.md**: ಅಧಿಕೃತ ಸ್ಯಾಂಪ್ಲಿಂಗ್ ಡಾಕ್ಸ್ ಲಿಂಕ್ ನವೀಕರಿಸಲಾಯಿತು 2025-11-25 ಗೆ
- **03-GettingStarted/05-stdio-server/README.md**: ಪ್ರಸ್ತುತಕಾಲीन "ಪ್ರಸ್ತುತ MCP ರೂಪುರೇಷೆ" ಉಲ್ಲೇಖ ಮತ್ತು ಹೆಚ್ಚುವರಿ ಸಂಪನ್ಮೂಲಗಳ ರೂಪುರೇಷೆ ಲಿಂಕ್ ನವೀಕರಿಸಲಾಯಿತು 2025-11-25 ಗೆ (ಇತಿಹಾಸಾತ್ಮಕ SSE-ನಿಯೋಜನೆ ಕಡಿತ ಟಿಪ್ಪಣಿಗಳು ಸತ್ಯತೆಯಿಗಾಗಿ ಉಳಿಸಲಾಗಿದೆ)

#### ಪ್ರಸ್ತುತ SDK ಗಳ ವಿರುದ್ಧ ಮಾದರಿ ಪರಿಶೀಲನೆ

- **TypeScript (03-GettingStarted/01-first-server/solution/typescript)**: `npm install` ನೋಡಿದಂತೆ `@modelcontextprotocol/sdk@1.29.0` ಪರಿಹಾರವಾಯಿತು; `tsc --noEmit` ಯಾವುದೇ ಕೋಡ್ ದೋಷವಿಲ್ಲದೆ ಪಾಸಾಯಿತು — ಇತ್ತೀಚಿನ `McpServer`/`StdioServerTransport` APIಗಳು ಪ್ರಾಮಾಣಿಕವಾಗಿವೆ
- **Python (03-GettingStarted/01-first-server/solution/python)**: ಒಂದು ತಾನಾ `.venv` ನಲ್ಲಿ `mcp[cli]` (1.27.2) ಉಪಯೋಗಿಸಿದ ಪರಿಶೀಲನೆ; `py_compile` ಪಾಸಾದುದು ಮತ್ತು `FastMCP.list_tools()` ಸರಿಯಾಗಿ `add` ಮತ್ತು `subtract` ಉಪಕರಣಗಳನ್ನು ಹಿಂತಿರುಗಿಸಿದೆ
- ಎಲ್ಲಾ ಮಾದರಿ `@modelcontextprotocol/sdk` ಆವೃತ್ತಿ ಶ್ರೇಣಿಗಳು (`>=1.26.0` / `^1.26.0` / `^1.27.0`) ಪ್ರಸ್ತುತ `1.29.0` ಗೆ ಸೌಮ್ಯವಾಗಿಯೇ ಪರಿಹಾರಗೊಂಡಿವೆ ಯಾವುದೇ ಬದಲಾಗುವ API ಬದಲಾವಣೆಗಳಿಂದಲೂ ಮುಕ್ತವಾಗಿವೆ ಎಂದು ದೃಢೀಕರಿಸಲಾಯಿತು

- **ಟೂಲ್ ಅನೋಟೇಶನ್‌ಗಳು**: ಉತ್ತಮ ಉಪಕರಣ ಕಾರ್ಯಾಚರಣೆ ನಿರ್ಣಯಗಳಿಗಾಗಿ `readOnlyHint`, `destructiveHint` ಟೂಲ್ಗಳ ವರ್ತನೆ ಕುರಿತು ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಸೇರಿಸಲಾಗಿದೆ  
- **ಸ್ಯಾಂಪ್ಲಿಂಗ್‌ನಲ್ಲಿ ಟೂಲ್ ಕಾಲಿಂಗ್**: ಮಾದರಿ ಚಾಲಿತ ಟೂಲ್ ಕರೆದಸ್ತುಗಳನ್ನು ಒಳಗೊಂಡಂತೆ ಸಂಪನ್ಮೂಲಗಳ `tools` ಮತ್ತು `toolChoice` ಪರಿಮಾಣಗಳು ಸೇರಿಸಿ ಸಂಗ್ರಹವಿವರಣೆಯ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ жаңಗೊಳಿಸಲಾಗಿದೆ  
- **URL ಮೋಡ್ ಪೋಷಣೆ**: ಸರ್ವರ್ ಆರಂಭಿಸಿದ ಬಾಹ್ಯ ವೆಬ್ ಸಂವಹನಕ್ಕಾಗಿ URL ಆಧಾರಿತ ಪೋಷಣೆ ಕುರಿತ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಸೇರಿಸಲಾಗಿದೆ  
- **ಕಾರ್ಯಗಳು (ಪ್ರಯೋಗಾತ್ಮಕ)**: ಸ್ಥಿರ ನಿರ್ವಹಣಾ обертки ಮತ್ತು ವಿಲಂಬಿತ ಫಲಿತಾಂಶಗಳನ್ನು ಪತ್ತೆಹಚ್ಚಲು ಪ್ರಾಯೋಗಿಕ ಕಾರ್ಯಗಳ ವಿಜ್ಞಾನ ಲೇಖನ ಸೇರಿಸಲಾಗಿದೆ  
- **ಐಕಾನ್‌ಗಳ ಬೆಂಬಲ**: ಟೂಲ್‌ಗಳು, ಸಂಪನ್ಮೂಲಗಳು, ಸಂಪನ್ಮೂಲ ಟೆಂಪ್ಲೇಟುಗಳು, ಹಾಗೂ ಪ್ರಾಂಪ್ಟ್‌ಗಳಲ್ಲಿ ಹೆಚ್ಚುವರಿ ಮೆಟಾಡೇಟಾ ಆಗಿ ಐಕಾನ್ ಸೇರಿಸಲು ಸೂಚಿಸಲಾಗಿದೆ  

#### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಅಪ್ಡೇಟ್ಗಳು  
- **README.md**: MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25 ಆವೃತ್ತಿ ಉಲ್ಲೇಖ ಮತ್ತು ದಿನಾಂಕ ಆಧಾರಿತ ಆವೃತ್ತಿ ವಿವರವನ್ನು ಸೇರಿಸಲಾಗಿದೆ  
- **study_guide.md**: ಮೂಲ ಕಲ್ಪನೆಗಳ ವಿಭಾಗದಲ್ಲಿ ಕಾರ್ಯಗಳು ಮತ್ತು ಟೂಲ್ ಅನೋಟೇಶನ್‌ಗಳನ್ನು ಸೇರಿಸಿ ಪಠ್ಯಕ್ರಮ ಮಾಪಿಯನ್ನು жаңಗೊಳಿಸಲಾಗಿದೆ; ದಾಖಲೆ ಸಮಯದ ಟೆಸ್ ಸ್ಟ್ಯಾಂಪ್ ಅಪ್ಡೇಟ್ ಮಾಡಲಾಗಿದೆ  

#### ಸ್ಪೆಸಿಫಿಕೇಶನ್ ಪಾಲನಾ ಪರಿಶೀಲನೆ  
- **ಪ್ರೋಟೋಕಾಲ್ ಆವೃತ್ತಿ**: ಎಲ್ಲಾ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಕೆಲವು MCP ಸ್ಪೆಸಿಫಿಕೇಶನ್ 2025-11-25 ಗಾಗಿ ಪರಿಶೀಲನೆ ಮಾಡಲಾಗಿದೆ  
- **ವಾಸ್ತುಶಿಲ್ಪ ಹೊಂದಾಣಿಕೆ**: ಎರಡು ಪದರ ವಾಸ್ತುಶಿಲ್ಪ (ದತ್ತ ಲೇಯರ್ + ಸಾರಿಗೆ ಲೇಯರ್) ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ನಿಖರತೆ ದೃಢೀಕರಿಸಲಾಗಿದೆ  
- **ಪ್ರಿಮಿಟೀವ್ ಗಳ ಡಾಕ್ಯುಮೆಂಟೇಶನ್**: ಸರ್ವರ್ ಪ್ರಿಮಿಟೀವ್‌ಗಳು (ಸಂಪನ್ಮೂಲಗಳು, ಪ್ರಾಂಪ್ಟ್‌ಗಳು, ಟೂಲ್ಗಳು) ಹಾಗೂ ಕ್ಲಯಿಂಟ್ ಪ್ರಿಮಿಟೀವ್‌ಗಳು (ಸ್ಯಾಂಪ್ಲಿಂಗ್, ಪೋಷಣೆ, ಲಾಗಿಂಗ್, ರೂಟ್ಸ್) ಪರಿಶೀಲಿಸಲಾಯಿತು  
- **ಸಾರಿಗೆ ಯಂತ್ರಗಳು**: STDIO ಮತ್ತು ಸ್ಟ್ರಿಮಿಂಗು HTTP ಸಾರಿಗೆ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ನಿಖರತೆ ಪರಿಶೀಲಿಸಲಾಗಿದೆ  
- **ಭದ್ರತಾ ಮಾರ್ಗದರ್ಶನ**: MCP ಭದ್ರತಾ ಉತ್ತಮ ಅಭ್ಯಾಸಗಳೊಂದಿಗೆ ಹೊಂದಾಣಿಕೆ ದೃಢೀಕರಿಸಲಾಗಿದೆ  

#### ಪ್ರಮುಖ MCP 2025-11-25 ವೈಶಿಷ್ಟ್ಯಗಳು ದಾಖಲಾಗಿವೆ  
- **OpenID Connect ಕಂಡುಹಿಡಿತ**: OIDC ಮೂಲಕ ಪ್ರಾಧಿಕಾರ ಲಭ್ಯವಿಕೆ  
- **OAuth ಕ್ಲಯಿಂಟ್ ID ಮೆಟಾಡೇಟಾ ಡಾಕ್ಯುಮೆಂಟ್**: ಕ್ಲಯಿಂಟ್ ನೋಂದಣಿ ಪ್ರಕ್ರಿಯೆಯನ್ನು ಶಿಫಾರಸು ಮಾಡಲಾಗಿದೆ  
- **JSON Schema 2020-12**: MCP ಪದವಿ ವ್ಯಾಖ್ಯಾನಗಳಿಗೆ ಡೀಫಾಲ್ಟ್ ಭಾಷೆ  
- **SDK ಶ್ರೆಣೀಕರಣ ವ್ಯವಸ್ಥೆ**: SDK ವೈಶಿಷ್ಟ್ಯಗಳ ಬೆಂಬಲ ಮತ್ತು ನಿರ್ವಹಣೆಗೆ ಅಧಿಕೃತ ಅಗತ್ಯಗಳು  
- **ಶಾಸನಾ ನಿರ್ಮಾಣ**: MCP ಆಡಳಿತದಲ್ಲಿ ಕಾರ್ಯಗಾರ ಗುಂಪುಗಳು ಮತ್ತು ಸೂಚನಾ ಗುಂಪುಗಳ ಅಧಿಕೃತೀಕರಣ

#### MCP ಭದ್ರತಾ ಶೃಂಗಸಭಾ ಕೆಲಸಾಗಾರ (ಶೆರ್ಪಾ) ಏಕೀಕರಣ

## ಅಕ್ಟೋಬರ್ 6, 2025

### ಆರಂಭಗೊಳ್ಳುವ ವಿಭಾಗದ ವಿಸ್ತರಣೆ – ಉನ್ನತ ಸರ್ವರ್ ಬಳಕೆ & ಸರಳ ಪ್ರಮಾಣೀಕರಣ

#### ಉನ್ನತ ಸರ್ವರ್ ಬಳಕೆ (03-GettingStarted/10-advanced)
- **ಹೊಸ ಅಧ್ಯಾಯ ಸೇರ್ಪಡೆ**: ಸಾಮಾನ್ಯ ಮತ್ತು ತಳ ಮಟ್ಟದ ಸರ್ವರ್ ಸ್ಥಾಪನೆಗಳನ್ನೊಳಗೊಂಡ ಉನ್ನತ MCP ಸರ್ವರ್ ಬಳಕೆಯ ಸಮಗ್ರ ಮಾರ್ಗದರ್ಶಿಯನ್ನು ಪರಿಚಯಿಸಲಾಗಿದೆ.
  - **ಸಾಮಾನ್ಯ ಮತ್ತು ತಳ ಮಟ್ಟ ಸರ್ವರ್**: ಎರಡರ ಮತ್ತಾಗಿ ಪೈಥಾನ್ ಮತ್ತು ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್ ನಲ್ಲಿ ಕೋಡ್ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ ವಿವರಾತ್ಮಕ ಹೋಲಿಕೆ.
  - **ಹ್ಯಾಂಡ್ಲರ್ ಆಧಾರಿತ ವಿನ್ಯಾಸ**: ಪರಿಮಾಣಯೋಗ್ಯ, ಬಲವಾದ ಸರ್ವರ್ ಅನುಸ್ಥಾಪನೆಗಾಗಿ ಹ್ಯಾಂಡ್ಲರ್ ಆಧರಿತ ಸಾಧನ/ಸಂಪನ್ಮೂಲ/ಪ್ರಾಂಪ್ಟ್ ನಿರ್ವಹಣೆಯ ವಿವರಣೆ.
  - **ಪ್ರಾಯೋಗಿಕ ಮಾದರಿಗಳು**: ಉನ್ನತ ವೈಶಿಷ್ಟ್ಯಗಳು ಮತ್ತು ವಾಸ್ತುಶಿಲ್ಪಗಳಿಗೆ ತಳ ಮಟ್ಟದ ಸರ್ವರ್ ಮಾದರಿಗಳು ಲಾಭದಾಯಕವಾಗಿರುವ ನೈಜ ಸಂದರ್ಭಗಳು.

#### ಸರಳ ಪ್ರಮಾಣೀಕರಣ (03-GettingStarted/11-simple-auth)
- **ಹೊಸ ಅಧ್ಯಾಯ ಸೇರ್ಪಡೆ**: MCP ಸರ್ವರ್‌ಗಳಲ್ಲಿ ಸರಳ ಪ್ರಮಾಣೀಕರಣವನ್ನು ಜಾರಿಗೆ ತರಲು ಹಂತದ ಹಂತದ ಮಾರ್ಗದರ್ಶಿ.
  - **ಅಥೆಂಟ್ ತತ್ವಗಳು**: ಪ್ರಮಾಣೀಕರಣ ಮತ್ತು ಅನುಮೋದನೆಯ ಸ್ಪಷ್ಟ ವಿವರಣೆ ಮತ್ತು ಕ್ರೆಡೆನ್ಶಿಯಲ್ ನಿರ್ವಹಣೆ.
  - **ಮೂಲ<Auth> ಜಾರಿಗೊಳ್ಳುವಿಕೆ**: ಪೈಥಾನ್ (ಸ್ಟಾರ್ಲೆಟ್) ಮತ್ತು ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್ (ಎಕ್ಸ್‌ಪ್ರೆಸ್) ನಲ್ಲಿ ಮಧ್ಯಸ್ಥಮಾಧ್ಯಮ ಆಧಾರಿತ ಪ್ರಮಾಣೀಕೃತ ಮಾದರಿಗಳು ಹಾಗೂ ಕೋಡ್ ಉದಾಹರಣೆಗಳೊಂದಿಗೆ.
  - **ಉನ್ನತ ಭದ್ರತೆಗೆ ಪ್ರಗತಿ**: ಸರಳ<Auth> ಮೂಲಕ ಪ್ರಾರಂಭಿಸಿ OAuth 2.1 ಮತ್ತು RBAC ಗೆ ಮುಂದುವರೆಯಲು ಮಾರ್ಗದರ್ಶನ, ಉನ್ನತ ಭದ್ರತಾ ಘಟಕಗಳಿಗೆ ಉಲ್ಲೇಖಗಳೊಂದಿಗೆ.

ಈ ಸೇರ್ಪಡೆಗಳು ಮೂಲಭೂತ ತತ್ವಗಳು ಮತ್ತು ಉನ್ನತ ಉತ್ಪಾದನಾ ಮಾದರಿಗಳ ನಡುವೆ ಸೇತುವೆಯಾಗುವ ಮೂಲಕ, ಹೆಚ್ಚು ಬಲವಾದ, ಭದ್ರವಾದ ಹಾಗೂ ಬಲಿಷ್ಟ MCP ಸರ್ವರ್ ಅನುಸ್ಥಾಪನೆಗಳನ್ನು ನಿರ್ಮಿಸಲು ಪ್ರಾಯೋಗಿಕ, ಕೈಗಾರಿಕಾ ಮಾರ್ಗದರ್ಶನವನ್ನು ನೀಡುತ್ತದೆ.

- **README.md**: ಕಸ್ಟಮ್ MCP ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ವಿಧಾನಗಳ ಸಂಪೂರ್ಣ ಜಾರಿಗೊಳ್ಳುವಿಕೆ ಮಾರ್ಗದರ್ಶಿ
  - **Azure ಇವೆಂಟ್ ಗ್ರಿಡ್ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್**: ಸಂಪೂರ್ಣ ಸರ್ವರ್‌ರಹಿತ ಇವೆಂಟ್-ಚಾಲಿತ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಜಾರಿಗೊಳ್ಳುವಿಕೆ
    - C#, ಟೈಪ್‌ಸ್ಕ್ರಿಪ್ಟ್ ಮತ್ತು ಪೈಥಾನ್ ಉದಾಹರಣೆಗಳು Azure ಫಂಕ್ಷನ್ಸ್ ಏಕೀಕರಣದೊಂದಿಗೆ
    - ಬೃಹತ್ ಸಹಜ ಇವೆಂಟ್-ಚಾಲಿತ ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳು MCP ಪರಿಹಾರಗಳಿಗೆ
    - ವೆಬ್‌ಹುಕ್ ಸ್ವೀಕಾರಕರು ಮತ್ತು ಪುಷ್ ಆಧಾರಿತ ಸಂದೇಶ ನಿರ್ವಹಣೆ
  - **Azure ಇವೆಂಟ್ ಹಬ್ಸ್ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್**: ಅಧಿಕ-ಡೇಟಾ ಹರಿವು ಹೊಂದಿದ ಸ್ಟ್ರೀಮಿಂಗ್ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ಜಾರಿಗೊಳ್ಳುವಿಕೆ
    - ಕಡಿಮೆ ವಿಳಂಬದ ಸಂದರ್ಭಗಳಿಗೆ ನಿಜಕಾಲ ಸ್ಟ್ರೀಮಿಂಗ್ ಸಾಮರ್ಥ್ಯಗಳು
    - ವಿಭಜನೆ ನೀತಿಗಳು ಮತ್ತು ಚೆಕ್‌ಪಾಯಿಂಟ್ ನಿರ್ವಹಣೆ
    - ಸಂದೇಶ ಬ್ಯಾಚಿಂಗ್ ಮತ್ತು ಕಾರ್ಯಕ್ಷಮತೆ ಸುಧಾರಣೆ
  - **ಎಂಟರ್‌ಪ್ರೈಸ್ ಏಕೀಕರಣ ಮಾದರಿಗಳು**: ಉತ್ಪಾದನಾ-ಸಿದ್ಧ ವಾಸ್ತುಶಿಲ್ಪ ಉದಾಹರಣೆಗಳು
    - ಬೇರೆಯ Azure ಫಂಕ್ಷನ್ಸ್‌ಗಳ ಮೇಲೆ ಹಂಚಿಕೆಗೊಂಡ MCP ಪ್ರಕ್ರಿಯೆ
    - ವಿವಿಧ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ರಕಗಳನ್ನು ಸಂಯೋಜಿಸುವ ಸಂಯುಕ್ತ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್ ವಾಸ್ತುಶಿಲ್ಪಗಳು
    - ಸಂದೇಶ ಸ್ಥಿರತೆ, ವಿಶ್ವಾಸಾರ್ಹತೆ ಮತ್ತು ದೋಷ ನಿರ್ವಹಣಾ ನೀತಿಗಳು
  - **ಭದ್ರತೆ ಮತ್ತು ವೀಕ್ಷಣೆ**: Azure Key Vault ಏಕೀಕರಣ ಮತ್ತು ವೀಕ್ಷಣಾ ಮಾದರಿಗಳು
    - ನಿರ್ವಹಿತ ಗುರುತು ಪ್ರಮಾಣೀಕರಣ ಮತ್ತು ಕನಿಷ್ಟ ಅನುಮತಿ 접근
    - ಅಪ್ಲಿಕೇಶನ್ ಇನ್ಸೈಟ್ಸ್ ಟೆಲಿಮೆಟ್ರಿ ಮತ್ತು ಕಾರ್ಯಕ್ಷಮತೆ ವೀಕ್ಷಣೆ
    - ಸರ್ಕ್ಯೂಟ್ ಬ್ರೇಕರ್‌ಗಳ ಮತ್ತು ದೋಷ ಸಹಿಷ್ಣುತನ ಮಾದರಿಗಳು
  - **ಪರೀಕ್ಷಣಾ ಚಟುವಟಿಕೆಗಳು**: ಕಸ್ಟಮ್ ಟ್ರಾನ್ಸ್‌ಪೋರ್ಟ್‌ಗಳ ಸಂಪೂರ್ಣ ಪರೀಕ್ಷಣಾ ನीतಿಗಳು
    - ಟೆಸ್ಟ್ ಡಬಲ್ಸ್ ಮತ್ತು ಮೊಕಿಂಗ್ ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳೊಂದಿಗೆ ಯೂನಿಟ್ ಟೆಸ್ಟಿಂಗ್
    - Azure ಟೆಸ್ಟ್ ಕಂಟೈನರ್ಸ್ ಜೊತೆಗೆ ಏಕೀಕರಣ ಪರೀಕ್ಷೆ
    - ಕಾರ್ಯಕ್ಷಮತೆ ಮತ್ತು ಲೋಡ್ ಪರೀಕ್ಷಣೆಯ ಪರಿಗಣನೆಗಳು

#### ಕಾನ್ಟೆಕ್ಸ್ಟ್ ಇಂಜಿನಿಯರಿಂಗ್ (05-AdvancedTopics/mcp-contextengineering/) - ಉದಯೋನ್ಮುಖ AI ವಿಭಾಗ
- **README.md**: ಉದಯೋನ್ಮುಖ ಕ್ಷೇತ್ರವಾಗಿ ಕಾನ್ಟೆಕ್ಸ್ಟ್ ಇಂಜಿನಿಯರಿಂಗ್ ವಿಸ್ತೃತ ಅನ್ವೇಷಣೆ
  - **ಪ್ರಧಾನ ತತ್ವಗಳು**: ಸಂಪೂರ್ಣ ಕಾನ್ಟೆಕ್ಸ್ಟ್ ಹಂಚಿಕೆ, ಕ್ರಿಯಾ ನಿರ್ಧಾರ ಜಾಗೃತಿ, ಮತ್ತು ಕಾನ್ಟೆಕ್ಸ್ಟ್ ವಿಂಡೋ ನಿರ್ವಹಣೆ
  - **MCP ಪ್ರೊಟೋಕಾಲ್ ಹೊಂದಾಣಿಕೆ**: MCP ವಿನ್ಯಾಸವು ಕಾನ್ಟೆಕ್ಸ್ಟ್ ಇಂಜಿನಿಯರಿಂಗ್ ಸವಾಲುಗಳಿಗೆ ಹೇಗೆ ಸ್ಪಂದಿಸುತ್ತದೆ
    - ಕಾನ್ಟೆಕ್ಸ್ಟ್ ವಿಂಡೋ ಮಿತಿ ಮತ್ತು ಕ್ರಮೇಣ ಲೋಡಿಂಗ್ ತಂತ್ರಗಳು
    - ಪ್ರಾಸಂಗಿಕತೆ ನಿರ್ಣಯ ಮತ್ತು ಗತಿಯ ಕಾನ್ಟೆಕ್ಸ್ಟ್ ಪಡೆಯುವಿಕೆ
    - ಬಹು ಮಾದ್ರಿಕ ಕಾನ್ಟೆಕ್ಸ್ಟ್ ನಿರ್ವಹಣೆ ಮತ್ತು ಭದ್ರತಾ ಪರಿಗಣನೆಗಳು
  - **ಅಮಲು ವಿಧಾನಗಳು**: ಏಕ-ಪಂಕ್ತಿ ವಿರುದ್ಧ ಬಹು-ಏಜೆಂಟ್ ವಾಸ್ತುಶಿಲ್ಪಗಳು
    - ಕಾನ್ಟೆಕ್ಸ್ಟ್ ಚಂಕಿಂಗ್ ಮತ್ತು ಆದ್ಯತೆ ನೀಡುವ ತಂತ್ರಗಳು
    - ಕ್ರಮೇಣ ಕಾನ್ಟೆಕ್ಸ್ಟ್ ಲೋಡಿಂಗ್ ಮತ್ತು ಸಂಕೋಚನ ತಂತ್ರಗಳು
    - ಪದರದ ಕಾನ್ಟೆಕ್ಸ್ಟ್ ವಿಧಾನಗಳು ಮತ್ತು ಪಡೆಯುವಿಕೆಯ ಆಪ್ಟಿಮೈಸೇಶನ್
  - **ಮಾಪನ ಚಟುವಟಿಕೆ**: ಕಾನ್ಟೆಕ್ಸ್ಟ್ ಪರಿಣಾಮಶೀಲತೆ ಮೌಲ್ಯಮಾಪನಕ್ಕೆ ಉದಯೋನ್ಮುಖ ಮಿತಿಗಳು
    - ಇನ್‌ಪುಟ್ ಕಾರ್ಯಕ್ಷಮತೆ, ಕಾರ್ಯಕ್ಷಮತೆ, ಗುಣಮಟ್ಟ, ಮತ್ತು ಬಳಕೆದಾರ ಅನುಭವ ಪರಿಗಣನೆಗಳು
    - ಕಾನ್ಟೆಕ್ಸ್ಟ್ ಆಪ್ಟಿಮೈಸೇಷನ್‌ಗೆ ಪ್ರಯೋಗಾತ್ಮಕ ವಿಧಾನಗಳು
    - ವೈಫಲ್ಯ ವಿಶ್ಲೇಷಣೆ ಮತ್ತು ಸುಧಾರಣಾ ವಿಧಾನಗಳು

#### ಹೃದಯ ತತ್ವಗಳು (01-CoreConcepts/) - ಸಂಪೂರ್ಣ ನವೀಕರಣ
- **ಪ್ರೋಟೋಕಾಲ್ ಆವೃತ್ತಿ ನವೀಕರಣ**: MCP ನಿರ್ದಿಷ್ಟಣ 2025-06-18 ಗೆ ಅಪ್ಡೇಟ್, ದಿನಾಂಕ ಆಧಾರಿತ ಆವೃತ್ತಿ ನಮೂನೆ (YYYY-MM-DD)
- **ವಾಸ್ತುಶಿಲ್ಪ ಸುಧಾರಣೆ**: Hosts, Clients, Servers ವಿವರಣೆಗಳನ್ನು MCP ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳಿಗೆ ಹೊಂದಿಸಲಾಗಿದೆ
  - Hosts ಈಗ ಸ್ಪಷ್ಟವಾಗಿ MCP ಕ್ಲೈಂಟ್ ಸಂಪರ್ಕಗಳನ್ನು ಸಂಯೋಜಿಸುವ AI ಅಪ್ಲಿಕೇಶನ್‌ಗಳಾಗಿ ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿದೆ
  - Clientsನನ್ನು ಪ್ರೋಟೋಕಾಲ್ ಸಂಪರ್ಕಕಾರಿಗಳಾಗಿ 1:1 ಸರ್ವರ್ ಸಂಬಂಧವನ್ನು ಕಾಯ್ದುಕೊಳ್ಳುವಂತೆ ವರ್ಣಿಸಲಾಗಿದೆ
  - Servers ಸ್ಥಳೀಯ ಮತ್ತು ದೂರಸ್ಥ ನಿಯೋಜನಾ ದೃಶ್ಯಗಳಿಗೆ ಸುಧಾರಣೆ
- **ಮೂಲಭೂತ ಪುನರಾರೋಪಣೆ**: ಸರ್ವರ್ ಮತ್ತು ಕ್ಲೈಂಟ್ ಮೂಲಭೂತಗಳ ಸಂಪೂರ್ಣ ಪುನರಲೇಖನ
  - ಸರ್ವರ್ ಮೂಲಭೂತಗಳು: ಸಂಪನ್ಮೂಲಗಳು (ಡೇಟಾ ಮೂಲಗಳು), ಪ್ರಾಂಪ್ಟ್‌ಗಳು (ಟೆಂಪ್ಲೇಟ್ಗಳು), ಸಾಧನಗಳು (ಕಾರ್ಯನಿರ್ವಹಣಾ ಫಂಕ್ಷನ್ಗಳು) ವಿವರಣೆ ಮತ್ತು ಉದಾಹರಣೆಗಳೊಂದಿಗೆ
  - ಕ್ಲೈಂಟ್ ಮೂಲಭೂತಗಳು: ಸ್ಯಾಂಪ್ಲಿಂಗ್ (LLM ಸಂಪೂರ್ಣತೆ), ಕೇಳುಕೆ (ಬಳಕೆದಾರ ಇನ್‌ಪುಟ್), ಲಾಗಿಂಗ್ (ಡೀಬಗ್/ಮೇಲ್ವಿಚಾರಣೆ)
  - ಪ್ರಸ್ತುತ ಅನ್ವೇಷಣೆ (`*/list`), ಪಡೆದು (`*/get`), ಮತ್ತು ಕಾರ್ಯಾಚರಣೆ (`*/call`) ವಿಧಾನ ಮಾದರಿಗಳೊಂದಿಗೆ ನವೀಕರಣ
- **ಪ್ರೋಟೋಕಾಲ್ ವಾಸ್ತುಶಿಲ್ಪ**: ಎರಡು-ಮಟ್ಟದ ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಯನ್ನು ಪರಿಚಯಿಸಿದೆ
  - ಡೇಟಾ ಮಟ್ಟ: JSON-RPC 2.0 ನೆಲೆ ಮತ್ತು ಜೀವನಚಕ್ರ ನಿರ್ವಹಣೆ ಮತ್ತು ಮೂಲಭೂತಗಳು
  - ಸಾರಿಗೆ ಮಟ್ಟ: STDIO (ಸ್ಥಳೀಯ) ಮತ್ತು ಸ್ಟ್ರೀಮಬಲ್ HTTP SSE (ದೂರಸ್ಥ) ಸಾರಿಗೆ ಕ್ರಮಗಳು
- **ಭದ್ರತಾ ಚಟುವಟಿಕೆ**: ಸುಪರಿಪೂರ್ಣ ಭದ್ರತಾ ತತ್ವಗಳು ಸ್ಪಷ್ಟ ಬಳಕೆದಾರ ಒಪ್ಪಿಗೆ, ಡೇಟಾ ಗೌಪ್ಯತೆ, ಸಾಧನ ಕಾರ್ಯನಿರ್ವಹಣಾ ಸುರಕ್ಷತೆ ಮತ್ತು ಸಾರಿಗೆ ಮಟ್ಟ ಭದ್ರತೆ
- **ಸಂವಹನ ಮಾದರಿಗಳು**: ಪ್ರೋಟೋಕಾಲ್ ಸಂದೇಶಗಳನ್ನು ಆರಂಭ, ಅನ್ವೇಷಣೆ, ಕಾರ್ಯಾಚರಣೆ ಮತ್ತು ನೋಟಿಫಿಕೇಶನ್ ಹರಿವುಗಳಾಗಿ ನವೀಕರಿಸಲಾಗಿದೆ
- **ಕೋಡ್ ಉದಾಹರಣೆಗಳು**: ಪ್ರಸ್ತುತ MCP SDK ಮಾದರಿಗಳಿಗೆ ಹೊಂದುವಂಥ ಬಹುಭಾಷಾ ಉದಾಹರಣೆಗಳನ್ನು (.NET, ಜಾವಾ, ಪೈಥಾನ್, ಜಾವಾಸ್ಕ್ರಿಪ್ಟ್) ನವೀಕರಿಸಲಾಗಿದೆ

- **ಉನ್ನತ ಧಮಕಿ ತಡೆ**: ವಿವರವಾದ ಭದ್ರತಾ ನಿಯಂತ್ರಣಗಳಿಗಾಗಿ
  - MCP-ವಿಶೇಷ ಸೆಷನ್ ಹೈಜ್ಯಾಕಿಂಗ್ ದಾಳಿ ದೃಶ್ಯಗಳು ಮತ್ತು ಕ್ರಿಪ್ಟೋಗ್ರಾಫಿಕ್ ಸೆಷನ್ ID ಅಗತ್ಯತೆಗಳೊಂದಿಗೆ
  - MCP ಪ್ರಾಕ್ಸಿ ದೃಶ್ಯಗಳಲ್ಲಿ ಕಾನ್ಫ್ಯೂಸ್‌ಡ್ ಡೆಪ್ಯೂ ಸಮಸ್ಯೆಗಳು ಸ್ಪಷ್ಟ ಒಪ್ಪಿಗೆ ಅಗತ್ಯತೆಗಳೊಂದಿಗೆ
  - ಟೋಕನ್ ಪಾಸ್ತ್ರೂ ದುರ್ಬಲತೆಗಳಿಗೆ ಅಗತ್ಯ ಪರಿಶೀಲನಾ ನಿಯಂತ್ರಣಗಳು
- **ಸರಬರಾಜು ಸರಪಳಿ ಭದ್ರತೆ**: ಫೌಂಡೇಶನ್ ಮಾದರಿಗಳು, ಎम्बೆಡ್ಡಿಂಗ್ ಸೇವೆಗಳು, ಕಾಂಟೆಕ್ಸ್ಟ್ ಒದಗಿಸುವವರು ಮತ್ತು ತೃತೀಯ-ಪಕ್ಷ APIಗಳು ಸೇರಿದಂತೆ AI ಸರಬರಾಜು ಸರಪಳಿಯ ವಿಸ್ತಾರ
- **ಫೌಂಡೇಶನ್ ಭದ್ರತೆ**: ಶೂನ್ಯ ನಂಬಿಕೆ ವಾಸ್ತುಶಿಲ್ಪ ಮತ್ತು Microsoft ಭದ್ರತಾ ಪರಿಸರದೊಂದಿಗೆ ಉನ್ನತ ಏಕೀಕರಣ
- **ಸಂಪನ್ಮೂಲ ಸಂಘಟನೆ**: ತುಂಡಗರಿಸಿರುವ ಸಂಪನ್ಮೂಲ ಲಿಂಕ್‌ಗಳು ವಿಧಗಳು (ಅಧಿಕೃತ ಡಾಕ್ಸ್, ಮಾನದಂಡಗಳು, ಸಂಶೋಧನೆ, Microsoft ಪರಿಹಾರಗಳು, ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳು) ಮೂಲಕ

### ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಗುಣಮಟ್ಟ ಸುಧಾರಣೆಗಳು
- **ಸಂರಚಿತ ಅಧ್ಯಯನ ಗುರಿಗಳು**: ನಿಗದಿತ, ಕ್ರಿಯಾಶೀಲ ಫಲಿತಾಂಶಗಳೊಂದಿಗೆ ಅಧ್ಯಯನ ಗುರಿಗಳನ್ನು ಹೆಚ್ಚಿಸಿದೆ
- **ಕ್ರಾಸ್-ರেফರೆನ್ಸ್‌ಗಳು**: ಸಂಬಂಧಪಟ್ಟ ಭದ್ರತಾ ಮತ್ತು ಹೃದಯ ತತ್ವ ವಿಷಯಗಳ ನಡುವೆ ಲಿಂಕ್‌ಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- **ಪ್ರಸ್ತುತ ಮಾಹಿತಿ**: ಎಲ್ಲ ದಿನಾಂಕ ಉಲ್ಲೇಖಗಳು ಮತ್ತು ನಿರ್ದಿಷ್ಟಣ ಲಿಂಕ್‌ಗಳನ್ನು ಪ್ರಸ್ತುತ ಮಾನದಂಡಗಳಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- **ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶಿಗಳು**: ಎರಡೂ ವಿಭಾಗಗಳಲ್ಲಿ ನಿಗದಿತ, ನಿರ್ದಿಷ್ಟ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶನಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ

## ಜುಲೈ 16, 2025

### README ಮತ್ತು ನ್ಯಾವಿಗೇಶನ್ ಸುಧಾರಣೆಗಳು
- README.md ನಲ್ಲಿ ಪಠ್ಯಕ್ರಮ ನ್ಯಾವಿಗೇಶನ್ ಅನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಪುನರ್ರಚಿಸಲಾಗಿದೆ
- `<details>` ಟ್ಯಾಗ್‌ಗಳನ್ನು ಟೇಬಲ್ ಆಧಾರಿತ ಮತ್ತಷ್ಟು ಪ್ರವೇಶಕಾರಿ ಸ್ವರೂಪದಿಂದ ಬದಲಿಸಲಾಗಿದೆ
- ಹೊಸ "alternative_layouts" ಫೋಲ್ಡರ್‌ನಲ್ಲಿ ಪರ್ಯಾಯ ವಿನ್ಯಾಸ ಆಯ್ಕೆಯನ್ನು ಸ್ರಷ್ಟಿಸಲಾಗಿದೆ
- ಕಾರ್ಡ್-ಆಧಾರಿತ, ಟ್ಯಾಬ್‌ ವಿನ್ಯಾಸದ ಮತ್ತು ಅಕಾರ್ಡಿಯನ್-ಶೈಲಿಯ ನ್ಯಾವಿಗೇಶನ್ ಉದಾಹರಣೆಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- ಎಲ್ಲ ಇತ್ತೀಚಿನ ಕಡತಗಳನ್ನು ಹೊಂದಿಸುವಂತೆ ರೆಪೊ ಸ್ಟ್ರಕ್ಚರ್ ವಿಭಾಗವನ್ನು ನವೀಕರಿಸಲಾಗಿದೆ
- "ಈ ಪಠ್ಯಕ್ರಮವನ್ನು ಹೇಗೆ ಬಳಸಬೇಕು" ವಿಭಾಗವನ್ನು ಸ್ಪಷ್ಟ ಶಿಫಾರಸುಗಳೊಂದಿಗೆ ಅಭಿವೃದ್ಧಿಪಡಿಸಲಾಗಿದೆ
- MCP ನಿರ್ದಿಷ್ಟಣ ಲಿಂಕ್‌ಗಳನ್ನು ಸರಿಯಾದ URL ಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
- ಪಠ್ಯಕ್ರಮ ಸಂರಚನೆಗೆ ಕಾಂಟೆಕ್ಸ್ಟ್ ಎಂಜಿನಿಯರಿಂಗ್ ವಿಭಾಗ (5.14) ಸೇರಿಸಲಾಗಿದೆ

### ಅಧ್ಯಯನ ಮಾರ್ಗದರ್ಶಿಯ ನವೀಕರಣಗಳು
- ಪ್ರಸ್ತುತ ರೆಪೊ ಸಂರಚನೆಗೆ ಹೊಂದುವಂತೆ ಅಧ್ಯಯನ ಮಾರ್ಗದರ್ಶಿಯನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಮರೆಯಲಾಗಿದೆ
- MCP ಕ್ಲೈಂಟ್‌ಗಳು ಮತ್ತು ಸಾಧನಗಳು, ಜನಪ್ರಿಯ MCP ಸರ್ವರ್‌ಗಳ ಹೊಸ ವಿಭಾಗಗಳನ್ನು ಸೇರಿಸಲಾಗಿದೆ
- ದೃಶ್ಯ ಪಠ್ಯಕ್ರಮ ನಕ್ಷೆಯನ್ನು ಎಲ್ಲಾ ವಿಷಯಗಳಿಗೆ ನಿಖರವಾಗಿ ಅನಿಸುಮಾಡುವಂತೆ ನವೀಕರಿಸಲಾಗಿದೆ
- ಉನ್ನತ ವಿಷಯಗಳ ವಿವರಣೆಗಳನ್ನು ಎಲ್ಲಾ ವಿಶೇಷಿತ ಕ್ಷೇತ್ರಗಳನ್ನು ಒಳಗೊಂಡಂತೆ ವೃದ್ಧಿಸಲಾಗಿದೆ
- ನಿಜವಾದ ಉದಾಹರಣೆಗಳನ್ನು ಪ್ರತಿಬಿಂಬಿಸುವಂತೆ ಪ್ರಕರಣ ಅಧ್ಯಯನಗಳು ವಿಭಾಗವನ್ನು ನವೀಕರಿಸಲಾಗಿದೆ
- ಈ ಸಮಗ್ರ ಚೇಂಜ್‌లಾಗ್ ಸೇರಿಸಲಾಗಿದೆ

- MCP ಕ್ಲೈಂಟ್ ವಿಭಾಗವನ್ನು ಎಲ್ಲಾ ಜನಪ್ರಿಯ ಕ್ಲೈಂಟ್ ಆಯ್ಕೆಗಳೊಂದಿಗೆ ನವೀಕರಿಸಲಾಗಿದೆ
